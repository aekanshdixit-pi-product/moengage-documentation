Push Template APIs help marketers define, reuse, update, and manage push templates created outside the MoEngage ecosystem effortlessly. One can create multiple versions of the same template and mark whether they can be used in the campaigns that are active currently. Users can create templates using the Create PushAPI and update them using the Update Push API or edit them in the Dashboard (provided they have the specific editing permissions for the templates allowed for their role). 

The following APIs are available:

## Create Push Template

This API creates a push template. For more information, refer to [Create Push Template](https://developers.moengage.com/hc/en-us/articles/14187268321428-Create-Custom-Push-Template-API).

## Update Push Template

This API updates a push template specified by its Template ID. For more information, refer to [Update Push Template](https://developers.moengage.com/hc/en-us/articles/14223882535828-Update-Push-Template-API).

## Search Push Template

This API fetches a push template using its Template ID or other filters like template name, template version, and so on. It can also list all the push templates created in MoEngage. For more information, refer to [Search Push Template](https://developers.moengage.com/hc/en-us/articles/14226294477972).
