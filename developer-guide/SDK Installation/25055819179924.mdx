# Plugin Installation

To add MoEngage's Flutter SDK to your application, edit your ***pubspec.yaml*** to add  ***moengage\_flutter*** as a dependency.

<CodeGroup>
``` pubspec.yaml
dependencies:
	moengage_flutter: $lastestVersion
```

</CodeGroup>

***$latestVersion*** refers to the latest version of the plugin.

Post including the dependency run ***flutter pub get*** command in terminal to install the dependency.

<Info>
A working Sample App can be found [here](https://github.com/moengage/Flutter-SDK).
</Info>

After installing the plugin, use the following platform-specific configuration.

* [Android](https://developers.moengage.com/hc/en-us/articles/11650947365012-Android)
* [iOS](https://developers.moengage.com/hc/en-us/articles/11650948962452-iOS)
* [Web](https://developers.moengage.com/hc/en-us/articles/4404305461012)
