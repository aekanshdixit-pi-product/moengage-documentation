# Integration through CocoaPods

[![CocoaPods](https://img.shields.io/cocoapods/v/MoEngage-iOS-SDK.svg?style=flat)](https://github.com/moengage/MoEngage-iOS-SDK)

Cocoapods is a dependency manager for Objective C & Swift projects and makes integration easier.

<Steps>
  <Step title="Install CocoaPods">
    If you don't have CocoaPods installed, you can do it by executing the following line in your terminal.

    <Tabs>
      <Tab title="Ruby">
        ```ruby
        sudo gem install cocoapods
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Configure Podfile">
    If you don't have a Podfile, then create one by using `pod init` command. Post this add `MoEngage-iOS-SDK` pod to your pod file as shown below:

    <Tabs>
      <Tab title="Ruby">
        ```ruby
        pod 'MoEngage-iOS-SDK',
        ```
      </Tab>
    </Tabs>

    <Info>
    **Information**
    * To ensure automatic minor version updates for MoEngage-iOS-SDK, configure your Podfile with `pod 'MoEngage-iOS-SDK', '~> Major.Minor.Build'`. For example: `'MoEngage-iOS-SDK', '~>10.03.2`
    * For automatic integration of the latest MoEngage-iOS-SDK version, including major changes, use `pod 'MoEngage-iOS-SDK'`.
    </Info>
  </Step>

  <Step title="Install Dependencies">
    Integrate MoEngage iOS SDK by executing the following in the terminal at your project's root directory:

    <Tabs>
      <Tab title="Ruby">
        ```ruby
        pod repo update
        pod install
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Verify Integration">
    Now, open your project workspace and check if MoEngage SDK is properly added.
  </Step>
</Steps>

# Integration through Swift Package Manager

Swift Package Manager (SPM) is a tool for managing the distribution of Swift code. It’s integrated with the Swift build system to automate the process of downloading, compiling, and linking dependencies.

To install the MoEngage-iOS-SDK through SPM, follow the below steps: 

<Steps>
  <Step title="Add Package">
    Navigate to **File** -> **Add Package**
  </Step>
  <Step title="Enter Repository URL">
    Enter the URL [https://github.com/moengage/apple-sdk.git](https://github.com/moengage/apple-sdk.git) for SDK versions equal and above 9.23.0, or [https://github.com/moengage/MoEngage-iOS-SDK.git](https://github.com/moengage/MoEngage-iOS-SDK.git) for other SDK versions and select the branch as master or required version to install the package.
  </Step>
  <Step title="Confirm Selection">
    Click on **Add Package**.
  </Step>
  <Step title="Complete Installation">
    Now, MoEngage-iOS-SDK package is installed.
  </Step>
</Steps>
