---
title: "Tracking Locale"
description: "Track the locale settings of your user's device in your iOS app using the MoEngage SDK."
---

For tracking the locale settings of the user device use [_trackLocale()_](https://moengage.github.io/ios-api-reference/Classes/MoEngageSDKAnalytics.html#/c:@M@MoEngageAnalytics@objc\(cs\)MoEngageSDKAnalytics\(im\)trackLocale) method as shown below:

<CodeGroup>

```swift Swift wrap
MoEngageSDKAnalytics.sharedInstance.trackLocale()
```


```objective-c Objective C wrap
[[MoEngageSDKAnalytics sharedInstance] trackLocale];
```

</CodeGroup>
