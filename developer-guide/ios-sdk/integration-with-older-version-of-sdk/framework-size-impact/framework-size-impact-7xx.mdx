For measuring the size impact of our SDK we created a Sample App and uploaded the build with different configurations and checked the AppStore Connect estimate for the Download Size(ipa size) and the Install Size(The amount of disk space the app takes in the customer's device).

<Info>
**SDK versions for analysis**

For the analysis below, the SDK versions considered are as follows:

* MoEngage Core - Includes [MoEngage-iOS-SDK](https://cocoapods.org/pods/MoEngage-iOS-SDK)(7.0.4) and [MORichNotification](https://cocoapods.org/pods/MORichNotification)(5.0.0)
* [MoEngageInApp](https://cocoapods.org/pods/MoEngageInApp) - 2.0.3
* [MoEngageCards](https://cocoapods.org/pods/MoEngageCards) - 2.0.4
* [MOGeofence](https://cocoapods.org/pods/MOGeofence) - 3.0.1
</Info>

<Info>
**iPhone 12 Pro as Reference**

Here for our analysis, we will not consider the Universal build because Apple doesn't install it in any of the user's devices, instead, they create an optimised version of your app, and the user only downloads the architecture that is required for the current device. Therefore for our analysis, we will consider iPhone 12 Pro as a reference point.
</Info>

# App Size Without MoEngage SDK

First, to get a baseline we uploaded a build of our Sample App without any of our frameworks, and the result for the same is shown below. And as mentioned above we will consider iPhone 12 Pro for our analysis, therefore here according to the size estimated the resultant ipa size is 151 KB and the install size is 595 KB.

![a1f42e4-WithoutSDK1.png](4565764305172)

# App Size With MoEngage Core SDK

In this second build, we included the Core SDKs which include [MoEngage-iOS-SDK](https://cocoapods.org/pods/MoEngage-iOS-SDK) and [MORichNotification](https://cocoapods.org/pods/MORichNotification). And the result of the estimated size by AppStore Connect is provided below, as we can see for iPhone 12 Pro the IPA size to be 583 KB and the Install size is 2019 KB, therefore there is an increase of IPA size by 432 KB and install size by 1424 KB here.

![14d2a4a-AppWithCore.png](4565789370772)

<Info>
**Overall Size Impact Of MoEngage**

The Core SDKs consists of the mandatory frameworks for MoEngage features to work, and it includes support for Data Tracking, Analytics, and Push Notification related features. Here we can see an overall impact of 432 KB on ipa size and 1.42 MB on the install size of the app.
</Info>

# App Size With MoEngage Core and InApp SDK

In the third build, we have our Core SDKs which are mandatory and along with it, we have integrated [MoEngageInApp](https://cocoapods.org/pods/MoEngageInApp) which is an optional module. And the result of the estimated size by AppStore Connect is provided below, as we can see the ipa size here is 743 KB and the Install size is 2511 KB, therefore there is an increase of ipa size by 160 KB and install size by 492 KB compared to our second build.

![71e9c46-AppWithCoreAndInapp1.png](4565796833044)

<Info>
**Overall Size Impact Of MoEngage**

With the inclusion of the InApp Module along with Core SDKs, here we can see an overall impact of 592 KB on IPA size and 1.91 MB on the install size of the app.
</Info>

# App Size With MoEngage Core, InApp and Cards SDK

In the fourth build, we have our Core SDKs and MoEngageInApp module and along with it, we have integrated [MoEngageCards](https://cocoapods.org/pods/MoEngageCards) which is again an optional module. And the result of the estimated size by AppStore Connect is provided below, as we can see the ipa size here is 976 KB and the Install size is 3205 KB, therefore there is an increase of ipa size by 233 KB and install size by 694 KB compared to our third build.  
![6e83ae9-AppWithCoreInAppAndCards.png](4565765666324)

<Info>
**Overall Size Impact Of MoEngage**

Here the overall size impact of our Core SDKs along with InApp and Cards module is of 825 KB on ipa size and 2.61 MB on the install size of the app.
</Info>

# App Size With MoEngage Core, InApp, Cards and Geofence SDK

In the fifth build, along with all the modules we had in the previous build, we have integrated [MOGeofence](https://cocoapods.org/pods/MOGeofence) which is again an optional module. And the result of the estimated size by AppStore Connect is provided below, as we can see the ipa size here is 1003 KB and the Install size is 3338 KB, therefore there is an increase of ipa size by 27 KB and install size by 133 KB compared to our previous build.

![9057f13-AppWithAllModules.png](4565791422100)

<Info>
**Overall Size Impact Of MoEngage**

Here the overall size impact of our Core SDKs along with InApp, Cards and geofence modules is of 852 KB on ipa size and 2.74 MB on the install size of the app.
</Info>
