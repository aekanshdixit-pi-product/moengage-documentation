To install via Swift Package Manager:

<Steps>
  <Step title="Add package">
    In Xcode, navigate to **File > Add Package** and enter the URL:
    - For SDK version 9.23.0 and above: `https://github.com/moengage/apple-sdk.git`
    - For earlier SDK versions: `https://github.com/moengage/MoEngage-iOS-SDK.git`
  </Step>
  <Step title="Select version">
    Set the branch to **master** or select the required version.
  </Step>
  <Step title="Add to project">
    Click **Add Package** to complete the installation.
  </Step>
</Steps>
