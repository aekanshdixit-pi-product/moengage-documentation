# Adding MoEngage Plugin

Add the **cordova-moengage-core** plugin to the Cordova project as shown below :

<CodeGroup>

```Shell Shell
$ cordova plugin add cordova-moengage-core
```

</CodeGroup>

After installing the plugin, use the following platform-specific configuration.

- [Android SDK Installation](https://developers.moengage.com/hc/en-us/articles/4404415440532) [Placeholder_Link]
- [iOS SDK Installation](https://developers.moengage.com/hc/en-us/articles/4404422426644) [Placeholder_Link]
