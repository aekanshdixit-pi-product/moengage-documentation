# Initialise Cordova Component

Initialise MoEngage plugin in the **_index.js_** of your application by calling the \*\*\*MoECordova.init()\*\*\*in the **_onDeviceReady()_**.

<CodeGroup>

```javascript JavaScript
var moe = MoECordova.init("YOUR_Workspace_ID");
```

</CodeGroup>
