Before proceeding further, make sure you have done the [Android Installation](https://developers.moengage.com/hc/en-us/articles/7778283953300-Android#configuring-build-settings-0-0) for the `react-native-moengage` plugin.

## Adding MoEngage Cards Dependency

![](https://maven-badges.herokuapp.com/maven-central/com.moengage/cards-core/badge.svg)

Add the following dependency in the *app/build.gradle* file.

<CodeGroup>
```bash build.gradle
dependencies {
    ...
implementation("com.moengage:cards-core:$sdkVersion")
}
```

</CodeGroup>

replace **$sdkVersion** with the appropriate SDK version. Minimum supported version **1.5.0**.
