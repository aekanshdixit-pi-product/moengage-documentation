Install MoEngage's Cards Plugin to your application, using the npm package manager. 

![Download](https://badge.fury.io/js/react-native-moengage-cards.svg?style=flat)

<CodeGroup>
```shell Shell
$ npm install react-native-moengage-cards
```

</CodeGroup>

After installing the plugin, use the following platform-specific configuration.

* [Android](https://developers.moengage.com/hc/en-us/articles/18719492640020-Android)
* [iOS](https://developers.moengage.com/hc/en-us/articles/18719495211028-iOS)

<Warning>
This plugin is dependent on `react-native-moengage` plugin. Make sure you have installed the `react-native-moengage` plugin as well. Refer to the [documentation](https://developers.moengage.com/hc/en-us/articles/4404205340564) for the same.
</Warning>
