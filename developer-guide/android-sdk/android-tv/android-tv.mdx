# Supported Features

MoEngage supports the following:

* [Data Tracking](https://developers.moengage.com/hc/en-us/sections/360013356931-Data-Tracking)
* [HTML In-App](https://developers.moengage.com/hc/en-us/articles/4403652537492)
* [Self-handled InApp](https://developers.moengage.com/hc/en-us/articles/4403652537492-In-App-NATIV#self-handled-inapps-0-12)
* [Self-handled Cards](https://developers.moengage.com/hc/en-us/articles/4703794844820-Self-Handled-Cards)
