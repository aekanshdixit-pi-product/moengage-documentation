# SDK Installation

Add the following dependency in the **_app/build.gradle(.kts)_** file to integrate the required modules. Make sure to replace **\$sdkVersion** with the appropriate SDK version

## Core

<CodeGroup>

```Kotlin build.gradle(.kts) wrap
dependencies {
    ...
    implementation("com.moengage:moe-android-sdk:$sdkVersion")
}
```

</CodeGroup>

<CodeGroup>
