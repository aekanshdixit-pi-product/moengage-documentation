# SDK Installation

## Installing using BOM

Integration using BOM  is the recommended way of integration; refer to the Install Using BOM [Placeholder_Link] document. Once you have configured the BOM add the dependency in the _app/build.gradle_ file as shown below

build.gradle

```
dependencies {
    ...
    implementation("com.moengage:realtime-trigger")
}
```

Once the BOM is configured, include the specific MoEngage modules required for the application. \
Note: Version numbers are not required for these dependencies; the BOM automatically manages them.
