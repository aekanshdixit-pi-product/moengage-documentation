# Troubleshooting

## Not getting Chrome/Firefox/Opera/Edge/Safari Push?

Chrome Push is supported for Chrome (version 42+) on MacOS, Windows (only Desktop), Linux, and Android.

Firefox push is supported for Firefox (version 44+) on MacOS, Windows (only Desktop), Linux, and Android.

Opera push is supported for Opera (version 37+) on MacOS, Windows (only Desktop), Linux, and Android.

Edge push is supported for Edge (version 17+) on MacOS, Windows (only Desktop), Linux, and Android.

Safari push is supported on MacOS (version 13+), Safari (version 16+), and iOS (version 16.4+)

For further assistance, please contact your MoEngage Customer Success Manager (CSM) or the Support team.
