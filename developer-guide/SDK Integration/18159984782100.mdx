Compared to the Native Android/iOS SDKs there are a certain set of features we either do not support or it requires native Android/iOS implementation when using our Unity plugin.

# Features not supported

* Nudge
* Action Buttons in iOS Notifications
* Huawei Push Kit

# Features that are supported but requires Native Integration

* Inbox Module
* Cards
