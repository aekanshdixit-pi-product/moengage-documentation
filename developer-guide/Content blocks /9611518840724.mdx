Content blocks allow marketers to reuse the same content across multiple campaigns. For example, a block might be a repeatedly used header, footer, or designed call-to-action button. [Read more about Content blocks in MoEngage.](https://help.moengage.com/hc/en-us/articles/4414768369172-Content-blocks-)

Content block API enables you to fetch, create and update content blocks on your MoEngage dashboard. 

The following APIs are available: 

* [Get content blocks by id](https://developers.moengage.com/hc/en-us/articles/9611726885652)
* [Search a content block](https://developers.moengage.com/hc/en-us/articles/9611848319508)
* [Create a content block](https://developers.moengage.com/hc/en-us/articles/9466929211412-Create-content-block)
* [Update a content block](https://developers.moengage.com/hc/en-us/articles/9612023786260-Update-content-block-)
