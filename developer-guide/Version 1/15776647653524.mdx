---

Email Template APIs help marketers define, reuse, update, and manage email templates created outside the MoEngage ecosystem effortlessly. The templates created and updated using these APIs are supported for the Custom HTML Editor (Froala Editor) and not the Drag and Drop Editor.

The following APIs are available:

# Create Email Template

This API creates an email template. For more information, refer to [Create Email Template](https://developers.moengage.com/hc/en-us/articles/15470503985172-Create-Email-Template).

# Update a Specific Email Template

This API updates an email template specified by its template id. For more information, refer to [Update a Specific Email Template](https://developers.moengage.com/hc/en-us/articles/15473261478164-Update-a-Specific-Email-Template).

# Bulk Create/Update Email Template

This API creates and updates email templates in bulk. For more information, refer to [Bulk Create/Update Email Template](https://developers.moengage.com/hc/en-us/articles/15475112849812-Bulk-Create-Update-Template).

# Get a Specific Email Template

This API fetches an email template using its template id. For more information, refer to [Get a Specific Email Template.](https://developers.moengage.com/hc/en-us/articles/15475075894164-Get-a-Specific-Email-Template)

# Get all Email Templates

This API fetches all the email templates in your MoEngage account. For more information, refer to [Get All Email Templates](https://developers.moengage.com/hc/en-us/articles/15506305637140-Get-all-templates).
