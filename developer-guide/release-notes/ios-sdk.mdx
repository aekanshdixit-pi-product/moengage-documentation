## Core

<span style={{ backgroundColor: '#e6fffb', color: '#008767', borderRadius: '12px', border: '1px solid #B5F5EC', fontSize: '0.9em', padding: '3px 10px', fontWeight: 'bold' }}>What's New:</span>

- `MoEngageBadgeUpdateEnabled`: Added a new Boolean key to the root level of `Info.plist` to control whether the SDK manages the app's badge count. Defaults to `YES`.
