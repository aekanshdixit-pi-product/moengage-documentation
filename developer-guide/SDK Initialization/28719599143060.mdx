# Initialize Plugin

Initialize the MoEngage Plugin by calling the **MoECapacitorCore.initialize({ appId: "YOUR\_Workspace\_ID" });**. In the case of Ionic-React initialize the plugin in the **useIonViewWillEnter()** callback, for Ionic-Angular initialize the plugin in **ngOnInit()**.

<CodeGroup>
```typescript Typescript
import { MoECapacitorCore} from 'capacitor-moengage-core'
MoECapacitorCore.initialize({ appId: "YOUR_Workspace_ID" })
```

</CodeGroup>

<Warning>
Make sure you are setting the Push/InApp callback listeners before calling the **initialize({ appId: "Workspace ID" })**.
</Warning>
