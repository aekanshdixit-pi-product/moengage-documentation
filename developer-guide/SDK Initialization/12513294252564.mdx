# Initialize Plugin

In the project's App Widget call `initialise()` of `MoEngageFlutter` plugin in `initState()` method as shown below:

<CodeGroup>
``` Dart
  import 'package:moengage_flutter/moengage_flutter.dart';
  final MoEngageFlutter _moengagePlugin = MoEngageFlutter();
  @override
  void initState() {
    super.initState();
    initPlatformState();
    _moengagePlugin.initialise();
  }
```

</CodeGroup>

For more information, refer to [Flutter SDK](https://github.com/moengage/Flutter-SDK).

Refer to the following for platform-specific initialization:

* [Android](https://developers.moengage.com/hc/en-us/articles/9496487053332)
