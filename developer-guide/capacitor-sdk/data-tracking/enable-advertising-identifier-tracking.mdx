---
title: "Enable Advertising Identifier Tracking"
---

For accurate analytics around devices and tracking Re-installs, it is essential to track the Advertising Identifier.

## Add Ad Identifier Library

Add the below dependency in the application-level **_build.gradle_** file.

<CodeGroup>

```groovy Groovy
implementation("com.google.android.gms:play-services-ads-identifier:18.0.1")
```

</CodeGroup>

To enable Advertising Identifier tracking use the _enableAdIdTracking()_ method as shown below.

<CodeGroup>

```javascript TypeScript
import { MoECapacitorCore } from 'capacitor-moengage-core'
MoECapacitorCore.enableAdIdTracking({ appId: "YOUR_WORKSPACE_ID" });
```

</CodeGroup>

Before you enable Advertising ID tracking please ensure the application is complying with the [Google Play Policy](https://support.google.com/googleplay/android-developer/answer/10144311) [Placeholder_Link] regarding Advertising ID tracking. Refer to our [help document](https://help.moengage.com/hc/en-us/articles/7457641031828) [Placeholder_Link] for more information on the policy.

In case, you need to disable advertising-id after enabling tracking use the following method.

<CodeGroup>

```javascript TypeScript
import { MoECapacitorCore } from 'capacitor-moengage-core'
MoECapacitorCore.disableAdIdTracking({ appId: "YOUR_WORKSPACE_ID" });
```

</CodeGroup>

<Info>
  The above APIs are available only starting plugin version 1.0.2. In the older versions, Advertising Identifier tracking is enabled by default.
</Info>
