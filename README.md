# MoEngage Documentation

The official MoEngage documentation. This repo contains SDK integration guides and API reference docs for MoEngage.

## Project Structure

```
├── api/                  # API reference docs (OpenAPI specs + overview pages)
├── developer-guide/      # SDK integration Docs
├── images/               # Documentation images and screenshots
├── logo/                 # Brand assets (favicon, light/dark logos)
├── snippets/             # Reusable content snippets
├── docs.json             # Site configuration
└── style.css             # Custom styles
```

## Developer Guide

SDK integration docs covering setup, data tracking, push notifications, in-app messages, cards, and more for:

- **iOS SDK** — Swift/Objective-C (CocoaPods, SPM)
- **Android SDK** — Kotlin/Java (Maven)
- **Web SDK** — JavaScript (npm, CDN, GTM)
- **React Native SDK** — React Native & Expo
- **Flutter SDK**
- **Cordova SDK**
- **Unity SDK**
- **Capacitor SDK**
- **Ionic SDK**
- **Personalize SDK** — Web personalization
- **Ecommerce Platforms** — Shopify, WooCommerce, Magento

Also includes release notes, migration guides, and partner integrations (Firebase, Segment, mParticle).

## API Reference

OpenAPI-powered reference docs for MoEngage REST APIs:

| Category | APIs |
|---|---|
| **Data** | User, Event, Bulk, Device, File Import, Tracking |
| **Campaigns** | Create, Update, Search, Test, Stats & Reports |
| **Content** | Templates (Push, Email, SMS, In-App, OSM), Content Blocks, Catalog, Coupons, Recommendations |
| **Segments** | File Segments, Filter Segments, Cohort Sync |
| **Messaging** | Push, Cards, Inform, Live Activities |
| **Subscriptions** | Email Subscription, Subscription Categories |
| **Personalize** | Experiences, Events, Metadata |
| **Compliance** | GDPR/CCPA, Message Archival |
| **Business Events** | Create, Trigger, Search |
