## Core

<span style={{ backgroundColor: '#e6fffb', color: '#008767', borderRadius: '12px', border: '1px solid #B5F5EC', fontSize: '0.9em', padding: '3px 10px' }}>**What's New:**</span>

**Custom Proxy Domain**: Introduced support for Custom Proxy Domains to route SDK traffic through a customer-owned subdomain for bypassing ad blockers and private DNS services. For more information, refer [here](/hc/en-us/articles/45199851580180).

<span style={{ backgroundColor: '#fffbe6', color: '#d46b08', borderRadius: '12px', border: '1px solid #FFE58F', fontSize: '0.9em', padding: '3px 10px' }}>**Improvements:**</span>

Fixed handling of KMM errors for CocoaPods integration.

## InApp

<span style={{ backgroundColor: '#e6fffb', color: '#008767', borderRadius: '12px', border: '1px solid #B5F5EC', fontSize: '0.9em', padding: '3px 10px' }}>**What's New:**</span>

* **Segment Re-evaluation:** Introduced trigger-time segment re-evaluation to ensure In-App messages are only shown to currently eligible users.
* **Frequency Capping:** Introduced frequency capping to limit the number of campaigns displayed to users.

## LiveActivity

<span style={{ backgroundColor: '#e6fffb', color: '#008767', borderRadius: '12px', border: '1px solid #B5F5EC', fontSize: '0.9em', padding: '3px 10px' }}>**What's New:**</span>

**Transactional Live Activity:** Added support for Transactional Live Activities. For more information, refer [here](/hc/en-us/articles/45243063940244).

## Messaging

<span style={{ backgroundColor: '#e6fffb', color: '#008767', borderRadius: '12px', border: '1px solid #B5F5EC', fontSize: '0.9em', padding: '3px 10px' }}>**What's New:**</span>

**Notification Click Tracking:** Added the new API `logNotificationClicked` that can accurately track clicks and dismiss of notifications.
