MoEngage has native integration for WhatsApp with Karix. This integration allows businesses to send templated WhatsApp messages using Senders powered by Karix. MoEngage can send messages, show Delivery and Read Analytics.

## Sender Configuration

To configure a Sender from Karix on the MoEngage Dashboard, do the following:

  1.      1.         1. Go to Settings->WhatsApp->Sender Configuration.
        2. Choose Karix from the left menu and click on \+ Sender![WA_IntegrationSetup.png](/images/partner_13548570593044.png)
        3. Add the following details:

| Field Name | Description

| --- | --- |  
| Sender name | This field denotes the name you want to provide to the Sender profile. This would be available in the Sender so that you can recognize this easily while using it to create a campaign inside MoEngage. The _Sender Name_ should be between 5-50 characters. This is a mandatory field.  
| WhatsApp business number | This field denotes the phone number registered with WhatsApp (directly or via Karix) using which you want to send out WhatsApp Messages to your users. On Karix, you can find Sender Details on _Channels & Numbers ->WhatsApp ->Senders. _This is a mandatory field and is also referred to as the WABA number.  
| API key |  [Login to your Karix account](https://www.unicel.in/lounge/sign_in.php) and go to the Power API section on the [Lounge Page](https://www.unicel.in/lounge/LoungePage/). Follow these steps to get the API key:
           1. Click _API KEYS_ in the Power API section.
           2. Click on the eye icon on the Manage API page to view the API key and copy it from there.
           3. Paste the API Key to the API key field in the Dashboard.  



## Delivery Tracking

To track the delivery of your WhatsApp Messages inside MoEngage, you would need to copy MoEngage Delivery Tracking URL and share it with Karix and get it updated on their side. You should be able to see the URL as shown below:

![DT_Karix.png](/images/partner_15976544979604.png)

| info |  **Information** The URL is a sample URL. Copy the exact URL from your MoEngage Dashboard before you share it with the Karix team.

| --- | --- |  
  
### **Steps to add Delivery Tracking URL**

  1.      1.         1. [Login to your Karix account](https://www.unicel.in/lounge/LoungePage/) and go to the [WhatsApp Campaign](https://help.moengage.com/hc/en-us/articles/4411175043220-Create-a-WhatsApp-Campaign) section.
        2. Navigate to the [Webhook Configuration](https://rcmui.instaalerts.zone/bot_builder_gui/#/whatsapp-campaign/webhook-configuration) section from the left menu.
        3. Select the WABA account and choose the Delivery Rule option.
        4. Paste the following URL in the URL field as shown below.![DeliveryCallback.png](/images/partner_13551268486676.png)



Steps for adding tracking URL for opt-out management

For opt-out management, add the delivery tracking URL to the following navigation:

  1.      1.         1. [Login to your Karix account](https://www.unicel.in/lounge/LoungePage/) and go to the [WhatsApp Campaign](https://help.moengage.com/hc/en-us/articles/4411175043220-Create-a-WhatsApp-Campaign) section.
        2. Navigate to the [Webhook Configuration](https://rcmui.instaalerts.zone/bot_builder_gui/#/whatsapp-campaign/webhook-configuration) section from the left menu.
        3. Select the WABA account and choose the _User Initiated (MO) Rules_ option.
        4. Paste the following URL in the URL field as shown below.![WA_APIURL.png](/images/partner_13550178658068.png)
