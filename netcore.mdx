MoEngage has native integration for WhatsApp with 8x8. This integration allows businesses to send templated WhatsApp messages using senders powered by 8x8. MoEngage can send messages and show Delivery and Read Analytics.  
  
## Sender Configuration


To configure a sender from 8x8 on the MoEngage Dashboard, perform the following steps:

  1. Go to **Settings** -> **Channels** -> **WhatsApp** -> **Sender configuration**.
  2. Click **\+ Add connector**. The WhatsApp connector pop-up window is displayed.  
![8x8 - 3.png](/images/partner_27662397720852.png)

  3. From the **Choose a connector** drop-down list, select 8x8 and click **Done**.

  4. Add the following details:  
![8x8 - 1.png](/images/partner_27641445388948.png)  

| Field | Description  
| --- | --- |  
| **Sender name** | This field denotes the name you want to provide to the sender profile. This would be available in the sender so that you can recognize this easily **while using it to create a campaign inside MoEngage**. The sender name should be between 5-50 characters. This is a mandatory field.  
| **WhatsApp business number** |  This field denotes the phone number registered with WhatsApp (directly or through 8x8) using which you want to send out WhatsApp Messages to your users. On 8x8 Connect, you can find your WhatsApp business numbers on the Left Menu -> Chat Apps -> Channels.  
| **API URL** | 
     1. Log in to the 8x8 Connect portal.
     2. From the left menu, go to the **API keys** section.
     3. Under instructions, click the sub-account of your choice to copy it.
     4. Replace "Your_Chosen_Sub-account" with the copied sub-account in the following URL:  
https://chatapps.8x8.com/api/v1/subaccounts/Your_Chosen_Subaccount/partners/moengage/wa  
| **Authorization** | From the same **API keys** section, you can scroll down and select an existing API key or create an API key specifically for MoEngage integration.  
  5. Click **Save**.  
Upon successful integration, you will get the following message.  
![SaveIntegration.png](/images/partner_27662320239252.png)



## Delivery Tracking

To track the delivery of your WhatsApp Messages inside MoEngage, you would need to copy MoEngage Delivery Tracking URL and share it with our support team for us to configure it for you. You should be able to see the URL as shown below:
