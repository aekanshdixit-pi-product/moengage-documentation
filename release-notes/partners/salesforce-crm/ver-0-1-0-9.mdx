This release addresses an issue encountered by users with the MoEngage User permission set, where the error “DML operation UPDATE not allowed on `moengage__Sync_Queue__c`” was occurring. The syncing process has now been enhanced to run under system mode, ensuring smooth DML operations and resolving the reported issue.

# Change Implemented

Due to users with the MoEngage User Permission set facing the error:

  * "DML operation UPDATE is not allowed on `moengage__Sync_Queue__c`" - We have now updated the process to perform DML operations in syncing batches under system mode.



## Impact

  * Resolves the issue for users with the MoEngage User permission set.
  * No additional changes or actions required from end users.
