<Update label="January 2026" description="New Enterprise Integrations & RCS in Flows">

### New Enterprise Integrations

MoEngage has expanded its ecosystem with new integrations across sales, support, and data teams:

- **Microsoft Dynamics 365 Sales**: Align sales and marketing by nurturing prospects based on pipeline stages
- **Microsoft Dynamics 365 Customer Service**: Automatically suppress promotional campaigns for users with open high-priority support tickets
- **Kafka**: Stream high-volume events directly into MoEngage in real-time
- **Adobe Experience Manager (AEM)**: Flow creative assets seamlessly from AEM into MoEngage campaigns
- **Amazon Connect**: Trigger retention flows or satisfaction surveys when a ticket closes
- **Adobe Experience Platform (AEP)** [Paid Add-on]: Sync MoEngage engagement data back to Adobe's CDP
- **Microsoft Dynamics 365 Sales (Stream)** [Paid Add-on]: Real-time intent signals for sales teams

### New Channel: RCS in Flows

MoEngage now supports Rich Communication Services (RCS) as a native channel within Flows:

- **Rich Campaign Creation**: Build campaigns directly on the flow canvas using custom templates and rich media
- **Advanced Personalization**: Use attributes and Locale settings for personalized messages
- **Testing & Optimization**: Validate with Personalized Previews and A/B Testing
- **Performance Tracking**: Access granular analytics including Read Rates, CTR, and Failure Breakdown
- **Actionable Flow Logic**: Branch journeys based on RCS events (Sent, Delivered, Read, Clicked)
- **Reliable Fallbacks**: Send fallback messages if RCS fails to deliver

</Update>

<Update label="December 2025" description="Computed Traits & Merlin AI Updates">

### Actionable Analytics

*Actionable Analytics* now lets you click specific data points to access an expanded action menu with **View sample users** and **Export users** options.

### Computed Traits

Create dynamic user attributes by applying calculations to existing event data:

- **Flexible computation methods**: Generate traits using Count, Aggregation, First/Last Value, or SQL
- **Automated updates**: Schedule computations to run one-time, weekly, or monthly
- **Seamless integration**: Use Computed Traits immediately in segmentation, personalization, and workflows

- **Predictive insights**: Predictive segments, recommendations, and best-time-to-send

- **Decisioning agents**: Offer and campaign decisioning agents
