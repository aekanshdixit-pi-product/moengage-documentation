## Overview

Email Template APIs enable marketers to define, reuse, update, and manage email templates created outside the MoEngage ecosystem with ease. The templates created and updated using these APIs are supported for the Custom HTML Editor (Froala Editor) and not the Drag and Drop Editor.

## Email Template V1 API Endpoints

- [Create Email Template](https://moengage.mintlify.app/api-reference/email-templates/create-email-template)
- [Update a Specific Email Template](https://moengage.mintlify.app/api-reference/email-templates/update-specific-template)
- [Bulk Create/Update Email Template](https://moengage.mintlify.app/api-reference/email-templates/bulk-createupdate-templates)
- [Get a Specific Email Template](https://moengage.mintlify.app/api-reference/email-templates/get-specific-template)
- [Get all Email Templates](https://moengage.mintlify.app/api-reference/email-templates/get-all-templates)
