After completing user authentication (2-step verification), you can select which workspace to log in to from the workspace selection screen. The screen lists all workspaces and environments you have access to, along with details to help you choose the right one.

<Tip>
  **Prerequisites**
  - Set up 2-step verification on your device. For more information, refer to [Set Up 2-Step Verification on Your Device](/user-guide/getting-started/initial-setup/set-up-2-step-verification-on-your-device).
  - Complete user authentication (Step 1 in [Log in to MoEngage](/user-guide/getting-started/initial-setup/log-in-to-moengage)) before the workspace selection screen appears.
</Tip>

# Workspace Selection Screen

The workspace selection screen shows the following details for each workspace you have access to:

| Detail | Description |
|---|---|
| Workspace Name | The name of the workspace. |
| Environment | Whether the workspace is a Live or Test environment. |
| User Role | Your role in that workspace. |
| Status | Whether the workspace is Active or Blocked. |
| Last Used | Indicates the workspace you most recently accessed. |
| Login Method | The authentication method configured for that workspace: password, Google Sign-In, or SSO. |

# Select and Log In to a Workspace

1. Complete user authentication (2-step verification). The workspace selection screen appears.
2. From the list, select the workspace you want to log in to.
3. Complete workspace authentication using the login method shown for that workspace:
   - **Password**: Enter your password and click **Login**.
   - **Google Sign-In**: Click **Sign in with Google** and select your Google account.
   - **SSO**: Click **Login using SSO** and authenticate with your organization's Identity Provider (IdP).

You are logged in to the selected workspace.

<Info>
  If a workspace shows a **Blocked** status, you cannot log in to it. Contact your Admin to restore access.
</Info>

# Where Workspace Selection Is Available

The workspace selection screen is available in three contexts:

- **MoEngage dashboard**: After user authentication, you can log in directly to your last used workspace or open the workspace selection screen to choose a different one. For more information, refer to [Log in to MoEngage](/user-guide/getting-started/initial-setup/log-in-to-moengage).
- **MCP connections**: The workspace selection screen always appears after user authentication. There is no option to skip to the last used workspace.
- **Mobile App connections**: The workspace selection screen always appears after user authentication. There is no option to skip to the last used workspace.

## Session Duration

MCP and Mobile App sessions have longer durations than dashboard sessions.

| Connection Type | Expiry | Idle Timeout |
|---|---|---|
| MCP | 30 days | 7 days |
| Mobile App | 180 days | None |

Dashboard sessions follow the automatic logout policy configured for your workspace. For more information, refer to [Automatic Logout](/user-guide/settings/account/security/automatic-logout).
