---
title: "Test Users"
description: "Mark users as test users in MoEngage to validate SDK integration and data tracking in near real time. Organize test groups and review device logs."
---

The _Test Users_ feature allows marketers to build and organize test groups and provide insights into the SDK or API logs available from a test device during SDK integration testing. Once a user is marked as a test user, their data will be processed in near real time to allow you to test your data tracking in Moengage. You can view the data tracked for test users in user profile and recent events.

You can mark up to 1000 users as test users.

# Steps

## Add new test users

1. On the sidebar menu in MoEngage, hover over **Test and Debug**. The Test and Debug menu appears.
2. Click **Test Users**. <img src="/images/Screenshot-2026-04-21-at-6.12.57-PM.png" alt="Screenshot 2026 04 21 At 6 12 57 PM" />
3. Click **\+ Test user** on the top right to mark users as test users. <img src="/images/moengage_38a5b9.png" alt="Screenshot_2022-03-07_at_9.24.55_AM.png" />
4. You can either mark all the users in the [custom segment](/user-guide/segment/segment-operations/custom-segments) as Test users or select one or more specific users. <img src="/images/moengage_a30341.png" alt="Screenshot_2022-03-07_at_9.25.47_AM.png" />
5. Select one of the options:
   - **Select custom segment**
     1. If you want to mark all users from a custom segment, click the **Select custom segment** card.
     2. Select your desired custom segment from the drop-down.
     3. You can also run a query to see the count of the users in the selected segment.
     4. Click **Mark as test** to mark these users as test users. Note: Anonymous users in the custom segment will not get added as test users since they do not have Client ID / UID. You can add anonymous users as test users via GAID. <img src="/images/moengage_ef8a5c.png" alt="Screenshot_2022-03-07_at_9.29.21_AM.png" />
   - **Select users**
     1. Click the **Select users** card if you want to mark some selected users as test users.
     2. You can select one of the user identifiers from the drop-down.
     3. Enter user detail or a comma-separated list in case of multiple users.
     4. Click **Mark as test** to mark these users as test users. Note:
        - When we add via **Client id**, the users are added as "Test User", thus their device ID is not shown in the Test Users dashboard ( where all test users are displayed)
        - When we add via **GAID**, the devices are added as "Test Device". On the test users' dashboard, both users and device details are shown. <img src="/images/moengage_0a374f.png" alt="Screenshot_2022-03-07_at_9.33.07_AM.png" />
6. Once you click **Mark as test**, users in the selected segments or selected users would be marked as Test users. And you will see the user details in the slide-out. <img src="/images/moengage_b4e286.png" alt="Screenshot_2022-03-07_at_9.41.31_AM.png" /> Note: If some of the users failed, you will see the failure details in the same slide-out. <img alt="Screenshot_2022-03-07_at_9.50.17_AM.png" src="/images/moengage_dbf4c5.png" />

## Mark test users from the user profile

If you are already on a user profile and want to mark that user as a Test user, click **Mark as test user** on the top right.

Note: Only registered users can be added as a "Test User" from the user profile page, not  anonymous users.

## Access test users

1. Go to **Test and Debug** at the bottom of your sidebar.
2. The page will show the details of all the test users.

![Screenshot\_2022-03-07\_at\_9.21.35\_AM.png](/images/moengage_95d2a6.png)

### Access user profile

You can click the MoEngage user ID to go to the user profile page.

### View recent events

1. Once you select one or more users, you will see an option (just below the search bar) to see the recent events for the selected users. Click _View recent events_.

Note: We can select up to 5 users and view recent events of those users.

![Screenshot\_2022-03-07\_at\_9.39.04\_AM.png](/images/moengage_077b19.png)

2. The slide-out will show the recent 10 events for the selected users. Note: In the slide-out, we show a maximum of 10 events( of the last 24 hours) for each user.

![Screenshot\_2022-03-07\_at\_9.40.08\_AM.png](/images/moengage_094910.png)
