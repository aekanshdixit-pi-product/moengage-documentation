---
title: "How can I import custom data to MoEngage?"
---

MoEngage has SDKs on all mobile platforms and also a Javascript SDK to capture user data. But not everything regarding a user can be captured at the SDK level, some additional user attributes can come in from your servers. We have provided [Data Import APIs](https://developers.moengage.com/hc/en-us/articles/4404674776724-Data-API) to feed any additional information to our servers. You can use these APIs to feed our system with User Attributes and also User Events.
