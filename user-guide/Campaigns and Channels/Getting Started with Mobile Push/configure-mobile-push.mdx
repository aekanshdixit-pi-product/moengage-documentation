To configure mobile push settings, navigate to **Settings** \> **Channels** \> **Push** \> **App Push**.

<Tip>
If [Portfolio](https://help.moengage.com/hc/en-us/articles/40394603054100) is enabled for your workspace, you must add at least one project to your MoEngage Portfolio before configuring Mobile Push settings. For more information on how to add a project, refer [here](https://help.moengage.com/hc/en-us/articles/40394603054100-Portfolio#h_01K30SJFY837Y71J156TXMXC6S).
</Tip>

# Project Selection

If your portfolio has multiple projects, a dropdown appears at the top of the App Push settings tab. Select the project you want to configure, and the related settings (such as the private key file) load automatically. For portfolios with a single project, all settings are applied to that project.

# Android (FCM)

To configure push notifications for Android devices you must authenticate FCM on MoEngage. To know more about configuring the fields under **FCM Authentication**, refer to [FCM Authentication.](https://developers.moengage.com/hc/en-us/articles/16909296490644-FCM-Authentication#h_01H9G1211H69JXMH5MVFMG7381)

# iOS (APNS)

## iOS Authentication

Apple Push Notification Service(APNS) is necessary for sending Push notifications to your iOS app. To configure push notifications for iOS devices, choose one of the following APNS iOS Authentication methods available, and configure their details according to your documentation.

1. [APNS Authentication Key](https://developers.moengage.com/hc/en-us/articles/8484447635348-APNS-Authentication-Key#apns-certificate--0-0)
2. [APNS Provider Certificate](https://developers.moengage.com/hc/en-us/articles/4403944011028-APNS-Certificate-PEM-file)

## Show iOS Impressions

Enable the *Show iOS Impressions* toggle to show the iOS impressions count on the Campaign Analytics and All Campaigns page. To enable iOS impressions count, additional integration with the iOS SDK is required. For more information, refer to the [Notification Service Extension](https://developers.moengage.com/hc/en-us/articles/43960004257428-iOS-Push-Integration-Tutorial) section.

<Info>
The *Show iOS Impressions* toggle is turned off by default and is enabled only when the iOS SDK integration is completed.
</Info>

# Huawei (Push AMP+)

You must configure the fields in Huawei to send push notifications to Huawei devices. To know more about this, refer to [Configuring Huawei Push on MoEngage](https://developers.moengage.com/hc/en-us/articles/4403644307732-Configuring-Huawei-Push-on-MoEngage).
