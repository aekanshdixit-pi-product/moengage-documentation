---
title: "How to Check if a User is Part of a Campaign Segment?"
sidebarTitle: "How to Check if a User is Part of a Campaign Segment?"
---

To check if a user is part of a campaign segment in MoEngage, follow these steps:

1. On the left navigation menu in the MoEngage dashboard, click **Segments**, and then click **Create segment**.
2. On the **Create Segment** page, use the same segment definition that was used in the campaign.
3. Specify the time range when the campaign was originally sent.
4. Add an additional query using **AND**. In the **AND** query, search for the user by user attributes such as email, user ID, or any other identifier.
5. Click **Show count**. If the results show the user profile, the user was part of the segment. Otherwise, the query will say 0 results.
