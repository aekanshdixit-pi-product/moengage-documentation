A campaign draft is a saved version of the campaign you created. MoEngage facilitates saving campaign drafts automatically and manually, as described in the following sections. This feature reduces the risk of losing work due to accidental navigation or browser crashes, ensuring your campaign creation process is seamless.

## Auto Save Campaign Drafts

This feature helps you save campaign drafts automatically without having to click **Save Draft** during campaign creation. Your progress will be saved automatically every 30 seconds, and also when you navigate away from the campaign editor, provided there are changes compared to the last saved version.

<img alt="AutoSave_CampaignDrafts.png" src="/images/moengage_4ede1e.png" />

<Info>
- If you are working on a draft and another user updates the same draft, you will see a message on top indicating that a newer version of the draft is available. The message will also prompt you to reload the page to get the updated changes.  
   <img alt="NewVersion.png" src="/images/moengage_8253e3.png" />
- Autosave will work only when the tab is active. While drafts are automatically saved, we recommend manual saves for critical changes or when experiencing unstable network connections.
- The Autosave feature is not supported for campaigns with the **Source** as *Imported*.
</Info>

## Where Can You See Your Drafts?

Campaign drafts are available on the All campaigns page in the **Drafts** section. To see your drafts, perform the following steps:

1. On the left navigation menu in the MoEngage dashboard, click **Engage**, and then click **Campaigns**.
2. On the All Campaigns page, click the **DRAFTS** tab.  
   Alternatively, you can view drafts by clicking **Engage** \> **Drafts** from the left navigation menu.  
    <img alt="campaigndraftsss.gif" src="/images/moengage_7f6995.png" />

The draft list contains rows of draft campaigns. You can see the following details on the **Drafts** page:

- **Name**: Specifies the campaign name.
- **Source:** Specifies if the draft was created in the current workspace or imported from another workspace. For more information on how to import campaigns, see [campaign migration](https://help.moengage.com/hc/en-us/articles/29259629034260).
- **Type**: Specifies the campaign type, for example, whether it is a *Push*, *Flow*, or *OSM*.
- **Last Saved On**: Specifies the date and time the draft was last modified.
- **Expires on**:  specifies the date the draft is scheduled to expire. This is set to 180 days after the last saved date and updates automatically each time the draft is modified.
- **Actions**: Clicking the ellipsis( <img alt="verticle ellipsis.png" src="/images/moengage_9ecaac.png" /> ) icon allows you to get the option to either edit or delete the specific draft campaign.

<img alt="DRAFFTSSS.png"  src="/images/moengage_e6c8d7.png"  />

## Who Can See Your Drafts?

Users with Manager, Admin, and Marketer roles can view, create, and manage drafts. For more information, refer to [Access Roles](https://help.moengage.com/hc/en-us/articles/210855743-Access-Roles#01G37YHFSY1TXHE7K4D7FHDBDH).
