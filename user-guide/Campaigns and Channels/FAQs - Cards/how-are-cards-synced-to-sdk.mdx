title: "How Are Cards Synced to SDK?"
sidebarTitle: "How Are Cards Synced to SDK?"
---

MoEngage Cards are synced to the SDK through the following process:

1. When the app initializes the MoEngage SDK, it connects with the MoEngage server.
2. The SDK sends a network request to the MoEngage server to fetch the latest cards.
3. The server responds with a payload containing the card data, which includes the content and metadata.
4. The SDK stores the fetched card data locally on the device.
5. The app then uses this locally stored data to display the cards to the user. The SDK may repeat this process to update the card data whenever the app is opened, or certain triggers occur (like app resumes or specific events).

This syncing mechanism ensures the app has the most recent card content in the inbox.
