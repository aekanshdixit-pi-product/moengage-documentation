# Overview

Every analysis you perform is transformed from being just informative to actionable through actionable analytics. The insights generated can be saved as custom segments, which allows you to take various actions on these segments.

<Info>
This feature is only available in the Product Analytics package of MoEngage Analytics. To enable it for your account, contact your MoEngage Customer Success Manager (CSM) or the Support team.
</Info>

On any of the chart views, click a data point (such as a bar on a bar chart or a point on a line chart) to open a menu with the following actions directly from the analysis:

- [View sample users](#h_01K99VQ174RP71Y8MJV68CYT25)
- [Create segment](#h_01K99VV2V55KZ0A3Y8CGHJ8867)
- [Create campaign](#h_01K99VV2V5Z9QTDA96SSQ8702Y)
- [Analyze users](#h_01K99VVD7SZMEM6WKDJPS1M58V)
- [Export users](#h_01K99VVWQ70N82E6G4CV93D4T7)

## View Sample Users

*View sample users* allows you to view a list of up to 50 sample users who are part of the selected data point, helping you validate the data. When you click **View sample users**, a window appears displaying the list of users. Click any user to view their **User Profile**.  
 <img alt="2025-11-03_18-20-33 (1).gif" src="/images/moengage_2d043c.png" />

## Create Segment

You can save the selected users as a custom segment to take action on them later. Follow these steps:

1. Click **Create segment**. The **Create segment for Analytics** dialog box appears.
2. In the **Segment name** box, enter a name.
3. Click **Create**.  
     
    <img alt="2025-11-03_15-38-35 (1).gif" src="/images/moengage_8df5b4.png" />

## Create Campaign

You can create a campaign directly for these users. Follow these steps:

1. Click **Create campaign**. The **Create a campaign for Analytics** dialog box appears.
2. In the **Segment name** box, enter a name.
3. Choose the **Campaign channel** and **Delivery type**.
4. Click **Create**. For more information on creating a campaign, refer to [Campaigns and Channels](https://help.moengage.com/hc/en-us/categories/360002707651-Campaigns-and-Channels).  
    <img alt="2025-11-03_16-09-29 (1).gif" src="/images/moengage_69de43.png" />

## Analyze Users

You can analyze the group of users further. Follow these steps:

1. Click **Analyze users**. The **Analyze users for Analytics** dialog box appears.
2. In the **Segment name** box, enter a name.
3. Choose your analysis type from the **Analysis types** list.
4. Click **Analyze**. For more information, refer [here](https://help.moengage.com/hc/en-us/sections/33114966894484-Analytics).  
    <img alt="2025-11-03_18-11-05 (1).gif" src="/images/moengage_f95792.png" />

## Export Users

This allows you to export the user data for the selected data point. For more information, refer [here](https://help.moengage.com/hc/en-us/articles/205424326-User-Data-Exports).
