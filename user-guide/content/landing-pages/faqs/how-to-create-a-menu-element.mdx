---
title: "How to Create a Menu Element"
---

Menu elements are currently not available in the editor. Follow the steps outlined to create menu elements on your landing page.

After choosing your desired template,

1. Access the editor by clicking the **+** icon on the left corner of the template.
2. Drag and drop the **Row** or **Column** element to the template.  
    <img alt="2024-11-22_09-54-25 (1).gif" src="/images/moengage_97f0b5.png" />
3. Drag and drop the **Text** element inside the row or column.  
    <img alt="2024-11-22_09-57-44 (1).gif" src="/images/moengage_27bf87.png" />
4. Rename the elements with the text of your choice.  
    <img alt="2024-11-22_09-59-55 (1).gif" src="/images/moengage_743659.png" />
5. To redirect upon clicking the menu, add a **URL** to the element.
   1. Click the text field, the toolbar opens.
   2. Click the link icon in the toolbar.
   3. In the **Link to** field, add the URL you want the user to be redirected to.  
       <img alt="2024-11-22_10-05-56 (1).gif" src="/images/moengage_4cc7a2.png" />

The menu element is now created. You can also add personalization to the menu element.  
For more information, refer to [Personalize](/user-guide/content/landing-pages/create/personalize-a-landing-page).
