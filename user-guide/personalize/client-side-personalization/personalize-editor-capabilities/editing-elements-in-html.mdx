---
title: "Editing elements in HTML"
---

The Edit HTML code enables you to edit the HTML code of the selected element for more advanced changes.

**To Edit an Element,**

- Click On the element you want to edit on your web page
- Select **Edit HTML code**_._ This will navigate you to the HTML code window of that particular element, where you can edit it in HTML.
