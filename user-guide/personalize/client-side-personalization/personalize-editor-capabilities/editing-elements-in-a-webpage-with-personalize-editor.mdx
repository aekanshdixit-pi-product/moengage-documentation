---
title: "Editing Elements in a Webpage with Personalize Editor"
---

You can **Edit elements** to customize the appearance, background color, alignment, and Call to action (CTA) action of the selected elements or buttons.

![edit element.png](/images/moengage_5c27f3.png)

<Tabs>
  <Tab title="Elements">
    You can edit the selected element in the following two sections:

    - **Appearance**
      |  |  |
      | --- | --- |
      | - **Text:** Add relevant text on the selected element. <br />- **Type Settings:** Customize the Font style, size, color, weight, and alignment of the text. <br />- **Background:** You can make the following edits to the background of the selected element:   <br />   - Change the background color.   <br />   - Embed an image URL.   <br />   - Upload a background image from your local storage. <br />- **Dimensions:** Set the Height, Width, and Border radius of the selected element in pixels or percentages. <br />- **Margin:** Set the Top, Bottom, Left, and Right margins for the selected element in pixels or percentages. <br />- **Padding:** Set the Top, Bottom, Left, and Right padding to generate space around the text inside the borders of the selected element. | <img alt="Text settings appearance.png" src="/images/moengage_0ea650.png" /> |
    - **Advanced**
      |  |  |
      | --- | --- |
      | You can also use custom scripts to edit the selected element.  <br />- **CSS Classes:** Enter the class name to apply the custom styles to the selected element. <br />- **Z index:** Enter the Z index value to change how the selected element is displayed when it overlaps with other elements. | <img alt="text settings advanced.png" src="/images/moengage_9bec74.png" /> |
  </Tab>
  <Tab title="Buttons">
    You can edit the selected button in the following three sections:

    - **Appearance**
      |  |  |
      | --- | --- |
      | - **Text:** Add a relevant button label. <br />- **Type Settings:** Customize the Font style, size, color, weight, and alignment of the button label. <br />- **Background:** You can make the following edits to the button background:   <br />   - Change the background color.   <br />   - Embed a URL to the button label that redirects the user to the relevant webpage.   <br />    - Upload a background button image from your local storage. <br />- **Dimensions:** Set the Height, Width, and Border radius of the button in pixels or percentages. <br />- **Margin:** Set the Top, Bottom, Left, and Right margins for the button in pixels or percentages. <br />- **Padding:** Set the Top, Bottom, Left, and Right padding to generate space around the label, and inside the borders of the button. | <img alt="button appearance.png" src="/images/moengage_bfdb0c.png" /> |
    - **Redirection**
      |  |  |
      | --- | --- |
      | - **Enter webpage URL:** Enter a URL that redirects the users to the relevant page when they click the button. | <img alt="button redirect.png" src="/images/moengage_3647b5.png" /> |
    - **Advanced**
      |  |  |
      | --- | --- |
      | You can also use custom scripts to edit the selected button.  <br />- **CSS Classes:** Enter the class name to apply  custom styles to the button. <br />- **Z index:** Enter the Z index value to change how the selected button is displayed when it overlaps with other elements. | <img alt="button advanced.png" src="/images/moengage_01f1e8.png" /> |
  </Tab>
</Tabs>
