---
title: "Editing Elements in a Webpage with Personalize Editor"
---

You can **Edit elements** to customize the appearance, background color, alignment, and Call to action (CTA) action of the selected elements or buttons.

<img
  src="/images/moengage_5c27f3.png"
  alt="edit element.png"
  style={{ width:"79%" }}
/>

<Tabs>
  <Tab title="Elements">
    You can edit the selected element in the following two sections:

    - **Appearance**
      |  |  |
      | --- | --- |
      | <ul><li>**Text:** Add relevant text on the selected element.</li><li>**Type Settings:** Customize the Font style, size, color, weight, and alignment of the text.</li><li>**Background:** You can make the following edits to the background of the selected element: <ul><li>Change the background color.</li><li>Embed an image URL.</li><li>Upload a background image from your local storage.</li></ul></li><li>**Dimensions:** Set the Height, Width, and Border radius of the selected element in pixels or percentages.</li><li>**Margin:** Set the Top, Bottom, Left, and Right margins for the selected element in pixels or percentages.</li><li>**Padding:** Set the Top, Bottom, Left, and Right padding to generate space around the text inside the borders of the selected element.</li></ul> | <br /><img alt="Text settings appearance.png" src="/images/moengage_0ea650.png" style={{ display:"inline-block",textAlign:"center",verticalAlign:"middle" }} /> |
    - **Advanced**
      |  |  |
      | --- | --- |
      | You can also use custom scripts to edit the selected element. <ul><li>**CSS Classes:** Enter the class name to apply the custom styles to the selected element.</li><li>**Z index:** Enter the Z index value to change how the selected element is displayed when it overlaps with other elements.</li></ul> | <br /> <img alt="text settings advanced.png" src="/images/moengage_9bec74.png" style={{ display:"inline-block",textAlign:"center",verticalAlign:"middle" }} /> |
  </Tab>
  <Tab title="Buttons">
    You can edit the selected button in the following three sections:

    - **Appearance**
      |  |  |
      | --- | --- |
      | <ul><li> **Text:** Add a relevant button label.</li><li> **Type Settings:** Customize the Font style, size, color, weight, and alignment of the button label. </li><li> **Background:** You can make the following edits to the button background:   <ul><li>Change the background color.   </li><li> Embed a URL to the button label that redirects the user to the relevant webpage. </li><li>Upload a background button image from your local storage. </li></ul></li><li> **Dimensions:** Set the Height, Width, and Border radius of the button in pixels or percentages. </li><li> **Margin:** Set the Top, Bottom, Left, and Right margins for the button in pixels or percentages. </li><li> **Padding:** Set the Top, Bottom, Left, and Right padding to generate space around the label, and inside the borders of the button. </li></ul>| <br /> <img alt="button appearance.png" src="/images/moengage_bfdb0c.png" style={{ display:"inline-block",verticalAlign:"middle" }} /> |
    - **Redirection**
      |  |  |
      | --- | --- |
      | <ul><li> **Enter webpage URL:** Enter a URL that redirects the users to the relevant page when they click the button.</li></ul> | <img alt="button redirect.png" src="/images/moengage_3647b5.png" style={{ display:"inline-block",textAlign:"center",verticalAlign:"middle" }} /> |
    - **Advanced**
      |  |  |
      | --- | --- |
      | You can also use custom scripts to edit the selected button.  <ul><li> **CSS Classes:** Enter the class name to apply custom styles to the button. </li><li> **Z index:** Enter the Z index value to change how the selected button is displayed when it overlaps with other elements.</li></ul> | <br /> <img alt="button advanced.png" src="/images/moengage_01f1e8.png" style={{ display:"inline-block",textAlign:"center",verticalAlign:"middle" }} /> |
  </Tab>
</Tabs>
