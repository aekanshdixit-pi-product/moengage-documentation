---
title: "Delete an element"
description: "Remove unwanted webpage elements during personalization setup using the delete element option in the MoEngage Personalize visual editor."
---

To Delete an element,

- Click on the element to be deleted.
- Select **Delete element**.

<Info>
When you delete an element on your web page, the element is removed from the DOM.
</Info>
