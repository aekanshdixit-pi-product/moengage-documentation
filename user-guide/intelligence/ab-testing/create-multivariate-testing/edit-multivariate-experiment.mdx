---
title: "Edit Multivariate Experiment"
---

<Warning>
  Editing distribution percentage for active campaigns provides unexpected results. MoEngage recommends creating a new campaign with a new desired distribution percentage.
</Warning>

You can edit multivariate experiments for the campaigns with the following status:

# Push

| Campaign Type | Status | Allowed edit options |
| --- | --- | --- |
| One-Time Push | Scheduled but not sent yet | <ul><li>Add or remove variations.</li><li>Edit variation message.</li><li>Edit the distribution percentage of variations.</li></ul> |
| Periodic Push | Scheduled and not even 1 instance is sent | <ul><li>Add or remove variations.</li><li>Edit variation message.</li><li>Edit the distribution percentage of variations.</li></ul> |
| Periodic Push | Scheduled but at least 1 instance is executed | <ul><li>Edit variation message but cannot add or remove a variation.</li><li>Edit the distribution percentage of variations.</li></ul> |
| Event-Triggered Push | Active or Paused | <ul><li>Edit variation message but cannot add or remove a variation.</li><li>Edit the distribution percentage of variations</li></ul> |

| Status | Allowed edit options |
