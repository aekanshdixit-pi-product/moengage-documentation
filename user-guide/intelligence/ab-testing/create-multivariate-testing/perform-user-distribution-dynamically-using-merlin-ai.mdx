# Procedure

Follow these steps:

1. In Step 2 of the campaign creation process, click <img alt="ABTestButton.png" src="/images/moengage_cd3b0c.png" /> .
2. Click New Variation to create the new variation.  
   OR  
   Click Copy Variation to copy the previously created variation.                                                              <img alt="A_B_Testing_Options.png"  src="/images/moengage_17342b.png"  />
3. Click <img alt="UserDistributionbutton.png" src="/images/moengage_f34040.png" /> .
4. Click Dynamically using Merlin AI.                                                      <img alt="UserDistribution_Sherpa.png"  src="/images/moengage_374bde.png"  />
5. Select the Metrics to be maximized.

   | Metric | Description |
   | --- | --- |
   | Email | |
   | Open Rate | Select to optimize the distribution across different variations based on the email open rate by each variant. |
   | Click rate | Select to optimize the distribution across different variations based on the click rate by each variant. |
   | Both | Select to optimize the distribution across different variations based on both email open rate and email click rate by each variant. |
   | In-App Message and Onsite Message | |
   | Conversion | Select to optimize the distribution across different variations based on the click-through rate of each variant. |
   | Engagement | Select to optimize the distribution across different variations based on the conversion rate of each variant. |
   | Revenue | Select to optimize the distribution across different variations based on the revenue generated by each variant. |
6. In Campaign duration, select 2 hours or 4 hours to set the Campaign duration for Email Campaigns.
7. Set the following:  

   | Field | Description |
   | --- | --- |
   | Objective (Optional) | Objective of experiment. **Note**: Applicable only to In-App Message, On-Site Message, and Web Personalization. |
   | Variation | Modify the variation name for the campaign variation. (Applicable only for In-App, On-Site Messaging, and Web Personalization). **Note**: You can create only up to five variations. |
   | Campaign control group | Set the percentage of users who will receive the campaign. **Note**: MoEngage does not guarantee receipt of campaigns the second time where: (1) Users belong to either global control group or campaign control group for any active campaign, (2) Users are qualified to receive the campaigns twice. |

   <img alt="DynamicAllocation.png" src="/images/moengage_692018.png" />
8. Click Done to save the campaign variations.  
   OR  
   Click Cancel to discard the campaign variations.

<Info>
Campaigns, where the user distribution is performed by Merlin AI, have the <img alt="Powered_by_Sherpa.png"  src="/images/moengage_d428d2.png"  /> icon in the Campaign Status on the All Campaigns page.
</Info>
