---
title: "Perform User Distribution Manually"
---

<Note>
  Available for all channels and campaigns.
</Note>

# Procedure

Follow these steps:

1. In Step 2 of the campaign creation process, click **\+ A/B test**.
2. Click New Variation to create the new variation.\
   OR\
   Click Copy Variation to copy the previously created variation.
3. Click **User Distribution**.
4. Click on 'Manual' in the User Distribution screen that pops up. <img src="/images/moengage_16ebdc.png" alt="UserDistribution_Manual.png" />
5. Set the following:
   | Field | Description |
   | --- | --- |
   | Objective (Optional) | Objective of experiment. <br />**Note**: Applicable only to In-App Message, On-Site Message, and Web Personalization. |
   | Variation | Modify the variation name for the campaign variation. (Applicable only for In-App, On-Site Messaging, and Web Personalization). <br />**Note**: You can create only up to five variations. |
   | Campaign control group | Set the percentage of users who will receive the campaign. <br />**Note**: MoEngage does not guarantee receipt of campaigns the second time where: <ul><li>Users belong to either the global control group or the campaign control group for any active campaign,</li><li>(2) Users are qualified to receive the campaigns twice.</li></ul> |
   <img src="/images/moengage_c36269.png" alt="SetDistribution.png" />
6. Click **Done** to save the campaign variations.\
   OR\
   Click **Cancel** to discard the campaign variations.
