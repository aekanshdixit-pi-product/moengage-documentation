# Introduction

With the preset workflows of On-site messages you can suggest products or items and automatically interact with users at various stages of their purchasing journey or when they show interest. These interaction points can include landing pages, pages visited, exit intent, and other parameters.

At MoEngage, creating an email campaign is a three-step process.

1. Target users
2. Content
3. Schedule and goals

You can personalize the campaign content with recommendations in the second step 'Content'. Let us create a recommendation based personalized OSM campaign.

<Warning>
Ensure that you have created the desired recommendations. You can use only active recommendations for campaign personalization. Learn how to [create and manage recommendations](https://help.moengage.com/hc/en-us/sections/36511334979860-Recommendations).
</Warning>

# Create OSM campaigns with Recommendations

1. Click **Create Campaign** and select the **On-site** channel. <img src="/images/moengage_38583a.png" />
2. Enter the basic campaign details and define the audience in step 1 'Target users' and click **Next**.
3. In step 2, on the **Content** tab, select a template from the available options. For the defined scope of this article, let us select a nudge **template** of 'Text based CTA'. <img src="/images/moengage_af6f05.png" />
4. Let's remove the existing text OSM template and personalize the value with recommendation to feature item that have very limited stock available. <img src="/images/moengage_447566.png" />
5. You can define change the background color or add a background image as suits your need.
6. On entering **'@'** in the text block, a personalization pop-up opens. Search and select the desired recommendation name under data personalization drop-down. <img src="/images/moengage_9b831e.png" />
7. Once selected, you need to update the Jinja code inside the Jinja code block as following:

   JINJA for item title

   ```auto
   {% if ProductSet.DemoOnlyFewLeft%}
   {% for product in ProductSet.DemoOnlyFewLeft[0:1]%}
   {{product.title}}
   {% endfor %}
   {% else %}
   MOE\_NOT\_SEND
   {% endif %}
   ```
8. In a similar fashion, let us personalize with image link of recommendation item as following

   JINJA for item image

   ```auto
   {% if ProductSet.DemoOnlyFewLeft%}
   {% for product in ProductSet.DemoOnlyFewLeft[0:1]%}
   {{product.image\_link}}
   {% endfor %}
   {% else %}
   MOE\_NOT\_SEND
   {% endif %}
   ```

   <img src="/images/moengage_a21133.png" />
9. You may also want to personalize with web page link for CTA as following

   JINJA for item image

   ```auto
   {% if ProductSet.DemoOnlyFewLeft%}
   {% for product in ProductSet.DemoOnlyFewLeft[0:1]%}
   {{product.link}}
   {% endfor %}
   {% else %}
   MOE\_NOT\_SEND
   {% endif %}
   ```

   <img src="/images/moengage_fd90be.png" />
10. We encourage you to preview the recommendations delivery in OSM by testing the campaign before publishing. Enter the page URL under test campaign input and click Test. <img src="/images/moengage_e93613.png" />
11. You will be able to preview the test results as following. <img src="/images/moengage_ed904c.png" />
12. After you complete the content setup with recommendations, click **Next** at the bottom and move to 'Schedule and Goals', which is the step 3 of campaign setup.
13. Set up the appropriate schedule and delivery controls and publish the campaign.
