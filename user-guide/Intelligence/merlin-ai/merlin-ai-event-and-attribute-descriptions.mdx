# Overview

Merlin AI can automatically generate descriptions for your custom events and attributes. These descriptions help you to quickly understand the purpose of an attribute, leading to more accurate and effective segmentation. Instead of dealing with confusing or similarly named attributes, you get clear, concise explanations directly within the segmentation UI.

# Enable AI-Generated Descriptions

To begin using AI-generated descriptions, an Admin or Manager must first enable the feature in your workspace settings. To enable, perform the following steps:

1. On the left navigation menu in the MoEngage dashboard, click **Settings** \> **Data** \> **Data Management**.
2. In the header section, click **AI Description**.  
    <img alt="Screenshot 2025-06-25 at 3.48.08 PM.png" src="/images/moengage_d92fbd.png" />   
   The **Merlin AI description settings** dialog box appears.
3. Turn the **MerlinAI description** toggle on.  
    <img alt="Screenshot 2025-06-25 at 3.50.09 PM.png" src="/images/moengage_d8b3e6.png" />
4. Click **Done**.

When enabled, Merlin AI begins generating descriptions for your existing and new custom attributes. This process may take up to 24 hours to complete for all relevant attributes. If you disable this setting, new descriptions will no longer be generated, but existing ones will remain intact.

<Info>
Standard attributes provided by MoEngage are not affected by AI-generated descriptions. For other custom events and attributes, Merlin AI generates a description based on the readable name and data type.
</Info>

# Use AI Descriptions in Segmentation

Once Merlin AI description is enabled for your workspace, the AI-generated descriptions appear directly in the attribute selection drop-down list in the sections below:

- Segment filters in Segmentations
- **Filter users** section in MoEngage Analytics
- **Target audience** section in campaign creation pages

When you select a custom attribute, you will see the following information:

<img alt="Screenshot 2025-06-25 at 3.29.02 PM.png" src="/images/moengage_f15558.png" />

- **Description:** A short, AI-generated explanation of the attribute's purpose.
- **Generated by Merlin AI tag:** Appears next to descriptions created by Merlin AI.

## Edit a Description

If you need to refine an AI-generated description or add more context, users with an **Admin** or **Manager** role can edit it. To edit:

1. Click the **edit** icon that appears.  
    <img alt="Screenshot 2025-06-25 at 3.29.46 PM.png"  src="/images/moengage_53a220.png"  />
2. Edit the text in the **Description** box.  
    <img alt="Screenshot 2025-06-25 at 3.30.53 PM.png"  src="/images/moengage_9fe6ac.png"  />
3. Click **Done**.

Editing a description will cause the "Generated by Merlin AI" tag and the feedback icons to be removed, indicating that it is now a user-modified description. The edited description is visible to all users across the workspace.  
 <img alt="Screenshot 2025-06-25 at 3.32.42 PM.png"  src="/images/moengage_9e00a2.png"  />

## Add a Manual Description

For custom attributes where Merlin AI did not generate a description (or for which a description was manually deleted), users with an Admin or Manager role can add one manually. To do so, perform the following steps:

1. Hover over the attribute that has no description.
2. Click **+ Add description**.  
    <img alt="Screenshot 2025-06-25 at 3.34.16 PM.png"  src="/images/moengage_6ada0f.png"  />
3. In the **Description** box, type your custom description.  
    <img alt="Screenshot 2025-06-25 at 3.35.32 PM.png"  src="/images/moengage_b2e276.png"  />
4. Click **Done**.

## Provide Feedback

Your feedback is valuable for improving the quality of AI-generated descriptions. For any description generated by Merlin AI, you can provide feedback using the thumbs-up or thumbs-down icons.

<img alt="Screenshot 2025-06-25 at 3.38.56 PM.png"  src="/images/moengage_77e311.png"  />

If you click the thumbs-down icon, a dialog box will appear where you can specify why the description was not helpful. This feedback is sent directly to our team, and it will help them improve the model.

<img alt="Screenshot 2025-06-25 at 3.40.35 PM.png"  src="/images/moengage_af62e4.png"  />

# FAQs

<Accordion title="Why don't all of my custom attributes have an AI-generated description?">

Merlin AI only generates descriptions in cases where it has high confidence in the accuracy and relevance of the output. If an attribute is new, has unclear naming conventions, or has not received recent data, a description may not be available. Admins and Managers can manually add descriptions for these attributes.

</Accordion>
<Accordion title="Who can enable and edit AI descriptions?">

Only users with **Admin** or **Manager** roles can enable the *AI Descriptions* feature from the settings page. These roles are also the only ones who can edit or manually add descriptions. All user roles can view the descriptions.

</Accordion>
<Accordion title="If I edit a description, will Merlin AI overwrite it later?">

No. Once a description has been edited or added manually by a user, Merlin AI will not change or overwrite it in the future.

</Accordion>
<Accordion title="Where will these descriptions appear?">

The descriptions and associated metadata (last received time, platform) will be visible in any user or event attribute drop-down list within the Target audience component. This includes creating segments on the Segmentation page, as well as in Analytics and Campaign creation flows.

</Accordion>
