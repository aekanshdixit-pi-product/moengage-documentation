# Procedure

Follow these steps:

1. In Step 2 of the campaign creation process, click <img alt="ABTestButton.png" src="/images/moengage_0a63d1.png" /> .
2. Click New Variation to create the new variation.  
   OR  
   Click Copy Variation to copy the previously created variation.
3. Click <img alt="UserDistributionbutton.png" src="/images/moengage_37280e.png" /> .
4. Click on 'Manual' in the User Distribution screen that pops up.  
    <img alt="UserDistribution_Manual.png" src="/images/moengage_16ebdc.png" />
5. Set the following:  

   | Field | Description |
   | --- | --- |
   | Objective (Optional) | Objective of experiment. **Note**: Applicable only to In-App Message, On-Site Message, and Web Personalization. |
   | Variation | Modify the variation name for the campaign variation. (Applicable only for In-App, On-Site Messaging, and Web Personalization). **Note**: You can create only up to five variations. |
   | Campaign control group | Set the percentage of users who will receive the campaign. **Note**: MoEngage does not guarantee receipt of campaigns the second time where: (1) Users belong to either the global control group or the campaign control group for any active campaign, (2) Users are qualified to receive the campaigns twice. |

     
    <img alt="SetDistribution.png" src="/images/moengage_c36269.png" />
6. Click Done to save the campaign variations.  
   OR  
   Click Cancel to discard the campaign variations.
