# Overview

Curating engaging email content can be a demanding and intricate task, particularly when it entails extensive research, analysis, and content refinement. That's where Merlin AI steps in - an advanced tool that utilizes the power of LLM to create top-notch written content for email marketing campaigns.

In this article, we'll walk you through the process of generating email subject line and preview text using Merlin AI. By the end of this article, you'll be proficient in creating captivating email subject line by simply providing the context or the AI-enhanced versions of your manually written emails subject line.

# Navigation

<Info>
All editors and templates of email content can access the Merlin AI copywriter from the user interface. <img src="/images/moengage_f9af72.png" />
</Info>

To access Merlin AI in the MoEngage dashboard, follow these steps:

1. Log in to your MoEngage account.
2. Click on the "Campaigns" tab on the left-hand side menu.
3. Click on the "Create Campaign" button at the top right corner of the screen.
4. Select any "Email" campaign type.
5. Set up your basic campaign details in the first step of creating the campaign e.g. target audience, target platforms, etc.
6. In the second step of 'Content', you will find a Merlin AI banner at the top <img src="/images/moengage_27af4c.png" />

From here, You can generate new Email subject line and preview text or swiftly create AI variants of your manually crafted subject line using Merlin AI copywriter, and execute A/B testing.

# Create New

This feature allows you to generate impactful Email content by providing the campaign context only.

1. On selecting the 'Create New' option, a prompt builder will open, enabling you to define your campaign intent and generate a relevant Email subject line along with preview text.
2. You must type the campaign use case to help Merlin AI learn the ultimate objective of your Email campaign, and get started. <img src="/images/moengage_4248d2.png" />
3. Additionally, you can provide more context by selecting the target audience, and intended content tone. You can also define a content style that can be inspired by trends, persona, pop-culture, or literature references, etc.
4. The high-performing keywords dropdown provides a sneak peek at the keywords that have performed well in the past. You can select up to three of these keywords to ensure that the generated content includes them. You can also type and select some new keywords to ensure their inclusion in the content.
5. Likewise, you can exclude historically low-performing keywords or specific keywords by selecting them in the "Exclude keywords" input. <img src="/images/moengage_f7338f.png" />
6. Sometimes you may need to customize the generated content with extra information. For example, the use case may include adding your coupon code that Merlin AI may not know. In such cases, enter the coupon code in the Add Coupon field in the Merlin AI prompt builder.
7. You can also add the user property personalization element from the Personalization dropdown. This will ensure the generated content positions the personalization element in a cohesive manner. <img src="/images/moengage_43f99a.png" />
8. You can re-generate content up to 20 times in a campaign using Merlin AI. If needed, you can go back to previously generated content by using the pagination button located at the bottom of the preview.
9. If you find a content that you like, click on "Select this" and the subject lines and preview text of the Email will be automatically filled with the generated content. <img src="/images/moengage_d52c44.png" />

**Additional Note**

- For each input type, there are example dropdowns available that you can select directly. On the right side of the dropdown, you will see an example content related to that specific input option. This can help you visualize the type of content that can be expected. <img src="/images/moengage_b02e14.png" />
- You can always select a custom value as input beyond the available options in the dropdown. Simply type the intended text and select it in custom placeholder values. <img src="/images/moengage_58cd46.png" />

# Create AI Variant

This feature allows you to generate a new version or a copy of the existing version that you can test against your manually created content using A/B testing. Here's how you can use this feature:

<Info>
To use this feature, you need to have at least one variant with a pre-defined content. The Quick A/B variant is generated by considering the reference of this pre-defined variant.
</Info>

1. To begin, input your manually defined content in the template.
2. When you select **+ Create AI variant**, it will briefly move to the processing step to generate an enhanced version of your reference content.  
    <img src="/images/moengage_63fbd2.png" />
3. Within seconds, a new variation will be created, and you will see the pre-filled AI variant of your previously defined content. <img src="/images/moengage_d36430.png" />
4. If you are unsatisfied with this generated content, click on the 'Resume' button to open the prompt builder and select the 'Existing campaign variant' option. If you have multiple variations in this campaign, you can choose which variations you want to consider as a reference for AI copy generation.
5. You can also provide additional contextual instructions, such as the target audience, impactful keywords, intended content tone, or even the content style. <img src="/images/moengage_46d0d8.png" />
6. You can use the Mobile preview option to view how the content will appear on a mobile device. <img src="/images/moengage_fa2da8.png" />

# Editing Generated Content

Sometimes you may need to edit generated content with extra information. The use case may include adding your coupon code that Merlin AI may not know etc.

Once you like a generated content, click 'Select this' on Merlin AI prompt builder. This action will auto-fill the generated content in the 'Content' tab of the campaign. You make any intended change to the content from the 'Content' tab.
