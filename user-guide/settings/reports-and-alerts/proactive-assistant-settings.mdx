# Overview

You can customize your RFM Insights and Key Insights preferences on the Proactive Assistant settings page. Additionally, you can adjust the Confidence Interval and Analysis Duration based on your needs. Configuring these settings helps populate accurate data on the **Proactive Assistant** page on the MoEngage dashboard. You can access the Proactive Assistant page by clicking **Assistant** in the header section.

# Setting Up Proactive Assistant

To access the Proactive Assistant settings page, navigate to **Settings** \> **Reports and alerts** \> **Proactive Assistant**. The **Proactive assistant settings** page appears.  
 <img alt="Screenshot 2025-04-28 at 2.11.03 PM.png" src="/images/moengage_eceb0d.png" />

You can configure the following settings:

- [Key insights](/user-guide/settings/reports-and-alerts/proactive-assistant-settings#key-insights)
- [RFM insights](/user-guide/settings/reports-and-alerts/proactive-assistant-settings#rfm-insights)
- [Retention insights](/user-guide/settings/reports-and-alerts/proactive-assistant-settings#retention-insights)

## Key Insights

The events and revenue attributes that you set up in the **Key Insights** section will be used for Key Metrics analysis. Proactive Assistant will provide relevant insight and alerts for any sizable movement of users for DAU and MAU Metrics. 
<img alt="Screenshot 2025-04-28 at 2.26.56 PM.png" src="/images/moengage_554542.png" />

Perform the following steps to set up Key Insights:

1. Specify the **App Open Event**, **Conversion Event**, and **Revenue Attribute** you would like to use for Key Insights.
2. Define the **Deviation interval.** The Deviation Interval is the range around the forecasted value within which the actual value is likely to fall for generating insights. This range can be between 5% to 20%.
3. Set the **Analysis duration** based on your needs.

## RFM Insights

The events and revenue attributes that you set up in the **RFM Insights** section will be used for RFM analysis. Merlin AI will provide relevant insight and alerts for any significant movement of users across the RFM segments.  
 <img alt="Screenshot 2025-04-28 at 2.19.58 PM.png" src="/images/moengage_2daba0.png" />

You can set up to five different RFM Metrics to be used as input to Merlin AI. Perform the following steps to set up RFM Insights:

1. By default, **Recency event** and **Frequency** **event** will be set up as *App/Site Opened.*
2. **Monetary event** will be picked from the **Settings** \> **General** \> **Goal mapping** \> **Revenue attribute** field.
3. These inputs will be analyzed over the duration added in **Analysis duration**. Set the analysis duration based on your needs. It is recommended to be above 7 days for meaningful insights.

## Retention Insights

The events and revenue attributes that you set up in the Retention Insights section will be used for Retention analysis. Merlin AI will provide relevant insight and alerts for anomalies around 7-day, 15-day, and 30-day retention.  
 <img alt="Screenshot 2025-04-28 at 2.41.41 PM.png" src="/images/moengage_f07ced.png" />

Perform the following steps to set up Retention Insights:

1. Specify the **First Event**, **Return Event**, and **Retention type** you would like to use for Retention insights.
2. Specify the **Deviation interval** and **Analysis duration** based on your needs.

Proactive Assistant will provide the relevant insight and alerts for any sizable movement of users for D7 and D30 Retention Metrics.
