You can manage your alerts from the Alert Management Dashboard. Any alert created is listed in the dashboard. To perform any action, such as edit, subscribe, duplicate, and so on, you would need the required access permission to do so. For more information, refer to [Access Permissions for Alert Management](https://help.moengage.com/hc/en-us/articles/210855743).

## Navigation

To access Alert management on the MoEngage dashboard, navigate to **Settings** **-\> Reports and Alerts -\> Alert management**.

The following tabs are available in the Dashboard:

1. **All alerts** - This section contains all the alerts created in your MoEngage account.
2. **System alerts** - This section contains the default alerts available as part of alert management. The system or default alerts are Campaign Expiry, Facebook Token Expiry, and APNS Token Expiry, and so on.
3. **Custom alerts** - This section contains custom alerts created on your request for your MoEngage account.

## Alert Information

The following information is available for all the alerts listed in all three tabs.

1. Alert name - This column contains the name of the alert.
2. Create alerts on - This column contains the alert type. For system alerts, the types are Campaign expiry, Facebook token expiry, APNS token expiry, Campaign Stats, Flows Stats, Events and User Attribute Usage, Events Volume and Active Users Volume.
3. Created by - This column specifies the email ID of the member who created the alert.
4. Subscribed by - This column specifies the email IDs of the members who've subscribed to the alert.
5. Subscription - This column specifies the subscription status of the logged-in user for the alert. For example, if you have subscribed to an alert, the status is <img alt="Subscirbed.png"  src="/images/moengage_7303b1.png"  /> and if you haven't subscribed to the alert, the status is <img alt="SubTo.png"  src="/images/moengage_9fa3a9.png"  /> . You can click on the *Subscribe to* link to subscribe to the alert.
6. Actions - Click on the <img alt="Actions.png"  src="/images/moengage_5b3a7f.png"  /> to view the available options. For more information, refer to [Actions that can be performed on alerts](#h_01HA1RKRDDJSGHM291ZTYCBF75).

## Actions That Can Be Performed on Alerts

You can perform the following actions on an alert by clicking the ellipsis icon <img alt="Actions.png"  src="/images/moengage_5b3a7f.png"  /> at the end of the row.

<img alt="2.png" src="/images/moengage_363a76.png" />

| Field | Description |
| --- | --- |
| View | This action lets you view the created alert. Any member with access to the Alert management dashboard can view the alert. |
| Edit | This action lets you edit the alert. You can edit the fields the following sections only:   - Alert condition - Filter by - Members   **Note**: You cannot edit the fields in the **General details** section.  You can also duplicate or delete the alert.  <img alt="9.gif" src="/images/moengage_eba429.png" /> |
| Duplicate | This action lets you duplicate the alert. You cannot duplicate system alerts. |
| Unsubscribe | This action lets you unsubscribe from receiving the alert. |
| Subscribe others | This action lets you add other members as subscribers to receive the alert. For example, if someone were to join your team and you wish to add them to the subscribers to receive alerts about certain events, you can do so with this option.  <img alt="3.gif" src="/images/moengage_8076da.png" /> |
| Delete | This action lets you delete the alert. Once deleted, the alert will not be available for viewing on the dashboard. |
