MoEngage offers MMS as an engagement channel, allowing you to reach out to users directly on their mobile devices and capitalize on the high open and read rates of text messaging. Perform the following steps to create an MMS campaign in MoEngage:

1. On the left navigation menu in the MoEngage UI, click **Engage**, and then click **Campaigns**.
2. On the All campaigns page, in the upper-right corner, click **\+ Create campaign**. The Create campaign page appears.

   ![MMS2.png](/images/moengage_9ce730.png)

   You can also click **\+ Create new** on the All campaigns page, and then click **Campaign**. The Campaign page appears.

   ![MMS1.png](/images/moengage_f8e6b3.png)
3. Under **Outbound**, click **MMS**.
4. Select one of the following campaign delivery types:
   - One Time
   - Periodic
   - Event Triggered
   - Business Event Triggered\
     For more information on delivery types, refer to [Delivery Types](/user-guide/campaigns-and-channels/getting-started/introduction/delivery-types-moengage-channels).

The MMS campaign creation page is displayed.

![MMS3.png](/images/moengage_ac7880.png)

# Steps to Create an MMS Campaign

The following are the steps to create MMS campaigns:

1. [Target users](/user-guide/campaigns-and-channels/sms-mms-and-rcs/mms/create-an-mms-campaign#step-1-target-users): Define the target audience for the campaign.
2. [Content](/user-guide/campaigns-and-channels/sms-mms-and-rcs/mms/create-an-mms-campaign#step-2-content): Define the content of the MMS campaign to be sent.
3. [Schedule and goals](/user-guide/campaigns-and-channels/sms-mms-and-rcs/mms/create-an-mms-campaign#step-3-schedule-and-goals): Define the campaign's delivery schedule and goals.

    3. Click **\+ Attributes** and add an event attribute with a filter condition.

### Control Groups

This section contains information about control groups for the campaign. A control group is a set of users added to an exclusion list. These users will not receive the marketing campaign. Control Groups are enabled only for promotional campaigns and are not applicable to transactional campaigns. You can enable the control groups to which you want to send the promotional campaign.

- A global control group is a list of users who will not receive any of the marketing campaigns.\
  Turn on the **Global control group** toggle to not send the campaign to all the users in the control group. 
- A campaign control group is created by combining the user base that qualifies for the campaign. Turn on the **Campaign control group** toggle to not send the particular campaign to the control group. \
  For more information, refer to [Control Group](/user-guide/campaigns-and-channels/getting-started/introduction/control-groups).

## Step 2: Content

This section describes how to create content for your MMS campaign.

1. In the **SMS/MMS provider** list, select the MMS connector (service provider).
2. In the **Message** field, enter the content to be displayed in the MMS. Ensure the message characters do not exceed 1600; otherwise, the system displays an error.
3. To personalize your message, enter @ in the **Message** field. The **MMS Personalization** pop-up window is displayed. For more information, refer to [Personalize MMS Campaigns](/user-guide/campaigns-and-channels/sms-mms-and-rcs/mms/personalize-mms-campaign).

   ![](/images/moengage_88a9ab.png)
4. In the **Add media** section, you can either upload a file or provide a media URL.
   - **Upload file**: Drag and drop a media file or upload it from your computer.
   - **Media URL**: Use this option to add a media file from a URL.

     You can add up to 10 media files, with supported formats including JPG, JPEG, PNG, GIF, MP4, and WEBM. The maximum file size is 5 MB.
5. Click **Add** to add the media file. All the uploaded media are displayed under the **Added media** section.

   ![](/images/moengage_ba7835.png)
6. You can verify the text and image preview for **MMS** and **SMS (fallback)** is available on the right side of the message editor.\
   The MMS campaign is delivered as an SMS along with a URL that contains the attached media in the following scenarios:
   - If you are sending MMS to a country where it is not supported.
   - If the recipient's device does not support MMS.

     ![](/images/moengage_d237f3.png)

1. Click the respective tabs for **MMS** or **SMS fallback** to preview the personalized content.
2. Select the **Preview type** as **Default** or **Personalized**.
3. Click the edit icon adjacent to the **Personalized Preview type**. You will be taken to the Personalized preview page. For more information, refer to [Personalized preview.](/user-guide/campaigns-and-channels/getting-started/message-personalization/personalized-preview)

### URL Addition and Shortening

Add the URL in the MMS message to track MMS clicks for the URL. MoEngage uses URL shortening services to shorten your URL and then track the users who click the URL.

Note the following points for URL Shortening:

1. Shortened links expire in 14 days and cannot be tracked after expiry. When a customer opens a link beyond its expiry, they see a "**This link is no longer active**" message.
2. If user and event attributes are used in the shortened links, the attributes' names must abide by the following:
   1. They must not contain spaces.
   2. If the attributes contain links, their values should not contain http:// or https://. These can be added before the attribute to shorten the link.
1. Special characters such as apostrophes and commas are not supported in shortened links.
2. Multiple JINJA operations are not supported in links that are to be shortened. For example, using a Content API link in a Product or vice versa (in a link that is to be shortened) is not supported.

### Add a Locale

Locale is a criterion based on which you can further sub-segment your target audience to send them a localized message. To add a locale to the campaign, click **\+ Locale** in the upper-right corner of the content creation step. You can add a new locale or add an existing locale to the campaign. For more information, refer to [Localize campaign message](/user-guide/campaigns-and-channels/getting-started/campaign-content/localize-campaign-messages).

### Add a Variation

Creating variations of campaigns allows you to determine the effect of a modified variable on other campaign variables. It also enables the comparison of performance between campaign variations. To add a variation to the campaign, click **\+ A/B test** in the upper-right corner of the content creation step. You can add a new variation or copy the existing variation. After adding a variation, you can see the variations as tabs on the top ribbon above the content editor, as shown below:\
<img alt="Variations_Added.png" src="/images/moengage_071c6d.png" /> You can specify the user distribution for each variation by clicking **User Distribution**, which is available in the upper-right corner. For more information, refer to [A/B Testing](/user-guide/intelligence/ab-testing/what-is-ab-or-multivariate-testing/overview).

  **When will event triggered campaigns be sent?**

  The campaign will be sent only when:

  - The event property/attribute being used to create these types of triggers (like the flight_time example) is in a valid date-time format. If it is not in a date-time format, the string should have data in one of the [supported formats](/user-guide/campaigns-and-channels/mobile-push/create/create-push-campaigns). If the selected attributes do not contain data in any of these formats, the campaign won't be sent to your users.
  - The date/time value from the attribute is greater than the time at which the trigger condition is fulfilled.
  - The date/time value from the attribute is less than 30 Days + the time at which the trigger condition is fulfilled. For example, if your trigger condition is fulfilled at date D, your notification will be sent only when the attribute value is less than D+30.
  - The intelligent delay optimization (**With Delay**) does not work for campaigns sent on relative timing.

- **Pause:** This option allows you to pause a campaign.
- **Archive**: This option allows you to archive a campaign. The archived campaigns can be unarchived.
