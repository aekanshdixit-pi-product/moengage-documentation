---
title: "Configure SMS, MMS & RCS General Settings"
---

This article describes how to configure the **General Settings** for the SMS, MMS & RCS channels.

1. On the left navigation menu in the MoEngage UI, go to **Settings** \> **Channels** \> **SMS, MMS & RCS** \> **General settings**[.](https://app.moengage.com/v3/#/settings/6/3)  

   The following option is available:

   | Field | Description |
   | --- | --- |
   | **User attribute that stores user's mobile numbers** | Select the attribute that stores the mobile number of users from the drop-down list. The standard attribute for storing the user's mobile number is *Mobile Number (Standard)* in MoEngage, and this is selected in the drop-down list by default. |
2. Click **Save** to save the configuration.
3. You can also set the mobile number format for the application. Click the **Set the mobile number format** link. You are taken to the Mobile number management page. For more information, refer to [Mobile Number Standardization](/user-guide/settings/advanced-settings/mobile-number-standardization).
