## Problem

The "Content upload failed" error occurs when uploading a video to the Nudge type In-app Messaging campaign.

<img  src="/images/moengage_604d49.png"  />

<img  src="/images/moengage_11b779.png"  />

## Explanation

The “Content upload failed” error is caused by a file name exception, specifically when the file name contains dots.

## Example

If the file name is UNLMTD\_Slot Machine\_25.02.2024.mp4, the dots in the name cause the upload to fail.

## Solution

1. Log in to the dashboard, hover over to the left, and select **+Create new** \> **Campaign** \> **In-app**.
2. Go to step 2, **Content** \> **Start from scratch** \> **Zip Upload**, and upload a sample video file.
3. Remove the dots from the file name and then attempt to upload it again.  
   <Info>
Check for any special characters before uploading files or assets.
</Info>
