# Overview

The View in Browser (VIB) feature in MoEngage allows your email recipients to open the email content in their web browser. This feature can be particularly useful if a recipient's email client has rendering issues or if they prefer to view the email in a web-based format.

# Use Cases

Including a VIB link is a best practice in email marketing to ensure your email is accessible and visually appealing to all recipients. By incorporating this link, you can effectively address the following cases:

- **Improved accessibility**: Provides an alternative way for recipients to view your email content, especially if their email client has difficulty rendering complex HTML or images.
- **Backup viewing option**: Ensures that your recipients can always access the full content of your email, even if their primary email rendering is incomplete or distorted.
- **Better interactivity**: For emails that incorporate advanced HTML, intricate CSS, or interactive elements that may not be fully supported by all email clients, the VIB link serves as a reliable fallback. It allows recipients to experience the full functionality and design of the email, enabling interactivity that might otherwise be broken.

# Add View in Browser (VIB) Link

<Info>
  The View in Browser Link position is visible in your email & can be modified only if the View in Browser feature is enabled. To enable it for your account, contact your MoEngage Customer Success Manager (CSM) or the Support team.
</Info>

In the MoEngage Email campaign module, the view in browser link is always positioned at the top of the email campaign, above the header by default. To position the VIB link at a custom position of your choice, you can follow the steps mentioned below based on your editor:

<Tabs>
  <Tab title="Drag and Drop Editor">
    Perform the following steps to insert the View in Browser (VIB) link directly in the drag and drop editor:

    1. Select the text in which you want to insert the VIB link.
    2. In the text editor, click **Special links** \> **Frequently used** \> **View in Browser Link**.

    ![VIB link.gif](/images/moengage_776194.png)
  </Tab>
  <Tab title="Custom HTML Editor">
     To customize the position of the View in Browser link using the custom HTML editor, place the following HTML tag in the required position.

     \<a id="moe\_email\_vib"\>Enter text here\</a\>

     The HTML tag can be added inside an HTML block in the drag and drop editor or directly in the HTML code editor and can be styled using CSS elements to customize the appearance, such as color and font.
  </Tab>
</Tabs>

<Info>
  - **Click tracking**: The VIB feature does not support click tracking. For example, if a user opens an email in their browser using the VIB link and then clicks on any links within that browser view, MoEngage does not record those clicks.
  - **Personalization:**
    - **Dynamic personalization**: The VIB page uses "in-moment" personalization. This means it displays content based on the user's data available at the time of opening the link, not when the email was originally sent. This may lead to different information in email vs the browser view of the same email.
    - **Event-triggered personalization**: For event-triggered campaigns, personalization based on event data does not work in the VIB view. This means, when a user opens the link, the event-specific personalization fails, and the system displays a fallback value if you have configured one. This may lead to different information in email (event data) vs the browser view (fallback data) of the same email. To guarantee that the VIB page exactly matches the content of the original email for all the personalization use cases, you can use the [Message Archival](/user-guide/campaigns-and-channels/campaign-management-and-reports/message-archival/message-archival) add-on. This feature stores an exact copy of the email at the time it was sent, ensuring perfect consistency between the email client and the browser view.
</Info>

# FAQs

<Accordion title="For how long does the VIB link remain active?">
  VIB links remain active indefinitely. Users can open emails via the VIB link regardless of when the email was sent or delivered.
</Accordion>

<Accordion title="Can I change the default top position of the VIB link in my email campaigns?">
  Yes, you can move the VIB link to a custom position. The method for changing the position depends on whether you are using the Drag and Drop Editor or the Custom HTML Editor. For more information, refer to [Add View in Browser (VIB) Link](#add-view-in-browser-vib-link).
</Accordion>

<Accordion title="How is the VIB link added when using the Drag and Drop editor?">
  By default, the VIB link is automatically added to the top of your email when the feature is enabled. You can also move the VIB link to a custom position. The method for changing its position depends on whether you use the Drag and Drop Editor or the Custom HTML Editor, as explained in this document.
</Accordion>

<Accordion title="Are VIB links added when I send a test email?">
  No, VIB links are not added to test campaigns.
</Accordion>
