---
title: "How to Clear the Opt-In Permission from the Website on Google Chrome?"
---

This article provides instructions to clear opt-in permission from the website on Google Chrome.

## Opt-In from an HTTP Site

1. Visit your HTTP site, right-click the browser and go to **Inspect** -\> **Application** -\> **Local Storage**. Clear everything present in the local storage.
2. Opt-in permission can only be provided on an HTTPS domain, so you must be using your own HTTPS domain or yoursitename.moengage.com for opt-ins. In a separate tab, visit this site and clear everything present in the local storage.
3. On the same site, click the lock icon (Secure) in the address bar and change Notifications to "Use global default". You will be prompted to reload the page, after entering the website.
4. Reload the page. You can close this website now.
5. Reload the HTTP page, you should be able to see the opt-in prompt now.

## Opt-In from an HTTPS Site

1. Visit your HTTPS site, right-click the browser and go to **Inspect** -\> **Application** -\> **Local Storage**. Clear everything present in the local storage.
2. Click the Lock icon (Secure) in the address bar and change Notifications to "Use global default". You will be prompted to reload the page, after entering the website.
3. Reload the page. You should be able to see an opt-in prompt now.
