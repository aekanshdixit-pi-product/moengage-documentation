---
title: "Using Locales in On-site Messages"
---

MoEngage's On-site campaigns let you target users in different languages and regions, helping you localize content, improve user engagement, and expand into new markets. You can tailor your messages to specific user groups based on their language, geography, or other characteristics.

You can add up to **256** Locales to your account and **21** Locales to each campaign (one of these 21 Locales is the default Locale).

For more details on adding Locales to your MoEngage account, see the [Localize Campaign Messages article](https://help.moengage.com/hc/en-us/articles/115003667986).

To set up and use Locales in MoEngage, follow these two main steps:

- **Step 1**: [Add New Locales to the MoEngage Account](#step1)
- **Step 2**: [Add Locales to On-site Campaigns](#step2)

# Add New Locales to the MoEngage Account

<Info>
To use Locales, select a **Drag & drop** template.
</Info>

To add a new Locale to your MoEngage account:

1. In the content creation step of your On-site campaign, click the <img alt="Locale button"  src="/images/moengage_3204ee.png"  /> **Locale** button.
2. From the dropdown list, click **+ New locale**.
3. The **Add new locale** dialog box appears with the following options:

   | Field | Description |
   | --- | --- |
   | Locale name | Type or paste the locale name you want to add. |
   | User property | Select the user property from the attribute dropdown. |
   | Filter | Click **+Filter** and enter the filter value for the Locale. For details, see the [Create Segments article](https://help.moengage.com/hc/en-us/articles/206169646). |
   | Exclude users | Click **Exclude Users** to remove users from the campaign. For details, see the [Preference Management article](https://help.moengage.com/hc/en-us/articles/360037357812-Preference-Management). |
4. Click **Add** to save the new Locale to your account, or click **Cancel** to close the dialog box without saving. <img alt="New locale added to list" src="/images/moengage_82ef8a.png" />

# Add Locales to On-site Campaigns

After you add Locales to your account, you can add them to an On-site campaign during creation or testing.

To add a Locale to an On-site campaign:

1. In the content creation step, click the <img alt="Locale button"  src="/images/moengage_3204ee.png"  /> **Locale** button.
2. From the Locale selector dropdown, select the Locales you want to add to the campaign.
3. Click **Add**.

   The **Choose content for newly added locales** dialog box appears. In this dialog box, you can choose to create unique content for each Locale or copy content from the default Locale.

   <img alt="Choose content for multiple locales dialog box" src="/images/moengage_367188.png" />
4. From the dropdown **Choose locale content from**, select one of the following options:
   - **Create new content**   
     This option adds the Locale to your campaign and lets you create content for it.
   - **Copy from locale Default**   
     This option adds the Locale and copies content from the default Locale to it.
5. Click **Done** to add the new Locale to the campaign, or click **Cancel** to close the dialog box without adding the Locale. <img alt="On-site campaign locales screenshot" src="/images/moengage_9bbe7e.png" />
