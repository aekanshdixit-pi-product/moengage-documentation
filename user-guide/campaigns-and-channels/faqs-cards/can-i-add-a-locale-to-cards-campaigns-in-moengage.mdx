---
title: "Can I Add a Locale to Cards Campaigns in MoEngage?"
sidebarTitle: "Can I Add a Locale to Cards Campaigns in MoEngage?"
---

Currently, the locale is not supported for Cards campaigns. However, the table below summarizes channels and their support for localization:

| Channel Type | Delivery Type | Supported |
| --- | --- | --- |
| Push | All + Flows | Yes |
| Email | All + Flows | Yes |
| SMS | No | No |
| In-app | All | Yes |
| On-site | No | No |

For more information, refer to [Localize Campaign Message](https://help.moengage.com/hc/en-us/articles/115003667986-Localize-campaign-message#h_01HHYDVT3VFTMCPXG5PBW5NE8D).
