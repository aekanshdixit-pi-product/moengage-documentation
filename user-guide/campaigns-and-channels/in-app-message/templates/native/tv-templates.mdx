TV templates deliver native In-App messages on connected TV apps. They are designed for remote-control navigation with a focus state that highlights the currently selected element.

<Tip>
  Ensure your TV app is integrated with the MoEngage SDK. For more information, refer to [TV and OTT integrations](/developer-guide/tv-and-ott-integrations/getting-started/tv-and-ott-integrations).
</Tip>

# Available TV templates

| Template name | Description |
| --- | --- |
| **TV Banner: Text and Image with 2 Buttons** | Background image and text, plus up to two buttons. Use to create banners. |
| **TV Pop-up: 3 Text with 3 Buttons** | Background image with up to three text fields and three buttons. Use to create pop-ups. |
| **TV Pop-up: 3 Text with 1 Button** | Background image with up to three text fields and one button. |
| **TV Pop-up: 3 Text with 2 Buttons** | Background image with up to three text fields and two buttons. |
| **TV Banner: Text with 1 Button** | Background image, text, and a button. |
| **TV Banner: Text with 2 Buttons** | Background image, text, and two buttons. |
| **TV Banner: Text and Image with 1 Button** | Background image, text, an image, and one button. |
| **TV Pop-up: 3 Text with No Buttons** | Background image with up to three text fields. |

# Create a TV In-App campaign

1. In **Step 1** of In-App campaign creation, in the **Target Platforms** section, select **TV** and choose at least one **TV OS Type**. The template selector displays all TV templates in **Step 2**.
2. Hover over a TV template and click **Select Template** to open the editor.
3. Configure the template using the editor sections below.
4. Preview the template. Click **View in TV preview** to see how the campaign appears on a TV screen. The TV preview does not show buttons in focus mode.
5. Test your campaign. For more information, refer to [Test your In-App campaign](/user-guide/campaigns-and-channels/in-app-message/create/test-your-in-app-campaign).
6. Schedule your campaign in **Step 3**, then publish.

<Info>
  - You cannot combine the **TV** platform with **Web** or **Mobile Web**. Select **TV** on its own.
  - The **On Specific Screen** trigger is not currently supported for TV In-App campaigns.
  - On Apple TV OS, certain delivery control settings may not work due to limited device storage.
  - You can add actions only to buttons in TV templates. Actions added to other widgets are ignored.
</Info>

# Edit buttons for the focus system

TV interfaces use a focus state to highlight the currently selected element. Users navigate with a remote control and click **OK** (or a similar button) to activate the focused element.

In the **Buttons** section of the editor, the **Focused** sub-section lets you customize how the button looks when in focus:

- **Font size**
- **Font color**
- **Button color**

Make the focused state visually distinct so users can clearly see the selected button on a large screen.

# Editor reference

## Template settings

| Section | Field | Description |
| --- | --- | --- |
| **Placement and Background** | **Phone Background Color** | Background color behind the message on the app screen. |
|  | **Template Background Image** | **Upload Image**, **Image URL**, or **Background Color**. |
| **Animation** | **Entry Animation** | Animation when the template first appears. |
|  | **Exit Animation** | Animation when the template is dismissed. |
| **Width and Margin** | **Width** | Width of the TV container. |
|  | **Margin** | Margin around the TV container (top, bottom, right, left). |
| **Border** | **Width**, **Radius**, **Border Color** | Border around the container. |

## Image element

| Section | Field | Description |
| --- | --- | --- |
| **Image Source** | **Upload Image** or **Image URL** | Choose the image to display. |
| **Margin & Padding** | **Margin** / **Padding** | Margins and padding for the image (top, bottom, right, left). |
| **Border** | **Width**, **Radius**, **Border Color** | Border for the image. |
|  | **Image Container Width (15% Max)** | Width of the image container, capped at 15%. |

## Text element

| Section | Field | Description |
| --- | --- | --- |
| **Text Content** | **Heading**, **Font Size**, **Font Color**, **Custom Font**, **Background Color** | Heading text and typography. |
| **Width, Margin & Padding** | **Width** / **Margin** / **Padding** | Width, margins, and padding for the text. |
| **Border** | **Width**, **Radius**, **Border Color** | Border for the text inside the TV container. |

## Buttons

| Section | Field | Description |
| --- | --- | --- |
| **Button Appearance** | **Button Type** | **Filled**, **Outlined**, or **Text**. |
|  | **Button Text** / **Font Size** / **Font Color** / **Custom Font** | Button label and typography. |
|  | **Button Color** | Button color (supports gradients). |
| **Focused State** | **Font Size** / **Font Color** / **Button Color** | Style applied when the button is focused via remote. |
| **Actions** | **Default** | Default action is **Dismiss**. Remove it via the **Delete** icon. |
|  | **+ Action** | Add a custom action from the **Custom** dropdown. |
| **Width, Margin & Padding** | — | Width, margins, and padding for the button. |
| **Border** | **Width**, **Radius**, **Border Color** | Border for the button inside the TV container. |
