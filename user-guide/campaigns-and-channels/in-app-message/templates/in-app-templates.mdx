MoEngage offers In-App templates across multiple editor types so you can build everything from a one-click pop-up to a fully custom HTML experience. This page helps you pick the right template for your use case.

# Choose by editor

The dashboard groups templates into five editors. Pick the editor that matches your team's skills and the level of customization you need.

| Editor | Best for | Reference |
| --- | --- | --- |
| **Native** | Quick pop-ups, full-screen messages, ratings, and native nudges using pre-built layouts. | [Native templates](/user-guide/campaigns-and-channels/in-app-message/templates/native/native-templates) |
| **HTML** | Surveys, lead gen, carousels, promotional pop-ups, and any custom layout you can build with HTML. | [HTML In-App templates](/user-guide/campaigns-and-channels/in-app-message/templates/html/html-in-app-templates) |
| **Drag and drop** | Building custom layouts visually without writing HTML, including forms. | [Drag and drop editor](/user-guide/campaigns-and-channels/in-app-message/templates/drag-and-drop-editor/drag-and-drop-editor) |
| **Self handled** | Rendering message content yourself while using MoEngage segmentation, triggers, and personalization. | [Self-handled In-App template](/user-guide/campaigns-and-channels/in-app-message/templates/self-handled-in-app-template) |
| **No-code gamification** | Spin the wheel, scratch cards, and countdown timers — fully customizable, no code or SDK update required. | [No-code gamification templates](/user-guide/campaigns-and-channels/in-app-message/templates/no-code-gamification-templates) |

# Choose by display style

Across editors, In-App messages render in one of three display styles. Use the **Filter By Type** menu in the template selector to narrow down templates by display style.

| Display style | Description | Use it for |
| --- | --- | --- |
| **Pop Up** | Appears in the center of the screen over a transparent overlay. | Promotions, focused announcements, ratings, surveys. |
| **Full Screen** | Occupies the entire screen and blocks the app. | Mandatory updates, onboarding walkthroughs, lead gen. |
| **Nudge** | Non-intrusive message anchored to the top or bottom of the screen. Users can keep interacting with the app. | Tips, soft promotions, alerts. Refer to [Native templates](/user-guide/campaigns-and-channels/in-app-message/templates/native/native-templates#nudge-templates) and [HTML nudge templates](/user-guide/campaigns-and-channels/in-app-message/templates/html/html-nudge-templates). |

# Specialized native templates

Some native templates are tailored to specific surfaces or use cases:

- [TV templates](/user-guide/campaigns-and-channels/in-app-message/templates/native/tv-templates) — banners and pop-ups designed for connected TV apps with focus-state navigation.
- [SMS Sign-Up templates](/user-guide/campaigns-and-channels/in-app-message/templates/native/sms-signup-templates) — collect SMS notification consent (North America region only).

# Save and reuse templates

After you customize a template, save it for future campaigns. For more information, refer to [Saving customized templates](/user-guide/campaigns-and-channels/in-app-message/templates/saving-customized-templates).
