---
title: "Using Locales in In-App Messages"
---

You can use locales in MoEngage to send In-App messages in multiple languages. Locales help you tailor your messages to different audiences, making your campaigns more effective.

Your MoEngage account can have up to 256 locales. For each In-App campaign, you can apply up to 11 locales, which includes one default locale.

For more information about adding locales for your account, see [Localize campaign message](/user-guide/campaigns-and-channels/getting-started/campaign-content/localize-campaign-messages).

# Add New Locales

To add a new locale to your account, follow these steps:

1. In the campaign creation flow, go to the **Content** step. Then, click the **Locale** button <img alt="Locale button icon" src="/images/moengage_085a44.png" /> . The locale selector list appears.
2. Click **+New locale**.
3. The **Add new locale** window appears with the following options:

| Field | Description |
| --- | --- |
| **Locale name** | Enter the locale name. |
| **User property** | Select a user property from the attribute dropdown list. |
| **Filter** | Click **+Filter** and enter the filter value for the locale. For more information, see [Create Segments](/user-guide/segment/segment-operations/create-segments). |
| **Exclude users** | Click **Exclude Users** to prevent specific users from receiving the campaign. For more information, see [Preference Management](/user-guide/campaigns-and-channels/mobile-push/getting-started-with-mobile-push/preference-management-for-push-notifications). |

4. Click **Add** to save the new locale to your MoEngage account, or click **Cancel** to discard it.

<img alt="Animation showing the process of adding a new locale by entering details in the 'Add New Locale' pop-up." src="/images/moengage_d9b3d0.png" />

Note: The locale you create here is added to your account's list of available locales, not directly to your campaign. To add it to your campaign, follow the steps in the next section.

<Info>
**Caching Policy for Locales**

There is a 3-hour server-side caching window for User Locale updates. As a result, if a user's locale attribute is updated, it may take up to 3 hours for the new locale to be reflected in In-App Messaging campaigns. Immediate content updates based on real-time locale changes may experience this delay.
</Info>

# Add Locales to a Campaign

To add existing locales to your In-App campaign, follow these steps:

1. In the campaign creation flow, go to the content step. Then, click the **Locale** button <img alt="Locale button icon" src="/images/moengage_085a44.png" /> .
2. From the locale selector list that appears, select the locales you want to add to your In-App campaign.
3. Click **Add**. The **Choose content for new added locales** window appears. Here, you can select whether to create new content for the added locales or copy content from the default locale.  
    <img alt="Screenshot of the Choose content for new added locales window." src="/images/moengage_43215d.png" />
4. From the dropdown list, select one of the following under **Choose locale content from**:
   - **Create new content**: This option adds the locale to your campaign and lets you create content for it.
   - **Copy from locale Default**: This option adds the locale and copies the content from the default locale to it.
5. Click **Done** to add the selected locales to your campaign, or click **Cancel** to close the window without adding them.

<img alt="Animation showing the process of adding locales to a campaign and choosing content options." src="/images/moengage_1f55bb.png" />
