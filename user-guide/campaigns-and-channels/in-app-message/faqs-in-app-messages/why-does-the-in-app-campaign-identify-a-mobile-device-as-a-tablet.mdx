title: "Why Does the In-App Campaign Identify a Mobile Device as a Tablet?"
description: "Understand why an in-app campaign may classify a mobile device as a tablet and how device type is detected."
---

## Problem

Some devices are incorrectly categorized as tablets, preventing in‑app campaigns from displaying on them. Devices include Google Pixel 6 Pro, Pixel 7, Pixel 7 Pro, and Samsung S21 FE.

## Solution

This usually occurs when the user changes the display size in the device settings; instruct the user to increase the display size from the settings menu.
