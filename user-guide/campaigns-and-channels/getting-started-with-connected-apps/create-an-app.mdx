# Introduction

The *Connected Apps* platform enables you to integrate MoEngage with your existing marketing technology (MarTech) tools. This integration creates new MoEngage channels for sending and receiving data, which helps you streamline your campaigns and customer interactions.

# Step 1: Create a Connected App

To create a connected app:

1. On the left navigation menu in the MoEngage dashboard, click **App marketplace**.
2. On the App marketplace page, click **+ Create app**. <img alt="Screenshot 2024-11-13 at 2.09.03 PM.png" src="/images/moengage_c50e01.png" />
3. On the **Create app** dialog box, enter the following details:

   | Field | Description |
   | --- | --- |
   | **App name** | Type the app name. |
   | **App type** | Click **Channel**. |

   <img alt="Screenshot 2024-11-13 at 2.07.07 PM.png" src="/images/moengage_5f333a.png" />
4. Click **Create**. This will add your connected app as a connected channel in MoEngage. You will then be directed to the connected app page.  
    <img alt="Screenshot 2024-11-14 at 9.35.46 PM.png" src="/images/moengage_a1b284.png" />

# Step 2: Add a Channel Connector

After creating a connected channel, you must define its use cases. These use cases help you define how MoEngage will run and analyze campaigns for the app.

Refer to the following examples to understand the use cases:

|  |  |
| --- | --- |
| **Channels** | **Connectors (Use Cases)** |
| Retargeting apps (such as TikTok, Snapchat, LinkedIn) | - Add users to the audience. - Remove users from the audience. |
| Messaging apps (such as Telegram, Line, Viber, WeChat) | - Send SMS to users. - Send MMS to users. - Send a message to a group. - Send a message to a community. |
| Support tools (such as Zendesk, Freshdesk) | - Create a new ticket. - Close a ticket. - Update an existing ticket. |

After you define the use cases, add these connectors to your app.

## Step 2.1: Create a Config File

Every connector needs a config to be used as a blueprint to instruct MoEngage to interact with the app. For more information on creating your config file for your connector, refer to the [Channel Connector Config Guide](https://help.moengage.com/hc/en-us/articles/33116671627924-Create-channel-connector-config).

## Step 2.2: Add a New Connector

After the config file is ready, perform the following steps to create a connector for your channel:

1. On the connected app page in App marketplace, click the **Configure** tab, and then click **+ Add configuration**.
2. In the **Upload config** section, upload the config file (.json) created in [step 2.1.](#h_01JCWRJE6BJ53ZX3PTBX63GYZ6) If the upload is successful, proceed to the next step. <img alt="Screenshot 2024-11-14 at 9.49.31 PM.png" src="/images/moengage_a8047c.png"  />
3. Click **Add**. <img alt="Screenshot 2024-11-14 at 9.58.30 PM.png" src="/images/moengage_615f2a.png" />

The channel connector is successfully added. It is ready to be integrated and used in campaigns.

<Info>
Click **+ Add configuration** to add multiple connectors.
</Info>

## Edit the Connector Config

If there are any issues with the earlier config, replace it with a new file.

1. In the **Upload config** section, click the ellipse icon ( <img alt="Screenshot 2024-11-14 at 10.07.30 PM.png"  src="/images/moengage_fd942b.png"  /> ).
2. Click **Edit**.
3. Click the cross icon.
4. Repeat the steps mentioned in [step 2.2](#h_01JCWQ7K35BJFV88AZVSCD4FBC) to add the new config. <img alt="Screenshot 2024-11-13 at 3.52.29 PM.png" src="/images/moengage_8111a5.png" />

<Warning>
Your existing campaigns will continue showing older fields and associated metrics. Changing your connectors while running campaigns is not recommended because this can lead to inconsistent behavior.
</Warning>

# Step 3: Integrate a Connected Channel

After adding a channel connector to your app, you must integrate your connected channel with MoEngage to use in your campaigns.

1. On the connected app page in App marketplace, click the **Integrate** tab, and then click **+ Add integration**.
2. In the **Connection details** section, enter the following details:

   | Term | Description |
   | --- | --- |
   | **Connection name** | It is the readable name that can be used to identify the connector while creating campaigns. |
   | **Additional fields** | These would be available only if you have defined any other input fields in your connector config. Enter the details in these fields accordingly. |

   <img alt="Screenshot 2024-11-14 at 10.51.07 PM.png"  src="/images/moengage_af625f.png"  />
3. Click **Connect**. You have successfully integrated your channel app with MoEngage.  
    <img  src="/images/moengage_553505.png"  />

<Info>
You can add multiple integrations by clicking **+ Add integration**.
</Info>

# Step 4: Run Campaigns

After integrating your connected channel with MoEngage, you can run MoEngage campaigns. Follow the steps here to [Create a Connected App Campaign](https://help.moengage.com/hc/en-us/articles/32006144498324-Create-a-Connected-App-Campaign) and analyze its performance.
