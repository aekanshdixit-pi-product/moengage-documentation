To view the alert info, perform the following steps:

1. On the sidebar menu in MoEngage, hover over the Inform menu item <img alt="Screenshot 2025-06-13 at 2.08.54 PM.png"  src="/images/moengage_8fe624.png"  /> . The **Inform** menu appears.
2. Click **All Alerts**.  
    <img alt="all alerts.png"  src="/images/moengage_c69a1a.png"  />
3. Click the desired alert and navigate to the **Info** tab of that alert.

This section contains information about the Alert, such as:

1. 1. Channels for which the Alert has been configured along with information about the default and fallback channels if the channel order is sequential sending.
   2. Request Details: <img src="/images/moengage_0e8a36.png" />
   3. Channel-level message previews  
       <img alt="Screenshot 2025-01-22 at 3.47.08 PM.png" src="/images/moengage_b6d29d.png" />   
       <img alt="AlertInfo" src="/images/moengage_e72c68.png" />

You can test the Alert on an external console using the code snippets in different languages, as shown below.

For more information, refer to [Test your Alert](/user-guide/inform/create/test-alert).

<Info>
- The Alert ID or Reference ID mentioned above in the code snippets must be used while sending Live Alerts. In case you use the Test Alert ID that is available during Alert creation, the request will fail.
- The logs get populated instantaneously when an API request is received, and you can view them in the Alert Logs tab.
</Info>
