## Problem

Unable to enter the flow even after performing the trigger event.

## Explanation

When using events or user attributes as filter conditions, the actions performed by the user must match the filters set by the marketer. Mismatched values or data types will result in incorrect evaluation.

## Example

1. For instance, the expected filters specify an *Add To Cart* event with five items in the cart.
2. The received filters specify an *Add To Cart* event with two items in the cart, which means only data entries for the items in the cart will be considered valid.
