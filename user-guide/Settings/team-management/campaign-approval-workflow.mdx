Campaign approval allows marketing teams to control/review the campaigns that are going live by putting in place a creator-approver (maker-checker) approach. Marketing teams can decide which users are allowed to create the campaigns and which users have the privilege to approve them.

## **<img alt="Revised-_Centered.png" src="/images/moengage_875542.png" />**

## Enable Campaign Approval Workflow for Campaigns

<Info>
**Permissions**: Any roles authorized to create, manage, and publish campaigns are available in the approver list. They can approve or reject the campaigns.

**Activation Time**: Once the toggle is turned on, it may take up to 5 minutes for the Campaign Approval features to become visible and active across the dashboard.
</Info>

1. Navigate to MoEngage Dashboard \> **Settings** \>  **Account** \> **Team management**. The Team management page is displayed.
2. On the **Members** tab, turn the **Approval Flow** toggle on to enable the campaign approval flow. The toggle is disabled by default.  
    <img alt="Screenshot 2024-08-22 at 2.36.50 PM.png" src="/images/moengage_998bfb.png" />   
   After the toggle is enabled, any roles authorized to create, manage, and publish will have approver rights. Only on approval, campaigns will become live.

## Assign Roles

#### Approver Role

Any roles authorized to create, manage, and publish will have approver rights when the **Approval Flow** toggle is turned on.

#### Marketer Role

Campaigns created by Marketers must be sent for approval.

Because the Marketer role is new, you need to add users to the Marketer role or change roles for existing users.

## Send a Campaign for Approval

After the **Approval Flow** toggle is turned on, marketers can create a campaign and send it to a list of approvers.

<img alt="Screenshot_2022-05-26_at_2.07.43_PM.png" src="/images/moengage_a8cc07.png" />

## Approved/Rejected

When the marketer clicks **Send For Approval**, all the assigned approvers will get the email for reviewing the campaign. Approvers can click the campaign link in the email and they will get redirected to the specified campaign.

<img alt="Screenshot_2022-05-26_at_3.45.41_PM.png" src="/images/moengage_133c5d.png" />

Now, they can review the campaign and choose to reject or approve it before the campaign sent time is passed.

- If *approved*, the campaign will become Live.
- If *rejected*, the marketers can edit the campaigns as per the Approver's comments and send them again for approval.

Any roles authorized to create, manage, and publish can also choose to edit the campaigns *Under Review*and make them Live with all the necessary changes.

<Info>
- Marketers cannot edit the campaigns when campaigns are *Under Review*.
- Only the approvers who were asked to review the campaigns can Approve/Reject them.
- The *Rejected* campaigns cannot be edited.
- Under the campaigns section, any custom role (excluding the Marketer role) that lacks Create, manage, and publish permissions will be required to send their campaigns for approval.
</Info>

function showTab(tabId) {
// Hide all tabs
const tabs = document.querySelectorAll('.tab');
tabs.forEach(tab =\> tab.classList.add('is-hidden'));
// Deactivate all tab links
const tabLinks = document.querySelectorAll('.tabs-link');
tabLinks.forEach(link =\> link.classList.remove('is-active'));
// Show the selected tab
const selectedTab = document.getElementById(tabId);
if (selectedTab) {
selectedTab.classList.remove('is-hidden');
}
// Activate the corresponding tab link
const activeLink = document.querySelector(`.tabs-link[onclick="showTab('${tabId}')"]`);
if (activeLink) {
activeLink.classList.add('is-active');
}
}
// Set initial active tab on load
document.addEventListener('DOMContentLoaded', () =\> {
showTab('audienceSyncTab'); // Default to Audience Sync and Imports tab
});
