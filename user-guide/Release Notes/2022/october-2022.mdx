# New Features

## Segmentation

The following features have been added:

### Nested AND/OR query support

This feature helps you create complex queries with the help of conjunction operators (AND/OR) with multiple levels of nesting to create granular user segments. For more information, refer to [Combining Segments](https://help.moengage.com/hc/en-us/articles/206169646#01G2PMG9GWKDDA4BCKXTN16ATB).

### Aggregation functions support

This feature helps you create user segments with aggregate functions like Minimum, Maximum, Average, Sum, and Median on the numeric attribute to filter the users. Also, you can now perform absolute or relative percentage comparisons on the numeric Attributes. For more information, refer to [Aggregation Operators](https://help.moengage.com/hc/en-us/articles/9722166234004).

### Last N Hour/Day/Week/Month

This feature helps you filter based on the Last N Hours/Days/Weeks/Months while creating segments. For more information, refer to [Date Time Filters](https://help.moengage.com/hc/en-us/articles/360046335412-Filters-in-segmentation#date-time-filters-0-6).
