# Merlin AI Studio

MoEngage has introduced the Merlin AI Studio, consolidating all Generative AI features (such as Copywriter, Designer, and Segmentation Assist) into one intuitive place. The Studio now includes all your AI agents and a new inspiration library with templates to simplify your workflows and quickly adopt new AI capabilities.

For more information, refer to [Merlin AI Studio](https://help.moengage.com/hc/en-us/articles/42710264728084-Merlin-AI-Studio).

# Native MMS Campaigns

MoEngage has now launched a channel, MMS (Multimedia Messaging Service), to help you move beyond text-only SMS (Short Message Service). You can now send visually engaging campaigns with images, GIFs, and videos directly to your customers' inboxes.

Key Highlights:

- **Rich media**: Upload up to 10 files (images, GIFs, videos) per message.
- **Expanded text**: Enjoy a 1600-character limit for detailed storytelling.
- **Smart delivery:** Includes automatic SMS fallback for unsupported devices.
- **Availability**: Now live for the US, Canada, and Australia (via Twilio).

For more information, refer to [MMS](https://help.moengage.com/hc/en-us/sections/42711562089108-MMS).

# Message Queuing

MoEngage has now launched a feature, Message Queuing, to help you maximize campaign reach and maintain timely communication. You can now automatically queue and retry messages blocked by campaign delivery settings, ensuring delivery as soon as the user is eligible.

Key Highlights:

- **Automated retries**: Messages blocked by Frequency Capping (FC), Minimum Delay, or DND are temporarily queued instead of dropped.
- **Maximized delivery**: MoEngage automatically attempts to resend the message as soon as the user becomes eligible.
- **Configurable duration**: Set the queue duration for up to 48 hours at the campaign level.
- **Availability**: Now live for Push campaigns (one-time, periodic, and business event-triggered).

For more information, refer to [Message Queuing](https://help.moengage.com/hc/en-us/articles/41888436184340-Message-Queuing#h_01K6W4BNT57QCXQEH0H261SNFV).

# OAuth 2.0 in Content and PII APIs

MoEngage has now launched an upgrade to enforce the highest security standards, helping you move beyond static API keys. You can now use industry-standard OAuth 2.0 authentication for accessing both the Content API and the Personally Identifiable Information (PII) Tokenization API.

Key Highlights:

- **OAuth 2.0 authentication**: Now the standard for accessing Content and PII APIs, enforcing the highest industry standard for machine-to-machine security.
- **Tokenization security**: Access to PII is restricted to applications with valid, short-lived tokens, ensuring a strict chain of custody for private data.
- **Automated token management**: Tokens can be programmatically refreshed and revoked, perfect for modern, secure CI/CD pipelines.
- **Business impact**: Enables Zero Risk, Total Compliance (for example, meeting GDPR/CCPA), provides clear audit trails, and boosts Developer Velocity.

For more information, refer to:

- [Configuring OAuth 2.0](https://help.moengage.com/hc/en-us/articles/23949234646292-Authorization-Configuration-with-OAuth-2-0)
- [Usage of OAuth 2.0 in Content API](https://help.moengage.com/hc/en-us/articles/36580648645780-Add-a-Content-API)
- [Usage of OAuth 2.0 in PII](https://help.moengage.com/hc/en-us/articles/9111881294996-PII-Tokenized-Sending#h_01JZ4RDXNZ719BY7W2BDHBVCRN)

# Expanded Language Support

Expanded Language Support includes the introduction of the full User Guide in Japanese and the addition of Brazilian Portuguese to the dashboard interface.

This update significantly improves accessibility and inclusivity for our global customers. By offering key documentation and dashboard navigation in local languages, users in these markets can utilize the platform more intuitively. This empowers your international teams to navigate the platform with ease and execute campaigns with greater confidence.
