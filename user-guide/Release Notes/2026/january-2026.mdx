# New Enterprise Integrations

MoEngage has now expanded its ecosystem with the following new integrations across sales, support, and data teams to provide a unified source of truth and faster campaign execution:

- **Microsoft Dynamics 365 Sales**: Align sales and marketing by nurturing prospects based on pipeline stages and providing sales teams with visibility into marketing engagement. For more information, refer to [Microsoft Dynamics 365 Sales](https://partners.moengage.com/hc/en-us/articles/42889850136980-Microsoft-Dynamics-365-Sales#h_01K3X7EA243ST3FTN7NG8FABAR).
- **Microsoft Dynamics 365 Customer Service**: Protect brand reputation by automatically suppressing promotional campaigns for users with open high-priority support tickets. For more information, refer to [Microsoft Dynamics 365 Customer Service](https://partners.moengage.com/hc/en-us/articles/43266523403796-Microsoft-Dynamics-365-Customer-Service#h_01K3X7BMQ0VSCXM76503THS9SP).
- **Kafka**: Stream high-volume events directly into MoEngage in real-time to trigger immediate, high-reliability campaigns. For more information, refer to [Kafka](https://partners.moengage.com/hc/en-us/articles/44408798821140-Kafka-to-MoEngage#h_01KAFKA023).
- **Adobe Experience Manager (AEM)**: Streamline content velocity by flowing creative assets seamlessly from AEM into MoEngage campaigns. For more information, refer to [Adobe Experience Manager](https://partners.moengage.com/hc/en-us/articles/44122961628308-Adobe-Experience-Manager#h_01KAEM002).
- **Amazon Connect**: Convert support signals into revenue by triggering retention flows or satisfaction surveys the moment a ticket closes. For more information, refer to [Amazon Connect](https://partners.moengage.com/hc/en-us/articles/42906473039892-Amazon-Connect#h_01K94QEQ9332EXQXRVPMH4G6BD).
- **Adobe Experience Platform (AEP) [Paid Add-on]**: Sync MoEngage engagement data (clicks, opens) back to Adobe’s CDP to power real-time personalization. For more information, refer to [Send MoEngage events to Adobe](https://partners.moengage.com/hc/en-us/articles/44426624434580-Send-MoEngage-events-to-Adobe#h_01KAEP002).
- **Microsoft Dynamics 365 Sales (Stream) [Paid Add-on]**: Empower sales teams with real-time intent signals by updating them the moment a lead interacts with a marketing campaign. For more  information, refer to [Stream Events to Microsoft D365 Sales](https://partners.moengage.com/hc/en-us/articles/43270361466516-Stream-Events-to-Microsoft-D365-Sales#h_01K3X7BMQ0VSCXM76503THS9SP).

# New Channel: RCS in Flows

MoEngage now supports Rich Communication Services (RCS) as a native channel within Flows, allowing you to deliver interactive, branded messaging experiences to Android users.

**Key Capabilities:**

- **Rich Campaign Creation**: Build campaigns directly on the flow canvas using custom templates, rich media, and interactive suggestions.
- **Advanced Personalization**: Use attributes like {{Name}} to personalize content and Locale settings to deliver messages in the user's preferred language.
- **Testing & Optimization**: Validate messages with Personalized Previews and optimize performance using built-in A/B Testing.
- **Performance Tracking**: Access granular analytics, including unique metrics like Read Rates, Click-Through Rates (CTR), and a detailed Failure Breakdown for troubleshooting.
- **Actionable Flow Logic**: Branch your customer journeys based on real-time RCS events: Sent, Delivered, Read, or Clicked.
- **Reliable Fallbacks**: Ensure reach by using subsequent flow nodes to send fallback messages (like SMS) if an RCS message fails to deliver.

For more information, refer to [Add Actions to Flows](https://help.moengage.com/hc/en-us/articles/5663599009300-Add-Actions-to-Flows#h_01HNJ5J8XDWT5BVCRX238J5JZZ).
