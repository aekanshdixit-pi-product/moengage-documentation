# New Features

## User Path Analysis

User path analysis shows the path traversed by users within your app or website. The traversed path is in terms of events and event sequences. The analysis provides action users are performing before or after a specific event.

For example, what users do after they open the app or install the app, or which path users take before they purchase or uninstall the app.

For more information, refer to [User Path Analysis](https://help.moengage.com/hc/en-us/articles/36048677165332-User-Paths-Overview "https://help.moengage.com/hc/en-us/articles/360045026991-User-Path-Analysis").

## Chart KPIs

Key Performance Indicators (KPIs) provide a quick analysis of chart reports. The chart KPIs are available for [Behavior](https://help.moengage.com/hc/en-us/articles/34174962989204-Behavior-Overview "https://help.moengage.com/hc/en-us/articles/360019719852-User-Behavior-Analysis"), [Funnel](https://\https://help.moengage.com/hc/en-us/articles/34174988942740-Funnels-Overview "https://help.moengage.com/hc/en-us/articles/360019719252-Funnels-Analysis"), [Cohort](https://help.moengage.com/hc/en-us/articles/35438085954196-Retention-Overview "https://help.moengage.com/hc/en-us/articles/360021418732-Retention-Analysis"), and [Custom Dashboard](https://help.moengage.com/hc/en-us/articles/360027636312-Custom-Dashboards- "https://help.moengage.com/hc/en-us/articles/360027636312-Custom-Dashboards-").

## Queries in App's Time Zone for Analytics

MoEngage Analytics used to query your data in UTC/GMT.  Analytics queries can now run based on the App time zone. The app time zone is configured in App Settings in MoEngage Dashboard.

## Data Center in EU for Analytics

The analytics module is available for the EU region along with the US and India regions. MoEngage Analytics module complies with GDPR.
