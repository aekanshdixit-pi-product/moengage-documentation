# Analytics

## Pinning analysis tables to custom dashboards

You can pin tables from Behaviour, Funnels, Retention, and Session and Source analysis on custom dashboards. This helps in analyzing scenarios where all the data points are not visible on the chart or when there are too many data points at once.

## Transpose tables on custom dashboards

You can transpose tables on custom dashboards as well as on Behaviour, Funnels, Retention, and Session and Source Pages! This helps you visualize the tables better.

For more information, refer to [Pin Table to a Dashboard](/user-guide/analyze/dashboards/custom-dashboards) and [Transpose Tables.](/user-guide/analyze/dashboards/custom-dashboards)

## Custom segment analysis using Actionable Analytics

You can perform custom segment analysis using Actionable Analytics in a click and switch from one analysis type to another. You can now analyze segments created on Behaviour, Funnels, and Retention on any product in the analytics suite directly from the modal. For more information, refer to [Actions on Insights](https://help.moengage.com/hc/en-us/sections/34174954726548-Behavior).
