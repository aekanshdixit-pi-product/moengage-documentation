This version improves the sync from MoEngage to Salesforce, enabling multiple mappings to function together. It also includes enhancements to error logs.

# What's New?

The following are the new features and improvements introduced in this version:

### Support for multiple mappings for MoEngage to Salesforce sync

  * Enhanced the synchronization functionality from MoEngage to Salesforce to partially save successful records. This enables support for multiple mappings to be active on Salesforce.
  * For failed records, a corresponding exception record will now be created.



### Detailed error messages for record failures

During MoEngage to Salesforce synchronization, a separate exception record will be generated for each failed record. The error details are captured as follows:

  * Short error message: Stored in the **Error Message** field.
  * Detailed error message: Stored in the **Detailed Error Message** field.
