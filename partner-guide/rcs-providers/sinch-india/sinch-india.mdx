Sinch India is a cloud communications platform that empowers businesses to connect with their customers through personalized, secure, and scalable messaging experiences. With a robust suite of APIs and solutions for SMS, voice, email, RCS, WhatsApp, and more, Sinch enables seamless omnichannel engagement across customer touchpoints.

# Configure Sinch India as an RCS Connector (Service Provider)

Perform the following steps to configure Sinch India as a connector:

  1. On the left navigation menu in the MoEngage UI, go to **Settings** > **Channels** > **SMS & RCS** > **Sender configuration**.
  2. On the **RCS** tab, click **\+ Add RCS Sender**.
  3. Click the **Sinch India** tile.
  4. On the Sender Details page, add the following details:  
![MoEngageConfiguration.png](/images/partner_39591320439444.png)  

| Field | Description

| --- | --- |  
| Sender type |  Select the sender type. It must be the same sender type as you have created with Sinch India during onboarding:
     * **Promotional** : Select this option when you want to use the sender to send information about your brand, promote deals, or engage with users.
     * **Transactional** : Select this option when you want to use the sender to send alerts about transactions, security information, or any information that can be classified as transactional in nature.
     * **OTP** : Select this option when you want to use the sender to send OTPs.  
| Sender Name |  Enter the name of the sender. This field identifies the readable sender name that can be used while creating a campaign or alert.  
| Sender ID |  Enter the sender ID. This is the Sinch platform app ID visible on SmartTA. You can get this from the Sinch India onboarding team.  
| API URL |  You can get this API URL from the Sinch India onboarding team.  
| ProjectID |  You can get this ProjectId from the Sinch India onboarding team.  
| AppID |  You can get this App ID from the Sinch India onboarding team.  
| User |  You can get this User from the Sinch India onboarding team.  
| Pass |  You can get this Pass from the Sinch India onboarding team  
  5. Click **Save** to save this sender configuration.
  6. **Incoming Messages and delivery callback URL** : Configuring delivery tracking allows you to monitor whether messages have been delivered to end users. Configuring incoming message URLs allows you to track messages' responses so that you can act on them accordingly. Copy this URL from the MoEngage UI and email it to the Sinch India onboarding team so they can configure it.
