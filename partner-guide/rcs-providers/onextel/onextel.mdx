# Introduction

[OneXtel](https://www.onextel.com/) is a leading Communication Platform as a Service (CPaaS) provider, offering seamless, end-to-end communication solutions. Their product offering includes channels like SMS, Voice, WhatsApp, Chatbot, RCS, and Email services tailored for businesses of all sizes.

<Info>
Ensure you have access to the following:

  * An active OneXtel account on 365cx.io
  * An active MoEngage Account
</Info>
# Configure OneXtel as an RCS Connector Provider for MoEngage

The configuration process includes two main parts:

  * Create a channel in 365cx.io (OneXtel)
  * Configure in MoEngage RCS Custom Connector



## Step 1: Create a Channel in 365cx.io (OneXtel)

To create a channel in 365.io(OneXtel), perform the following steps:

  1. Log in to your 365cx.io account.

  2. Navigate to the channel creation section.

  3. Create a new channel by entering the following details:

| Field | Description |
| --- | --- |
| Channel Name | Enter a unique name to identify the channel. |
| Department | Select the appropriate department responsible for managing and accessing this channel. |
| Type | Select RCS as the channel type in Microsoft 365 and choose the appropriate vendor. |
| Credentials | Provide the following details: **Client ID** , **Client Secret** , and **Bot ID**. |
  4. Click **Create Channel** to complete the setup.




## Step 2: Configure in MoEngage RCS Custom Connector

To set up a campaign using the RCS Custom Connector on the MoEngage UI, perform the following steps:

  1. On the left navigation menu in the MoEngage UI, click **Engage** > **Campaigns**. The All campaigns page is displayed.
  2. In the upper-right corner, click **\+ Create campaign**. The Create campaign page is displayed.![](/images/partner_42818300831764.png)  
You can also click **Create New** > **Campaign**. The Campaign page is displayed.
  3. Under **Connectors** , click **Custom** and select **One Time** as the campaign delivery type. For more information, refer to [Delivery Types.](https://help.moengage.com/hc/en-us/articles/360058616131-Delivery-Types-MoEngage-Channels)  
![](/images/partner_42818303616916.png)

  4. On the Campaign details screen, fill in the following details:

     1. Enter the **Campaign name** and **Campaign tags**.

     2. Click **Audience** , then click **Next**.

  5. Now, add the **POST API** details, which typically include the URL and headers for the Custom Connector.![](/images/partner_42818445932948.png)
