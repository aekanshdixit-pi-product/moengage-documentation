[TikTok Ads](https://ads.tiktok.com/business/en-US/how-it-works) are paid promotional content that businesses and brands can use to reach a wider audience on the TikTok platform, leveraging the app's large and engaged user base.

# MoEngage × TikTok Ads

The MoEngage and TikTok Ads integration allows MoEngage users to add user data to TikTok Ads for delivering advertisements based on behavioral triggers and segmentation.

## Use Cases

Integrating TikTok with MoEngage helps you solve the following use cases:

  * **Enhanced user engagement and conversions** : You can target high-value users via multiple channels to drive purchases or engagement.
  * **Re-engagement of less responsive users** : You can target users who are less responsive to other marketing channels by showing them targeted display ads based on their past behavior.
  * **Optimized advertising spend** : You can create suppression audiences to prevent users from receiving advertisements when they’re already loyal consumers of your brand.
  * **Efficient new user acquisition** : You can create lookalike audiences to acquire new users more efficiently.

## Step 3: Create a Campaign on MoEngage to Sync Audiences to TikTok Ads

To create a connector campaign on MoEngage, perform the following steps:

  1. On the left navigation menu in the MoEngage UI, click **Engage** , and then click **Campaigns**.
  2. On the **All campaigns** page, click**\+ Create campaign**.
  3. On the **Create campaign** dialog box, under **Connected Apps** , click **TikTok** and then click the most suitable delivery type.  
![](/images/partner_38765492883988.png)
  4. Under **Select audience** , click **Filter users by**.
  5. On the **User property** tab, select the user segment for Pinterest syncing. You can combine this with other segment criteria.
  6. In step 2 of the campaign, **Content** , select the pre-configured connectors and specify the content you want to send out. Enter the following details:  

| Field | Description |
| --- | --- |
| **Connector** | You can perform the following actions: * **Create Audience Segment** * **Update Audience Segment** |
| **Connections** | Select the connection you’ve added to the MoEngage App marketplace. ![](/images/partner_38765492884244.png) |



### Sync to an Existing Audience on TikTok

If you want to add/remove users from an existing Audience on TikTok, you must provide that Audience’s ID. You can view the audience on TikTok by going into your [TikTok Audience Manager](https://ads.tiktok.com/):

  1. On the Header Navbar, select **Tools**.
  2. Under **Audiences** > select **Audience Manager**.
  3. View your Audience ID by locating your desired Audience.
  4. From the **Audience ID** column, note the Audience ID.  
![](/images/partner_38765534637460.png)



### Create a New Audience on TikTok Ads

To create a new Audience on TikTok, you can use the **Create Audience Segment** connector and follow these steps:

  1. Select the **Create Audience Segment** connector and the connection you created.
  2. Enter the Audience Segment Name and other required details.
  3. Run a test campaign and observe the response.  
![](/images/partner_38765534637716.png)
  4. From your **Audience list** , copy the **Audience ID** for the new audience you created.   
**Note** : This ID is required to sync users to this audience from MoEngage. You can always find this ID by navigating to your Audience list in TikTok Ads Manager.

Refrain from publishing this campaign, as it will lead to the creation of duplicate audiences on TikTok.

## Step 4: View Your TikTok Ads Audience Segments

In your TikTok Ads Manager account, select **Audience** > **Audience Manager** to view your audiences. The size of each audience is visible on the **Audiences** page after it reaches approximately 1,000.

<Info>
TikTok processes ID updates asynchronously. Therefore, it takes 48 hours for the audience size to be updated.
</Info>
