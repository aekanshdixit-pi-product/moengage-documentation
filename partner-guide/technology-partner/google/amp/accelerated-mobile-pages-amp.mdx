---

Accelerated Mobile Pages (AMP) is a web component framework and a website publishing technology developed by Google aimed to provide a user-first format for web content.  
AMP an open-source project that helps web pages to load instantaneously on mobile devices.  
You might have seen the lightning bolt icon similar to ⚡ while searching anything(usually News articles) in Google. This icon indicates that these pages are AMP pages.

For more information, refer to the [open-source project](https://www.ampproject.org/).

Moengage provides User Data and Event Tracking and Web Push Notification channel in AMP.

To integrate Accelerated Mobile Pages (AMP) with MoEngage Web SDK

  1. Ensure that your AMP pages are set properly.  
For more information on setting up, refer to [How to create an AMP](https://www.ampproject.org/docs/tutorials/create).
  2. [Configure and integrate AMP Event Analytics ](/partner-guide/technology-partner/google/amp/configure-and-integrate-amp-event-analytics)(Optional but Recommended)
  3. [Configure and integrate AMP Web Push ](/partner-guide/technology-partner/google/amp/configure-and-integrate-amp-web-push)(Required)
