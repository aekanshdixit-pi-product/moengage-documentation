# Introduction  
  
<Info>
The Twilio Segment integration does not currently support [Unified Identity (Identity Resolution)](https://help.moengage.com/hc/en-us/articles/24050999467284-Unified-Identity-Identity-Resolution) or [Portfolio](https://help.moengage.com/hc/en-us/articles/40394603054100-Portfolio).
</Info>
[Twilio Segment](https://segment.com/) is a customer data platform that helps you collect, clean, and activate your customer data. If you are already using Segment for tracking data, the MoEngage integration makes it easy to send your event and attribute data to MoEngage.

After you've tracked your data through Segment's source libraries, Segment translates and routes your data to MoEngage in a format MoEngage understands, thereby saving your engineering efforts. For more information, refer to [MoEngage with Twilio Segment](https://segment.com/docs/connections/destinations/catalog/moengage/).

# Scope

The MoEngage and Segment integration allows you to track your users and route data to a wide variety of user analytics providers. Segment allows you to:

| Integration scope | Details |
| --- | --- |
| Send Segment data (users and events) to MoEngage | For more information, refer to [MoEngage (Destination)](https://partners.moengage.com/hc/en-us/articles/9120306530452-MoEngage-Destination#h_introduction). |
| Sync Segment audiences and computed traits to MoEngage | For more information, refer to [Twilio Segment Engage](https://partners.moengage.com/hc/en-us/articles/5873974672148-Segment-Personas) (Earlier Segment Personas). |
| Send MoEngage campaign integration data back to Segment | For more information, refer to [MoEngage (Source)](https://partners.moengage.com/hc/en-us/articles/12661517099284-MoEngage-Source). |
