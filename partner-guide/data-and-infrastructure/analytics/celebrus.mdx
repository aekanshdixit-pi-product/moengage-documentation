# Introduction 

[Celebrus](https://www.celebrus.com/) Customer Data Platform (CDP) is a pioneering digital data capture solution which collects, contextualizes and delivers data about how individuals are behaving across a brand's digital channels including, websites, mobile apps, social and streaming media.

# MoEngage × Celebrus

Celebrus captures rich profile data for each individual customer, which can be synced with MoEngage. This empowers you to create effective MoEngage Segmentation and engagement strategies based on comprehensive, accurate, and detailed first-party data.

This capability is further bolstered by Celebrus’ Machine Learning-driven Signals, allowing for hassle-free data capture without the need for extensive tagging. With a robust first-party identity graph in place, all data becomes instantly accessible for immediate use.

# Integration

<Info>
**Prerequisites**

\- Ensure you have access to an Celebrus Account \- Ensure you have access to one of the data warehouse
</Info>

Customer attributes can be synced with MoEngage via any data warehouse imports - supported by MoEngage.

## Step 1: Configure Datawarehouse details in Celebrus

The Celebrus connector for MoEngage custom attributes sends custom attributes to an intermediate database. In Celebrus you configure the connection details for the database, which will depend on what type of database you are using (such as Snowflake or Redshift).

Follow the Celebrus guide to export data to a data warehouse. 

##  Step 2: Configure Data warehouse imports on MoEngage

Follow the instructions in [Data warehouse imports document](/user-guide/data/imports) to set up and configure the MoEngage sync
