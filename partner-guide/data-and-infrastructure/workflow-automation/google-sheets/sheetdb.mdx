# Introduction

[Sheetdb](https://sheetsu.com/) turns a Google Spreadsheet into a JSON API. It allows you to connect your sheets to CRM, API, Website, any application or tool. 

# MoEngage × Sheetdb

The MoEngage and Sheetdb integration allows you to read and write data to your Google spreadsheet.

# Use cases

1\. You have a spreadsheet with different campaign messages for different users with slight tweaks. You can want to send the exact message (from this spreadsheet) to your users. Using the integration, you can reference the spreadsheet from the MoEngage campaigns to send right messages to your users. 

2\. You have some attributes are very specific to a campaign that needs to be send out. This is just a one-time campaign, so you don't want to store these details on MoEngage. You can simply add these attributes to a google spreadsheet. Now reference the sheet using MoEngage content APIs. 

3\. You want to keep updating (in near real time) a spreadsheet to keep track of users who perform a specific action on MoEngage. Using the integration, you can set up your sheetdb spreadsheet endpoints in a MoEngage connector campaign. 

# Integration

<Info>
**Prerequisites**

* Ensure you have access to your Sheetdb account.
</Info>

Using the integration, you can:

1\. Fetch details from a spreadsheet at the run time and use them in your MoEngage campaigns using Content APIs. 

2\. Update a spreadsheet in near real-time with the details for a specific user segment or as soon as the user performs as a specific event. 

## A) Use data from a spreadsheet in MoEngage campaigns 

Let's say your file has detail on the Message title, message and button name for your users. 

![Screenshot_2022-06-29_at_12.42.43_PM.png](/images/partner_21979659923732.png)

Note- In this example, we are using ID to search users. You can use any other attribute such as email id or mobile number to identify a user. 

### Step 1: Upload a sheet on Sheetbu 

On your Sheetbu dashboard homepage, 

  1. Click _Create new_ on top right ![Screenshot_2022-06-29_at_3.26.29_PM.png](/images/partner_7401327382036.png)
  2. Add your spreadsheet URL ![Screenshot_2022-06-29_at_3.27.50_PM.png](/images/partner_7401381763220.png)
  3. Click _Create API_
  4. You can now test if your API is working fine by clicking on _Go to API URL.![Screenshot_2022-06-29_at_3.30.46_PM.png](/images/partner_7401397437204.png)_
  5. Copy your sheets API URL. It should look like this -
         
         https://sheetdb.io/api/v1/2ru3wpxjfdc2s




### Step 2: Add Sheetdb as Content API on MoEngage 

Perform the following steps:

  1. Navigate to **Settings** > **Advanced settings** > **Content API**. Click **\+ Add content API** at the top right corner of the Content API screen.![ContentAPI_Navogation.png](/images/partner_21979659930388.png)
  2. Add your Sheetdb name and API URL.  
![Screenshot_2022-06-29_at_3.33.10_PM.png](/images/partner_7401438488980.png)

**API URL**  
Enter the following URL. Please note that _your sheetsb API URL_ should be the same as copied from the previous step. 
         
         https://sheetdb.io/apis/v1/&lt;your sheetbn API url&gt;/search?ID=&#123;&#123;UserAttribute['ID']&#125;&#125;

  3. Click Save. And you will see your Sheetdb API added in MoEngage. ![Screenshot_2022-06-29_at_3.34.52_PM.png](/images/partner_7401480179732.png)



Note - You can also test your APIs. Provide a sample ID and click Test. You can your API response and validate. ![Screenshot_2022-06-29_at_3.33.40_PM.png](/images/partner_7401479515924.png)

### Step 3: Using Content API's data in MoEngage Campaigns 

You can now use these content APIs in your campaigns. [Read here](/user-guide/campaigns-and-channels/getting-started/campaign-content/content-apis#step-2-using-content-apis-to-create-personalized-campaigns-0-3) to know more about Content APIs 

  1. On step 2 on your campaign creation, enter @.
  2. In the pop up, you can select the Sheetdb Content API configured in Step 2. ![Screenshot_2022-06-29_at_3.37.10_PM.png](/images/partner_7401483557652.png)
  3. Click done to include the response in your campaigns. 

  4. You can use MoEngage templating language to parse through the API response and use the content in your campaigns. 




## B) Write data to spreadsheet using MoEngage Connectors 

### Step 1: Upload a sheet on Sheetbu 

Make your sheet available for Sheetbu APIs. You can follow the same steps as in A.Step 1 in the previous example. 

### Step 2: Add your Sheetbu endpoint in MoEngage connector campaigns 

To create a connector campaign on MoEngage, 

1\. Navigate to Create Campaign >> Connectors >> Custom and choose delivery type most suitable to your needs. 

![Screenshot_2022-02-21_at_10.48.46_PM.png](/images/partner_21979659934356.png)

2\. On step 2 of the campaign, 

  * Select method as POST
  * Add Webhook URL as : 
        
        https://sheetdb.io/apis/v1/&lt;your sheetdb API URL&gt;

![Screenshot_2022-06-29_at_3.39.35_PM.png](/images/partner_7401499731732.png)
  * In Body, define the keys in your spreadsheet and the values that you want to send out. 



3\. At this point, your campaign should be ready to test and send. You can also test your connector configuration using the Test functionality available on Step 2 of Create Connector Campaign. 

![Screen_Shot_2022-02-22_at_11.12.42_AM.png](/images/partner_21979691834388.png)

If the data is updated successfully on your spreadsheet, your set-up is good to go. You can go ahead and Publish the campaign.
