## Introduction

[Instapage](https://instapage.com/) is a landing page platform for creating, personalizing, and optimizing landing pages for marketing campaigns.

## MoEngage and Instapage

The MoEngage and Instapage integration lets marketers seamlessly capture user data from landing pages and use it for personalized engagement campaigns. With this integration, you can enable targeted messaging, dynamic content personalization, and improved conversion rates.

## Integration

<Info>
  **Prerequisites**

  - Ensure you have an active Instapage account.
  - You need a Zapier account to use this integration. Follow the instructions in the [Zapier overview](/partner-guide/data-and-infrastructure/workflow-automation/zapier/overview#connecting-zapier-to-moengage-0-4) to connect your Zapier and MoEngage accounts.
</Info>

## Import Instapage leads to MoEngage

<Note>
  This workflow uses the [Create or Update User](/partner-guide/data-and-infrastructure/workflow-automation/zapier/create-or-update-users) action.
</Note>

You can import your leads as users in MoEngage and store their values as user attributes. This lets you create custom segments in MoEngage for targeted campaigns.

### Step 1: Select Instapage as the trigger app

![Zapier trigger configuration screen with Instapage selected as the trigger app](/images/partner_26219218323860.png)

1. Select **Instapage** from the list of apps.
2. Choose **Form Submission** as the event, then click **Continue**.
3. Connect your Instapage account. Select your Instapage account from the dropdown and click **Continue**.
4. Select the page you want to connect to and click **Continue**.

   ![Instapage page selection screen in Zapier showing available landing pages](https://help.instapage.com/hc/article_attachments/19297089046423)

5. Click **Test trigger**. Zapier attempts to fetch a recent lead. Click **Continue** to move to the next step.

   ![Zapier trigger test results showing a fetched Instapage lead](https://help.instapage.com/hc/article_attachments/19297043707543)

### Step 2: Select MoEngage as the action app

Search for **MoEngage** from the apps list and select it.

![Zapier action configuration screen with MoEngage selected as the action app](/images/partner_26218603194900.png)

1. From the event dropdown, select **Create or Update User**, then click **Continue**.
2. If you have followed the [prerequisites](/partner-guide/data-and-infrastructure/workflow-automation/zapier/overview), your MoEngage account already appears in the list. Choose it or connect a new one, then click **Continue**.
3. In the **Set up action** step, you see a list of attributes and user properties to either fill in manually or map from the previous step. Refer to the [Create or Update User API reference](https://www.moengage.com/docs/api/user/track-user) for details. Add the relevant custom user attributes as required.
4. Once you are satisfied with your mappings, click **Continue**.
5. In the final step, click **Test & continue**.
6. If the test was successful, you should see a `success` response.

### Verify the lead in MoEngage

You can verify the lead in your MoEngage dashboard. Go to your Instapage lead's user profile and look under custom attributes (if you set them up).

![MoEngage user profile showing custom attributes captured from an Instapage lead](/images/partner_26219216089108.png)

You can now send campaigns by filtering users that have the relevant user attributes, or create targeted custom segments to engage your leads.

<Check>
  The Instapage lead appears as a user in MoEngage with all mapped attributes populated.
</Check>
