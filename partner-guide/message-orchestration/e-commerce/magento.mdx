# Introduction

[Magento](https://business.adobe.com/products/magento/magento-commerce.html) is an open-source e-commerce platform that helps businesses build and manage online stores. It includes tools for marketing, search engine optimization, and catalog management.

# MoEngage <> Magento

The Magento-MoEngage integration offers a seamless way to integrate the MoEngage Web SDK into your brand's Magento store, enabling you to track user events and data with ease. With this integration, you can leverage a variety of channels - including Web Push, Onsite Messaging, SMS, Email, and Web Personalization - to engage and nudge your store visitors toward conversion.

Read our [MoEngage <> Magento guide ](https://developers.moengage.com/hc/en-us/articles/29334278938388-Magento#h_01J3Z4H5BW5YJAVS7A62V7JFRA)to start using the integration for your account.
