# Introduction

[WooCommerce](https://woocommerce.com/) is an open-source e-commerce plugin for WordPress. It is designed for small to large-sized online merchants using WordPress.

# MoEngage × WooCommerce

The WooCommerce-MoEngage integration offers a seamless way to integrate the MoEngage Web SDK into your brand's WooCommerce store, enabling you to track user events and data with ease. With this integration, you can leverage a variety of channels - including Web Push, Onsite Messaging, SMS, Email, and Web Personalization - to engage and nudge your store visitors toward conversion.

Read our [MoEngage × WooCommerce guide ](https://developers.moengage.com/hc/en-us/articles/29334304720148-WooCommerce)to start using the integration for your account.
