## Typeform Essentials for MoEngage Integration

Below are a few essential things on Typeform that you would need to know for integrating surveys in MoEngage campaigns -

  1. Survey URL: A unique survey URL is generated for each survey. You can find this survey URL on the Typeform dashboard, as shown below. This URL will need to be inserted on MoEngage dashboard when you want to embed a survey on MoEngage.  
  
![ezgif.com-video-to-gif__2_.gif](/images/partner_19131579395604.png)  
  

  2. Personalizing the Surveys: You can personalize your surveys by recalling the earlier responses submitted for the same by the same user. You can refer to this [doc](https://help.typeform.com/hc/en-us/articles/360029252232-Recall-Information) for more details on how to do this.
  3. Hidden Fields: Hidden Fields on Typeform allow you to pass additional parameters with the survey. A common example of using Hidden Fields is to pass User ID along with the responses for analysis.
  4. Understand survey responses: You can analyze the responses on the Typeform dashboard here. To know more about response analysis, refer to the documentation [here](https://help.typeform.com/hc/en-us/articles/360029614691-Understand-your-results-insights).  
You can also download a report with detailed responses by following the steps mentioned [here](https://help.typeform.com/hc/en-us/articles/360029253572-Export-your-results).
  5. Google Sheets Integration: You can connect your Typeform surveys to Google Sheets so that every time a response is recorded for a survey, it will be visible in the Google sheet automatically. Follow the steps mentioned in this [doc](https://help.typeform.com/hc/en-us/articles/360029256192-Google-Sheets-for-Typeform) for more details.



## On-site Messaging Campaigns

To embed surveys in On-site Messaging Campaigns, please follow the below steps -

  1. Select one of the two available survey templates before starting with campaign creation:  
![Screen_Shot_2020-04-14_at_1.47.17_PM.png](/images/partner_19131568720404.png)  
  
**Survey Pop-up:** This will render your survey inside a blocking popup on your website. The user will have to either close the survey or complete it. A survey pop-up will look something like this on your website -  
![Screen_Shot_2020-04-14_at_1.55.57_PM.png](/images/partner_19131568721428.png)  
  
**Survey Sticky Banner:** This will render on your website as a non-blocking banner. The user can continue to interact with the website and choose to take this survey at any time. The survey can also be closed when required. This will look something like this on your website -   
![Screen_Shot_2020-04-14_at_2.00.23_PM.png](/images/partner_19131579421972.png)  
  

  2. Once a survey template is selected, you can go to the step-2 and insert your survey URL as shown below -  
  
![Screen_Shot_2020-04-14_at_1.52.54_PM.png](/images/partner_19131579426836.png)  
  

  3. You can personalize your survey URL with user attributes if you need to pass additional parameters for analysis.  
For example, if your Survey URL is this -  _https://typeformsales7.typeform.com/to/OLMOfS?user_id=xxxxx_  
you can pass the unique user id (User Attribute - **ID**) and save this in the response by updating the URL to be something like this -  _https://typeformsales7.typeform.com/to/OLUWfS?customer_id=[&#123;&#123;UserAttribute['uid']&#125;&#125;](https://flipkartseller.typeform.com/to/ka1g7I?moe_user_id=&#123;&#123;UserAttribute\[%27uid%27\]&#125;&#125;)_  
Please note that to be able to pass additional parameters, you need to pre-configure the hidden fields in your Typeform as mentioned in point 3 of this section.



## Email Campaigns

To link surveys to your Email Campaigns, please follow the below steps -

  1. Start Creating Email Campaigns as mentioned in the user guide [here](https://help.moengage.com/hc/en-us/articles/206184606-Creating-Email-Campaigns).  
  

  2. On Step-2 of campaign creation, paste the survey URL as the call to action for one of the elements as shown below -![Screenshot_2021-01-05_at_11.54.23_AM.png](/images/partner_19131568737172.png)  
  

  3. When the element on which the survey URL is linked is clicked by a user, the survey will open up in a new browser tab. This is how it will look on a mobile device -  
  
![Screenshot_2021-01-05_at_12.05.29_PM.png](/images/partner_19131579442196.png) ![Screenshot_20200414-142315.png](/images/partner_19131579444628.png)



## Mobile InApp Campaigns

To link surveys to your Mobile InApp Campaigns, please follow the below steps -

  1. Start Creating Mobile InApp Campaigns as mentioned in the user guide [here](https://help.moengage.com/hc/en-us/articles/206184196-Creating-In-App-NATIV-Campaigns).  
  

  2. On Step-2 of campaign creation, paste the survey URL as the call to action for one of the template widgets.  
![Screen_Shot_2020-04-14_at_2.29.51_PM.png](/images/partner_19131568762772.png)  
  

  3. Once the widget which has the survey URL as the call to action is clicked by any user, the survey opens inside a web view. 



<Info>
Using Typeform in HTML In Apps Typeform surveys currently cant be used in HTML In Apps. However, you can use the survey link as call to action link.
</Info>
## Push Campaigns

To link surveys to your Mobile InApp Campaigns, please follow the below steps -

  1. Start Creating Mobile InApp Campaigns as mentioned in the user guide [here](https://help.moengage.com/hc/en-us/articles/208735856-Creating-General-Push-Campaign).  
  

  2. On Step-2 of campaign creation, paste the survey URL as the call to action as shown below.  
![Screen_Shot_2020-04-17_at_3.36.53_PM.png](/images/partner_19131579465364.png)  
  

  3. Once the push notification is clicked by the user, the survey URL will open within a webview inside the app (Rich Landing Page) or open in a browser (deep link) based on the configuration selected.



## SMS Campaigns

To link surveys to your SMS Campaigns, please follow the below steps -

  1. Start Creating SMS as mentioned in the user guide [here](https://help.moengage.com/hc/en-us/articles/115003114083-SMS-Getting-Started).  
  

  2. On Step-2 of campaign creation, you can insert the URL and paste it directly in your SMS Campaign as shown below -  
![Screen_Shot_2020-04-17_at_3.45.28_PM.png](/images/partner_19131568772756.png)
  3. Once the link in the SMS is clicked by a user, the survey will open in a new browser tab.
