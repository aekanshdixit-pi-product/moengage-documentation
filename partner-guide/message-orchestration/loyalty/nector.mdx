[Nector](https://www.nector.io/) is a customer loyalty and rewards platform that helps businesses increase customer retention and engagement. It enables merchants to reward customers with wallet coins, coupons, and referral bonuses, and allows them to collect and manage customer reviews.  
  
## MoEngage × Nector

The integration of Nector and MoEngage empowers businesses to leverage Nector’s loyalty data to trigger timely, personalized automations directly in MoEngage. By connecting these platforms, you ensure that your loyalty events (such as points earned, coupons redeemed, or reviews submitted) drive customer engagement without manual intervention. With this integration, you can:

  * Push real-time loyalty events from Nector to MoEngage.

  * Trigger automated email or push notification workflows in MoEngage based on Nector activities.

  * Maintain a unified view of customer loyalty behavior within your marketing automation platform




## Use Cases

Integrating MoEngage with Nector helps you with the following use cases:

  * **Reward notifications** : Automatically trigger an email or push notification via MoEngage whenever a customer is rewarded with wallet coins in Nector.
  * **Coupon delivery** : Instantly send discount codes or coupons to customers via MoEngage workflows when they redeem their coins on Nector.
  * **Referral updates** : Notify customers when their referrals are successful or when they earn referral bonuses.



# Integration

<Info>
Before you begin the configuration, ensure you have the following:

  * An active Nector UI account.
  * Access to your MoEngage UI.
  * Your MoEngage App ID, Data API Key, and Server Prefix.
</Info>
## Step 1: Locate the Integration in Nector

To start the configuration, you must locate MoEngage in your Nector UI. To locate the Integration in Nector, perform the following steps:

  1. Log in to the Nector UI.
  2. On the sidebar, click **[Integrations](https://merchant.nector.io/integration-list)****.**
  3. Click **Email** **& WhatsApp Marketing**.![](/images/partner_43964249403924.png)
  4. Click **Configure**.   
The MoEngage integration page appears.



## Step 2: Retrieve API keys from MoEngage

To retrieve API keys from MoEngage, perform the following steps:

  1. On the MoEngage UI, navigate to **Settings** > **Account** > **APIs**.
  2. Copy the ID under **Workspace ID (earlier app id)**.
  3. In the Nector dashboard, paste the Workspace ID (earlier app id) in the **App Id** box.
  4. Paste the Workspace ID (earlier app id) again in the **Public API Key** box.
  5. Now, from the MoEngage UI, copy the **Data API Key** under **API Keys**.![](/images/partner_43967128905620.png)
  6. In the Nector UI, paste the Data API Key in the **Private API Key** box.
  7. In the **Server Prefix** list, select your MoEngage server prefix (for example, api-01).  
**Note** : Ensure you have entered all information correctly. The integration will fail if any details are missing or incorrect.
  8. Click **Initialize**.![](/images/partner_43964290031252.png)



## Step 3: Configure Server Prefix and Initialize

To configure the server prefix, perform the following steps:

  1. Check your MoEngage UI URL.
  2. Select the Server Prefix in Nector based on your URL: 
     * If the URL starts with dashboard-01.moengage.com → Select api-01.

     * If the URL starts with dashboard-02.moengage.com → Select api-02.

  3. Under **Choose Common Property** , select **Email**.   
This ensures Nector maps events to the correct user profile in MoEngage.
  4. Click**Initialize** to enable the integration.![](/images/partner_43964964785300.png)



## Step 4: Send Events to MoEngage

You can configure Nector to send events to MoEngage when specific actions occur. For example, when a customer is rewarded with wallet coins, Nector can send an event to MoEngage, which then triggers an automation to email the customer.

To add a new event, perform the following steps:

  1. In the MoEngage integration page, click the **Events** tab.
  2. Click **Create An Event** (or click the **+** button).
  3. On the **Send Events** tab, select the event for which you want to notify the external integration.  
**Note** : Nector supports many events. Two common and relevant events are: 
     * **Wallet Coins Rewarded** : Send whenever wallet coins are rewarded in Nector.

     * **Coupon Is Created** : Send whenever a customer redeems wallet coins for a discount or coupon code in Nector.




To configure a Wallet Coins Rewarded event, perform the following steps:

  1. In the **Event** list, click **Wallet Coins Rewarded**.
  2. In the **Event Sent to** list, click MoEngage. (This is the default selection.)
  3. Click **Create**.  
![](/images/partner_43966341419412.png)  
![](/images/partner_43966341421076.png)  
The event is created and appears in the **Events** list.



## Step 5: Test the Integration

To test the integration and event availability, perform the following steps:

  1. On the Events page, click **Send Test Event** with the event you created.
  2. In the **Send Test Event** dialog box, ensure that email is selected in the **Search customer via** list.
  3. In the **Email id of customer to whom event has to be sent** box, type the valid email address of a customer.  
**Note** : The email address must be associated with an existing customer in your store's Customer database.
  4. Click **Send Test Event** to trigger the event.
  5. Check your MoEngage UI to confirm the event was received.![](/images/partner_43966001101716.png)
