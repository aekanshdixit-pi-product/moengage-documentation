# Introduction

[LoyaltyLion](https://loyaltylion.com/) is an e-commerce customer loyalty and engagement platform, that powers growth and retention through data-driven loyalty programs.

# MoEngage <> LoyaltyLion

The MoEngage and LoyaltyLion integration enables you to track your customers' activities via MoEngage and reward them accordingly inside LoyaltyLion. Using the LoyaltyLion Rest APIs, you can forward purchases made on your app to LoyaltyLion and send out personalized campaigns to engage them post-purchase.

# Integration<Info>
**Prerequisites**
* You would need your LoyaltyLion token and a secret to use their APIs.

## Display customer rewards in MoEngage Campaigns

Using MoEngage's Content APIs, you can dynamically pull customers' reward points inside MoEngage campaigns for a more personalized experience.

### Step 1: Set up a MoEngage Content API

  1. Navigate to **Settings** -> **Advanced settings** -> **Content API**. Click **\+ Add content API** at the top right corner of the Content API screen.![ContentAPI_Navogation.png](/images/partner_21979026575124.png)
  2. Add your LoyaltyLion API to the endpoint URL. Depending on your use case, the endpoint will change.   


![](/images/partner_19615149966356.png)

To display your customers' rewards, use LoyaltyLion's [Get Available Rewards API](https://developers.loyaltylion.com/api/resources/rewards#get-v2customersmerchant_idavailable_rewards).

While setting up your content API, insert the following URL:
         
         https://api.loyaltylion.com/v2/customers/&#123;&#123;UserAttribute['customer_id']&#125;&#125;/available_rewards

Under Headers, add the following:

     1. `Authorization`: Use an online tool to create your basic auth with `token` as a username and your `secret` as a password:  
![](/images/partner_19614378396692.png)  
In the above example, the value of this header would be `Basic dG9rZW46c2VjcmV0`
     2. `Content-Type`: `application/json`
  3. Click on "Save" and then "Done" once you see a successful response.
  4. Note - You can also test your APIs. To test your APIs, from the Content API's action menu, click on "Test". You can your API response and validate it. 



### Step 2: Display your customers' rewards inside MoEngage campaigns

To use this API in your MoEngage campaigns,

  1. On step 2 on your campaign creation, enter @. 
  2. In the pop up, you can select the LoyaltyLion Content API configured in Step 1.
  3. You can use [MoEngage Templating Language](https://help.moengage.com/hc/en-us/articles/115002757783-MoEngage-Templating-Language-JINJA-#content-api-0-3) to parse through the API response and use the content in your campaigns.  
![](/images/partner_19616447980820.png)
  4. You can click on the Preview tab to view your campaign in real-time personalization:  
![](/images/partner_19616419328532.png)
  5. Once you are satisfied with the results, you can publish your campaign.
