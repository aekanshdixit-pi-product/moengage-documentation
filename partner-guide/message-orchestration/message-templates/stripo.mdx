# Introduction

[Stripo](https://stripo.email/) is a drag-and-drop email template builder that helps you create and design responsive emails with interactive elements. Stripo users can also edit in HTML and decide which elements to display or hide on various devices through the Stripo editor.

# MoEngage × Stripo

The MoEngage and Stripo integration allows you to export your customized _Stripo emails_ and send them as email templates to MoEngage.

# Integration

<Info>
**Prerequisites**

Ensure you have a Stripo account.
</Info>

## Step 1: Create a Stripo email

On your Stripo dashboard, create any email template. And then click _Export_ on the top.  
![Screenshot_2022-12-12_at_12.19.42_PM.png](/images/partner_11531901735188.png)

## Step 2: Export templates to MoEngage 

  1. In the pop up, navigate to the section _Export to ESV_
  2. Search for _MoEngage_   
![Screenshot_2022-12-12_at_12.21.29_PM.png](/images/partner_11531938039956.png)
  3. Click on the _MoEngage_ card. A pop up will appear.   
![Screenshot_2022-12-12_at_12.23.16_PM.png](/images/partner_11531939376148.png)
  4. Add the details of your MoEngage account and click _Export._

| **Field** | **Description** |
| --- | --- |
| Connector name | Name of the connector- You can enter the name of your MoEngage app for easy identification |
| Select your instance | Region of your MoEngage data cluster. Read more |
| API key | API key is the Workspace ID. The Workspace ID of your MoEngage account is available at **Settings - > Account -> APIs ->** **Workspace ID**. The Data API ID and the Workspace ID of your MoEngage account are the same. |
| API Secret key | API Secret key is the Data API Key. The Data API key of your MoEngage account is available at **Settings - > Account -> APIs -> API keys **>**Data** API settings. |
  
Note: This is a one-time setup. Any exports in the future will automatically utilize the same credentials. ![Screenshot_2022-12-12_at_12.30.35_PM.png](/images/partner_11531983065236.png)

## Step 3 - Using translation in MoEngage campaigns

Once the templates are created, you will be able to access them in Step 2 of your Email Campaigns. You will see the newly created templates under the _My saved templates_ section.
