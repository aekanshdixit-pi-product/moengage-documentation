## Introduction

[Rocketium](https://rocketium.com/) is a creative automation platform helping brands and businesses improve in-app and website conversions through video and display banner automation. Its creative automation platform helps marketers and designers run more relevant campaigns faster across the customer life cycle.

## MoEngage × Rocketium

MoEngage Rocketium integration empowers you to use Rocketium creatives in MoEngage campaigns.

## Integration

<Info>
  **Prerequisites**

  Ensure you have a Rocketium account.
</Info>

There are two type of integrations supported:

1. Use your Rocketium creatives directly in MoEngage campaigns by adding them as image URLs.
2. Fetch personalised Rocketium creatives (rendered on the fly based on user attributes) in MoEngage

### 1. Using direct image URLs

#### Step 1: Create a project on Rocketium

1. Create a new project using the required template.
   ![1-2](https://knowledge.rocketium.com/hs-fs/hubfs/1-2.png?width=688&name=1-2.png)
2. To create different variants (based on user personalised data), you can go through [Rocketium guide on spreadsheet imports](https://docs.rocketium.com/campaign/scale/spreadsheet-import) to build and publish the project. \<This can be used if you want to use personalised creatives in MoEngage\>

#### Step 2: Get your Rocketium project URL

1. Click on the Share button from the overflow menu.
   ![4-1](https://knowledge.rocketium.com/hs-fs/hubfs/4-1.png?width=688&name=4-1.png)
2. Copy the link from the bottom of this modal. This will be used in the next step while creating a campaign in MoEngage.
   ![6](https://knowledge.rocketium.com/hs-fs/hubfs/6.png?width=688&name=6.png)

### 2. Use personalised URLs

If you want to use your Rocketium creative across campaigns, you can store it content APIs in MoEngage. Once you get your URL as mentioned in Step 1 and Step 2 above,

#### Step 1: Add content API on MoEngage

1. Navigate to **Settings** -\> **Advanced settings** -\> **Content API**. Click **\+ Add content API** at the top right corner of the Content API screen.
   ![ContentAPI\_Navogation.png](/images/partner_21979803447572.png)
2. Add your Rocketium subdomain to the endpoint URL.

##### API URL

This URL should be the same as copied from the previous step.

`https://cdn.rocketium.com/api/v1/creatives/jzHSMr-EpLBP`

##### Parameters

Parameters give personalized parameter names which you have created in your spreadsheet.

![Screenshot\_2022-03-14\_at\_6.30.08\_PM.png](/images/partner_4769504056084.png)

3. Click Save. And you will see your Rocketium API added in MoEngage.
   ![Screenshot\_2022-03-14\_at\_6.32.11\_PM.png](/images/partner_4769549932436.png)

**Note -** You can also test your APIs by clicking on the Test button. You can your API response and validate.

![Screenshot\_2022-03-14\_at\_6.33.02\_PM.png](/images/partner_4769591925908.png)

#### Step 2: Using content APIs in MoEngage campaigns

You can now use these content APIs in your campaigns. [Read here](https://www.moengage.com/docs/user-guide/campaigns-and-channels/getting-started/campaign-content/content-apis) to know more about Content APIs

1. On step 2 on your campaign creation, enter `@`.
2. In the pop up, you can select the Rocketium Content API configured in Step 2.
   ![Screenshot\_2022-03-14\_at\_6.37.19\_PM.png](/images/partner_4769784416788.png)
3. Click done to include the response in your campaigns.
4. You can use MoEngage templating language to parse through the API response and use the content in your campaigns. Rocketium sends image urls in the key "Media". Sample rocketium response -

```json
{
  "Variation Name": "Bangalore",

  "Caption 1": "Bangalore",
  "Caption 2": "SALE",
  "Caption 3": "up to",
  "Caption 4": "70% OFF",
  "Caption 5": "SELECT PRODUCT",
  "Caption 6": "SHOP NOW",
```
