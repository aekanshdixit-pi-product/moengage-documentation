## Introduction

[Dynamic Yield](https://www.dynamicyield.com/) is an Experience Operating System that algorithmically matches content, products, and offers to each customer across digital channels to increase revenue and drive loyalty.

## MoEngage and Dynamic Yield

The MoEngage and Dynamic Yield integration brings together personalized content and targeted email campaigns. With this integration, businesses can embed Dynamic Yield's content blocks and recommendation blocks directly into MoEngage email campaigns.

By combining Dynamic Yield's AI-powered personalization with MoEngage's email marketing capabilities, you can deliver the right content to the right customer at the right time and maximize engagement and ROI.

## Integration

<Info>
  **Prerequisites**

  - Access to your Dynamic Yield dashboard.
</Info>

Dynamic Yield's Experience Email lets marketers transform static emails into fully personalized experiences using a visual drag-and-drop editor.

### Step 1: Design an Experience Email campaign in Dynamic Yield

1. On your Dynamic Yield dashboard, go to **Email** > **Experience Emails**, then click **Create New**.

   ![Dynamic Yield Experience Emails screen with the Create New button highlighted](https://support.dynamicyield.com/hc/article_attachments/4404282767377)
2. Choose how to create your email: start from scratch, copy a previous email, or use a template.
3. Enter a campaign name, note, and labels for your campaign in the heading area.
4. Insert and edit the email building blocks. Drag and drop from the menu on the right of the screen to add, arrange, and edit elements (click a building block to see all its design options). Email building blocks include:
   1. [Recommendations](https://support.dynamicyield.com/hc/en-us/articles/4404013832465-Experience-Email#h_01FQBRAK3MEJCQ9MVGDZ32RA2T): A widget that offers users fully personalized recommendations.
   2. [Dynamic Content](https://support.dynamicyield.com/hc/en-us/articles/4404013832465-Experience-Email#h_01FQBRAZ76DCD881CTC1MCJ2AM): Target different promotions and messages to different audiences.
5. Update your settings:
   1. Use URL parameters to track clicks within your analytics software (optional). Add parameters to the displayed defaults as needed.
   2. Select an attribution window.
6. Click **Save & Exit**. You can return to edit any element of the email until the code is generated.

#### Configure a recommendations block

The recommendations block lets you set up algorithms and filters to deliver customized content that is rendered when the email is opened. To set up your recommendations block, follow the instructions in the Dynamic Yield [help docs](https://support.dynamicyield.com/hc/en-us/articles/4404013832465-Experience-Email#h_01FQBRAK3MEJCQ9MVGDZ32RA2T).

#### Configure a dynamic content block

Use Dynamic Content to direct different promotions and messages to different users. Targeting can be based on either affinity or audience. When the email is opened, Dynamic Yield decides which personalized experience to deliver. To set up your dynamic content block, follow the instructions in the Dynamic Yield [help docs](https://support.dynamicyield.com/hc/en-us/articles/4404013832465-Experience-Email#h_01FQBRAZ76DCD881CTC1MCJ2AM).

### Step 2: Prepare your code to integrate with MoEngage

This integration lets you embed personalized recommendation widgets and dynamic content powered by Dynamic Yield into your MoEngage email campaigns. Embedding the campaigns in MoEngage is simple — paste a single embed code into the MoEngage email editor.

Click the **Code Parameters** icon to set the ESP merge tags for all future emails:

![Dynamic Yield Code Parameters modal showing the ESP merge tag fields](https://support.dynamicyield.com/hc/article_attachments/4406402305937)

| Field | Description |
| --- | --- |
| `cuid_type` | Set the user ID type to match how you [identified your users](https://support.dynamicyield.com/hc/en-us/articles/360034332473-Identifying-Users-#identifying-users-via-events-0-1) (for example, by email address). |
| `cuid` | `{{UserAttribute['Email (Standard)']}}` |
| `locale` | `{{UserAttribute['Locale Language']}}` |
| `version` | `{{UserAttribute['version_number']}}` |

When the email is ready, generate the code and embed it in your MoEngage campaign:

1. In **Experience Emails**, click **Generate Code** in the campaign's row.

   ![Dynamic Yield Experience Emails list with the Generate Code button highlighted](https://support.dynamicyield.com/hc/article_attachments/4404708221457)
2. Click **Generate Code** again. The code is generated and a preview of the email is displayed.

   ![Dynamic Yield code generation modal showing the generated embed code and an email preview](https://support.dynamicyield.com/hc/article_attachments/4404016645777)
3. Click **Copy to Clipboard**.

Once you have the code, create a new MoEngage email campaign and paste the code into the email body.

### Step 3: Create a new MoEngage campaign

To include the code in your MoEngage campaign, on step 2 of campaign creation:

![MoEngage email campaign step 2 with the Custom HTML editor template selected](/images/partner_23084176021140.png)

1. Select the **Custom HTML editor** template, then click **Edit/Paste HTML template**.
2. Paste the code copied from Step 2.

Click the **Preview** icon to view the rendered campaign or test your campaign.
