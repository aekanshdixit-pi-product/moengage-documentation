# Introduction

[Aampe](https://www.aampe.com/) provides mobile app engagement and personalization infrastructure for short-form notifications and messaging.

# MoEngage × Aampe

The MoEngage and Aampe integration helps you to send more humane communications to each user by taking advantage of Aampe's personalized messaging software, which can turn your customer communication into a retention growth engine. Aampe makes use of MoEngage' Push APIs.

# Integration

<Info>
**Prerequisites**

Ensure you have access to your Aampe account.
</Info>

To integrate Aampe with MoEngage, follow these steps:

  1.      1. Go to your Aampe Dashboard, and in the sidemenu, look for Configure >> Push Provider. Look for MoEngage and click on the Activate switch next to it.

![Step-1.png](/images/partner_11203975042964.png)

     2. Aampe will then ask you to configure MoEngage details in a popup form:

![Step-2.png](/images/partner_11204028742804.png)

     3. Refer to the table below to fill out the above form.

| Field | Details

| --- | --- |  
| Workspace ID |  The Workspace ID of your MoEngage account is available at **Settings - > Account -> APIs ->** **Workspace ID**. The Data API ID and the Workspace ID of your MoEngage account are the same.  
| API secret |  API secret is the API key. The Data API key of your MoEngage account is available at **Settings - > Account -> APIs -> API keys **>**Data** API settings.  
     4. Click Activate to complete integration.![Step-3.png](/images/partner_11204129113108.png)
     1. You can now create campaigns in Aampe and send them via MoEngage.
