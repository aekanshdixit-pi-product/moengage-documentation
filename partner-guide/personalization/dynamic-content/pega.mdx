# Introduction

[Pega](https://www.pega.com/) is a leading provider of customer engagement and digital process automation software. Their platform enables businesses to build and deploy applications for customer service, sales automation, marketing, and operations. Pega's solutions use artificial intelligence (AI) and machine learning (ML) to analyze customer data, automate processes, drive efficiencies, and improve customer experience across various industries, including finance, healthcare, insurance, and telecommunications.

# MoEngage × Pega

The integration between Pega system and MoEngage allows businesses to automate end-to-end customer workflows and campaign orchestration. Pega's workflow automation capabilities combined with MoEngage's campaign management tools enable businesses to create, execute, and monitor complex, multi-step customer journeys seamlessly, ensuring consistent and relevant messaging throughout the customer lifecycle.

## Embed Pega content in MoEngage Campaigns

Using MoEngage's Content APIs, you can dynamically add offers, generated by Pega, inside MoEngage campaigns for a more personalized experience. 

### Step 1: Set up a MoEngage Content API

  1. Navigate to Settings APIs Content API  
![Screenshot_2022-03-08_at_6.21.00_PM.png](/images/partner_29522196676628.png)
  2. Add your Pega API details here. Refer to the [Pega API docs](https://docs-previous.pega.com/data-management-and-integration/87/pega-api-services) to select the API that best suites your use-case. 
  3. Click on "Save" and then "Done" once you see a successful response.  
_Note - You can also test your APIs. To test your APIs, from the Content API's action menu, click on "Test". You can your API response and validate it._




### Step 2: Display your content inside MoEngage campaigns

To use this API in your MoEngage campaigns,

  1. On step 2 on your campaign creation, enter @. 
  2. In the pop up, you can select the Pega Content API configured in Step 1.
  3. You can use [MoEngage Templating Language](https://help.moengage.com/hc/en-us/articles/115002757783-MoEngage-Templating-Language-JINJA-#content-api-0-3) to parse through the API response and use the content in your campaigns.   
_Note that - You can click on the Preview tab to view your campaign in real-time personalization._
  4. You can go ahead and Publish the campaign.
