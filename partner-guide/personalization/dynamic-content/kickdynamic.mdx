# Introduction

[Kickdynamic](https://www.tigergraph.com/kickdynamic/) enables users to create live images for use in email. The platform is built to enable users to create everything they need directly in the UI with no customization.

# MoEngage × Kickdynamic

With the MoEngage and Kickdynamic integration, you can enrich your emails with dynamic content that is personalized and rendered upon the opening of your email.

# Integration

## Step 1: Create a Personalized Image Template in Kickdynamic

  1. On your Kickdynamic dashboard, navigate to the Templates area of the platform, then Content Automation, and select ‘+ New Template’.
  2. Name the template in the Description. To create a branded Personalized image in the Kickdynamic platform, ensure you have already designed a background image. Then, click on the canvas and use the menu options on the right-hand side to upload the created background image.  
![](https://675544707-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-LaLlWyy5jddi_WjlHBr%2F-M1_ylEtBN4bbufZnl96%2F-M1a01vS4AVufcMEnru5%2FPI%20Template%20-%20Image%203.png?alt=media&token=89f3d15c-4178-43fc-812b-66beba7ecdcd)
  3. To add the first layer of personalization, select the ‘Add Block’ button and a block will appear in the center of the canvas.
  4. Then select ‘Connect Source’, highlighted below:  
![](https://675544707-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-LaLlWyy5jddi_WjlHBr%2F-M1_ylEtBN4bbufZnl96%2F-M1a07NzrDrNkZdJfIr2%2FPI%20Template%20-%20Image%205.png?alt=media&token=3c42e93e-d6f2-4fcb-93da-7099eefd3180)
  5. To display the personalization, connect to the URL parameter which allows you to merge any data you store in MoEngage. 
  6. Name the parameter something identifiable to the user while building this out. For example, for first name personalization, it could be named `firstname` or `first_name` like in the example below:  
![](https://675544707-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-LaLlWyy5jddi_WjlHBr%2F-M1_ylEtBN4bbufZnl96%2F-M1a0Cq-UH2Qdvol-VwP%2FPI%20Template%20-%20Image%207.png?alt=media&token=fb647153-4349-42b9-b7ea-b405db36a507)
  7. You can look at the additional functions provided by Kickdynamic [here](https://help.kickdynamic.com/widgets/personalised-image-overview/creating-a-personalized-image-template#additional-functionality).



## Step 2: Create a Personalized Image Campaign in Kickdynamic

  1. On your Kickdynamic dashboard, navigate to a campaign / create a new campaign and select the Content Automation widget.
  2. Name the mailing and/or tag and select the right template:  
![](https://675544707-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-LaLlWyy5jddi_WjlHBr%2F-M1a4we2xSlSGpIU-VXN%2F-M1aBRI6PvBnea7Yugys%2FPI%20Campaign%20-%20Image%202.png?alt=media&token=e4ee424e-af12-43c5-8bf6-7a98b7cfeef8)
  3. Define the link where you want the image to link to and click next:  
![](https://675544707-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-LaLlWyy5jddi_WjlHBr%2F-M1a4we2xSlSGpIU-VXN%2F-M1aBYHW8WkuG6bKt-bZ%2FPI%20Campaign%20-%20Image%203.png?alt=media&token=15aa2331-dd1f-4fa0-9ad9-1f79df6c1e97)
  4. All URL parameters linked in the template will automatically pull through into the Tag like so:  
![](https://675544707-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-LaLlWyy5jddi_WjlHBr%2F-M1a4we2xSlSGpIU-VXN%2F-M1aBa1Fo9nfuru3pOtY%2FPI%20Campaign%20-%20Image%204.png?alt=media&token=40ad1ca2-86b6-4662-8287-5efb3af2fa95)
  5. Within the generated Tag, replace `value` with MoEngage Personalization code like so:


```html
&lt;a href="https://img.kickdyn.com/clickimgdf.ashx?c=28495.TxzzAhrqBwPFnJiC5zNy&firstname=%7B%7BUserAttribute%5B'First%20Name'%5D%7D%7D"&gt;
<img src="https://img.kickdyn.com/imgdf.png?c=28495.TxzzAhrqBwPFnJiC5zNy&firstname=%7B%7BUserAttribute%5B'First%20Name'%5D%7D%7D" width="640" height="427" border="0"/> &lt;/a&gt;
```

At the time of opening the email, the merge code will pull through the data associated with the particular subscriber who is opening. The value will tell the Kickdynamic tag what to display, e.g. ‘Katie’ or ‘Henry’ for first name merges.
