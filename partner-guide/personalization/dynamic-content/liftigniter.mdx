# Introduction

[LiftIgniter](https://www.liftigniter.com/) applies machine learning to first-party behavioral data (without invading privacy) to personalize product and content recommendations on your site.

# MoEngage × LiftIgniter

With the MoEngage and LiftIgniter integration, you can send out personalized recommended content in MoEngage Campaigns to help boost adoption and CTRs.

# Integration

| library_add_check |  **Prerequisites**

  * You would need your LiftIgniter API token to use their APIs.
  * You would need to know your LiftIgniter JavaScript key.

| --- | --- |  
  
Using MoEngage's Content APIs, you can dynamically pull recommendations inside MoEngage campaigns for a more personalized experience.

### Step 1: Set up a MoEngage Content API

  1. Navigate to **Settings** -> **Advanced settings** -> **Content API**. Click **\+ Add content API** at the top right corner of the Content API screen.![ContentAPI_Navogation.png](/images/partner_21979738782100.png)
  2. Add your LiftIgniter API to the endpoint URL. Depending on your use case, the endpoint will change.



![](/images/partner_20130868953364.png)

To display your customer's recommendation, use LiftIgniter's [Model Query API](https://documenter.getpostman.com/view/2166502/liftigniter-api-collection/7TFGvSV#340404f6-68fd-4844-90d9-7b93994440b9).

While setting up your content API, insert the following URL:
    
    
    https://query.petametrics.com/v3/&lt;jsk&gt;/&#123;&#123;UserAttribute['ID']&#125;&#125;/model

Remember to change the `&lt;jsk&gt;` with your [Javascript Key](https://console.liftigniter.com/settings/keys). Keep the URL parameters empty.

Under Headers, add the following:

  1.      1. `x-api-key`: For this, you will need your [API Key](https://documenter.getpostman.com/view/2166502/liftigniter-api-collection/7TFGvSV#intro) from LiftIgniter.
     2. `Content-Type`: `application/json`
  2. Click on "Save" and then "Done" once you see a successful response.
  3. Note - You can also test your APIs. To test your APIs, from the Content API's action menu, click on "Test". You can your API response and validate it. 



### Step 2: Display your recommendations inside MoEngage campaigns

To use this API in your MoEngage campaigns,

  1. On step 2 on your campaign creation, enter @. 
  2. In the pop up, you can select the LiftIgniter Content API configured in Step 1.
  3. You can use [MoEngage Templating Language](https://help.moengage.com/hc/en-us/articles/115002757783-MoEngage-Templating-Language-JINJA-#content-api-0-3) to parse through the API response and use the content in your campaigns.  
![](/images/partner_20131004588436.png)
  4. You can click on the Personalized tab under "Preview type" to view your campaign in real-time personalization:  
![](/images/partner_20131075562004.png)
  5. You can go ahead and Publish the campaign.
