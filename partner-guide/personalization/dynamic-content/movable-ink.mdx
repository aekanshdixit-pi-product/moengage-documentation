## Introduction

[Movable Ink](https://movableink.com/) is a cloud-based software platform that gives digital marketers a way to create compelling, unique visual experiences. The Movable Ink platform provides customization options that you can drop into your campaigns.

## Scope

Expand MoEngage's creative capabilities by leveraging Movable Ink's Intelligent Creative features — polling, countdown timers, scratch-off, and more. The Movable Ink and MoEngage integration powers a more well-rounded approach to dynamic, data-driven messages, providing users with real-time elements about the things that matter.

## Benefits and use cases

The Movable Ink and MoEngage integration helps you level up your messaging strategy:

1. Automatically create personalized content for every customer at the moment of engagement. For example:
   - Use the Countdown Timer feature to notify users when a big sale is approaching (Black Friday, Valentine's Day, holiday deals, and so on).
   - Use the Scratch Off feature as a fun, interactive way to distribute promo codes.
   - Build creative for rich push notifications with real-time data from APIs, the web, CSV, CRM sources, and more.
2. Easily access your data to create unique and innovative content that resonates with your customers.
3. Target and generate relevant content using real-time data from customer profile, inventory, location, and more.

## Integration

<Info>
  **Prerequisites**

  - Login credentials for your Movable Ink account.
</Info>

MoEngage supports including Movable Ink content in emails, push notifications, and in-app campaigns. First create the campaign content in Movable Ink, get the image source URL, and then reference the URL in your MoEngage campaigns.

### Step 1: Create a data source for Movable Ink

Create a data source — it can be a CSV, website import, or API integration. Read more about data sources in [Movable Ink's documentation](https://support.movableink.com/hc/en-us).

![Movable Ink data source creation screen](/images/partner_4421164890132.png)

### Step 2: Create a campaign on the Movable Ink platform

From the Movable Ink home screen, create a campaign. You can choose from email-from-HTML, email-from-image, or a block that can be used in any channel — including push, in-app, and Content Cards (suggested). We also recommend reviewing the various content options available through blocks.

![Movable Ink campaign creation screen with the available campaign types](/images/partner_4421153293076.png)

Movable Ink has an easy editor for dragging and dropping elements such as text and images. If you have populated your data source, you can dynamically generate an image using the data properties. You can also create fallbacks for users who do not match the personalization criteria.

![Movable Ink editor with personalization data being mapped to image elements](/images/partner_4421165077012.png)

Before you finish, preview the dynamic images and test the query parameters to see what the images will look like. Once complete, Movable Ink generates a dynamic URL that you can use in MoEngage.

For more information, see the [Movable Ink documentation](https://support.movableink.com/hc/en-us).

### Step 3: Obtain the Movable Ink content URL

To include Movable Ink content in MoEngage campaigns, locate the source URL Movable Ink provides:

1. Go to the **Finish & Export** tab to export your creative tag.
2. Copy the image URL (`img src`) from the creative tag.

![Movable Ink Finish & Export tab with the image URL highlighted](/images/partner_4421174149652.png)

### Step 4: Use the Movable Ink URL in MoEngage campaigns

To include the Movable Ink image URL in your MoEngage campaign, on step 2 of campaign creation:

#### Push campaigns

<Tabs>

    Paste the URL into the **Image URL** or **Rich Landing URL** field, depending on your use case.

  </Tab>
  <Tab title="Android">
    Paste the URL into the **Image URL** or **Rich Landing URL** field, depending on your use case.

    ![MoEngage Android push editor with the Movable Ink image URL pasted in](/images/partner_4421176055572.png)
  </Tab>
  <Tab title="Web">
    Paste the URL into the **Image URL** field.

    ![MoEngage Web push editor with the Movable Ink image URL pasted in](/images/partner_4421176436884.png)
  </Tab>
</Tabs>

#### In-app campaigns

Insert the Movable Ink URL into your in-app image URL field.

![MoEngage in-app campaign editor with the Movable Ink image URL configured](/images/partner_4421156145172.png)

#### Email campaigns

Insert the Movable Ink personalized URL into your email content via the code editor.

![MoEngage email code editor with the Movable Ink URL inserted into the HTML](/images/partner_4430361651092.png)

#### RCS campaigns

Paste the URL into the **Image URL** field.

![MoEngage RCS campaign editor with the Movable Ink image URL pasted in](/images/partner_43544630371732.png)

#### WhatsApp campaigns

Paste the URL into the **Image URL** field.
