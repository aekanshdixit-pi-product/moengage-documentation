# Introduction

[Worth AI](https://worthy.ai/) is an underwriting platform that leverages AI algorithms to deliver lightning-fast processing times, unparalleled accuracy, and a unique user experience.

# MoEngage <> Worthy

The Worthy and MoEngage integration allows you to easily create personalized, rich in-app experiences using Worthy’s drag-and-drop editor and deliver them through MoEngage campaigns.

Additionally, Worthy will automatically do the following:

  * Create a Connected Content server and secured API for your messaging.
  * Construct your in-app messages with analytics and click-tracking that will appear directly in MoEngage.
  * Automatically export HTML via Worthy’s drag-and-drop editor to use in Custom Code in-app message campaigns in MoEngage, complete with the required API connections and dynamic content you configure.



# Use cases

  * Custom welcome experiences based on user onboarding selections
  * In-app experiences for special events and promotions
  * Gathering customer feedback and ratings based on app behavior
  * Quickly testing potential app product ideas
  * Rich notices, news, and community updatesa



# Integration<Info>
**Prerequisites**
* You would need admin access to your Worthy account.
</Info>## Step 1: Create personalized messaging in Worthy

Navigate to your app in the Worthy dashboard, select the Message Creator, and create a personalized message you want to use to engage your users.

## Step 2: Create a MoEngage campaign

Create an [in-app campaign in MoEngage.](https://help.moengage.com/hc/en-us/articles/206184196-Create-In-App-Campaign) Select _Start From Scratch_ tab _.  
![Screenshot 2025-02-14 at 4.34.29 PM.png](/images/partner_34807870395540.png)_

## Step 3: Copy your personalized message into MoEngage

In the Worthy message creator,

  1. Click Export
  2. Copy the exported content and paste in the Campaign Step 2 editor. 



That’s it! You can immediately test your personalized message.
