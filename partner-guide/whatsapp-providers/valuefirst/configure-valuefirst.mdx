MoEngage has native integration for WhatsApp with ValueFirst. This integration allows businesses to send templated WhatsApp messages powered by ValueFirst. MoEngage can send messages and show Delivery and Read Analytics.  
  
## Sender Configuration

To configure a Sender from ValueFirst on the MoEngage Dashboard, do the following:

  1.      1.         1. Go to Settings->WhatsApp->Sender Configuration.
        2. Click +Add connector on the left menu. 
        3. Choose ValueFirst from the list of WA providers or connectors available in the dropdown available and click _Done_. 

![ValueFirst_ChooseConnector.png](/images/partner_18328807133588.png)

        4. Click on \+ Sender at the top right corner of the screen to configure the sender details.

![ValueFirst_SenderConfig.png](/images/partner_18093478902164.png)

5. Add the following details:

| Field Name | Description |
| --- | --- |
| Sender name | This field denotes the name you want to provide to the Sender profile. This would be available in the Sender so that you can recognize this easily while using it to create a campaign inside MoEngage. The _Sender Name_ should be between 5-50 characters. This is a mandatory field. |
| WhatsApp business number | This field denotes the phone number registered with WhatsApp (directly or via ValueFirst) using which you want to send out WhatsApp Messages to your users. This is the same business number that you have boarded on the ValueFirst platform. This is a mandatory field and is also referred to as the WABA number. |
| Authorization | This field denotes the _Basic Token_ available in the Configuration section of the ValueFirst plugin. For more information, refer to Configuration Steps. 6. Upon successful integration, you will get the following message. |

![SaveIntegration.png](/images/partner_18067308059924.png)




## Steps to configure MoEngage on ValueFirst

  1.      1.         1. Sign-up in the application portal(link shared below) with your name, email, and company details. 
               
               https://moengage.vfplugin.com/signup

        2. Log in to the application portal using email and password given in the Signup Form.
        3. Click the ‘ValueFirst WhatsApp’ tab and configure ValueFirst Username, Password & WhatsApp Phone number. 

![VF_Login.png](/images/partner_18329918462100.png)

        4. You can copy the Inbound webhook from the portal and share it with the ValueFirst sales or support team.
        5. Click the ‘MoEngage WhatsApp’ tab and copy the Basic Token, which is used as Authorization in Sender Configuration, as shown in the image below. 

![ValueFirst_Steps \(1\).png](/images/partner_18069170183060.png)




## Delivery Tracking

To track the delivery of your WhatsApp Messages inside MoEngage, you would need to copy MoEngage Delivery Tracking URL and add it to the Configuration settings in ValurFirst. Do the following:

  1.      1.         1. You should be able to see the URL as shown below in MoEngage by navigating to Settings->WhatsApp->Sender Configuration-> ValueFirst. 

![ValueFirst_EnableDT.png](/images/partner_18069481393556.png)

        2. Copy the exact URL from your MoEngage Dashboard.
        3. Log in to the Connector app and go to the ‘MoEngage WhatsApp’ tab.
        4. Configure Status Callback and Incoming Callback by pasting the copied Delivery Tracking URL from MoEngage WhatsApp connector, as shown below. 

![ValueFirst_DTConfig.png](/images/partner_18069637665556.png)

        5. You should have pre-approved templates with ValueFirst to create a WhatsApp campaign for a ValueFirst Sender in MoEngage. You can create templates from the ValueFirst platform in the Approved Templates section.
        6. You can create the same template verbatim in MoEngage. For more information, refer to [Add Templates](https://help.moengage.com/hc/en-us/articles/4951072814100-WhatsApp-Templates).
