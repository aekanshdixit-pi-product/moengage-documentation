title: "Configure Gallabox"
---

MoEngage has native integration for WhatsApp with Gallabox. This integration allows businesses to send templated WhatsApp messages powered by Gallabox. MoEngage can send messages show Delivery and Read Analytics.

## Sender Configuration

To configure a Sender from Gallabox on the MoEngage Dashboard, do the following:

  1.      1.         1. Go to Settings->WhatsApp->Sender Configuration.
        2. Click +Add connector on the left menu.
        3. Choose Gallabox from the list of WA providers or connectors in the dropdown and click _Done_. 

![WA_Gallabox.png](/images/partner_18391710233876.png)

        4. Click on \+ Sender at the top right corner of the screen to configure the sender details.

![WA_GallaboxConfig_MoE.png](/images/partner_18391764774164.png)

|         5. Add the following details:  Field Name | Description

| --- | --- |  
| Sender name |  This field denotes the name you want to provide to the Sender profile. This would be available in the Sender so you can recognize this easily while creating a campaign inside MoEngage. The _Sender Name_ should be between 5-50 characters. This is a mandatory field.  
| WhatsApp business number |  This field denotes the phone number registered with WhatsApp (directly or via Gallabox) using which you want to send out WhatsApp Messages to your users. This is the same business number you have boarded on the Gallabox platform. This is a mandatory field and is also referred to as the WABA number. To get the WABA Number from Gallabox:
           1. Navigate to Settings -> WhatsApp Channel -> WhatsApp
           2. Copy the WhatsApp Number and paste it here.  
| Authorization |  This field denotes the API Key and will be used to identify the account making the API call. For more information, refer to Configuration Steps.  
        6. Upon successful integration, you will get the following message.

![SaveIntegration.png](/images/partner_18391676705556.png)




## Steps to configure MoEngage on Gallabox

  1.      1.         1. Sign in to your Gallabox account.

        2. Click on the Integrations in the left navigation pane.

        3. Choose MoEngage from the Available apps and click on integration, as shown below.![WA_Gallabox_Step1.png](/images/partner_18392225089172.png)

        4. In the integration window that opens, type the name you wish to add for this integration and click _Connect_.![WA_Gallabox_Step2.png](/images/partner_18392235896084.png)
        5. You will be redirected to the Configuration Page of MoEngage Integration. Click the _Configuration_ tab and select the channel as Whatsapp. ![WA_Gallabox_Step3.png](/images/partner_18392235897364.png)
        6. Add the delivery callback URL available in MoEngage in the _Status URL_ field. For more information, refer to Delivery Tracking.
        7. Click Update Configuration. Once the configuration is updated, the API Secret will be available in the Configuration section, as shown below. ![WA_Gallabox_Step4 \(1\).png](/images/partner_18392256616340.png)
        8. Copy the API Secret and paste it into the Authorization field on the MoEngage Dashboard. For more information, refer to Step 5 in the Sender Details section.



## Delivery Tracking

To track the delivery of your WhatsApp Messages inside MoEngage, you would need to copy the MoEngage Delivery Tracking URL and add it to the Configuration settings in Gallabox. Do the following:

  1.         1.            1. You should be able to see the URL as shown below in MoEngage by navigating to Settings->WhatsApp->Sender Configuration-> Gallabox. 

![WA_Gallabox_DT.png](/images/partner_18391736599316.png)

           2. Copy the exact URL from your MoEngage Dashboard.
           3. Paste it in the Status URL field in the Gallabox configuration window.
