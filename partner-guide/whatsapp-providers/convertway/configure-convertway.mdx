title: "Configure ConvertWay"
---

MoEngage natively integrates with ConvertWay to send‬ WhatsApp templates‬‭ over Flows and campaigns directly‬‭ from MoEngage to end users. This‬ enhances customer engagement by leveraging WhatsApp as a communication channel for‬ automated messages and marketing campaigns.‬

Configuring ConvertWay with MoEngage is a two-step process:

Step 1: [Sender Configuration](#sender-configuration)

Step 2: [Delivery Tracking](#delivery-tracking)

<Info>
  Before integration, ensure you have the following:

  - A MoEngage account
  - A ConvertWay (CW) account
  - A WhatsApp Business API (WABA) number
</Info>

# Sender Configuration

To configure a Sender from ConvertWay on the MoEngage Dashboard, follow the steps below:

1. Go to **Settings** **\>** **WhatsApp \>** **Sender Configuration**.
2. Click the **\+Add connector** from the left pane under the **Sender configuration** tab.

<Info>
  You can choose ConvertWay from the left pane if you have already added ConvertWay to your connector list.\
  <img src="/images/partner_35797513598740.png" alt="image (19).png" />
</Info>

3. Select **ConvertWay** from the **Choose a connector** dropdown and then click **Done**.\
   <img src="/images/partner_35797530059412.png" alt="Convertway.png" />
4. Add the following details:

| Field Name | Description |
| --- | --- |
| Sender Name | <ul><li>Provide a name to the Sender profile.</li><li>This would be available in the Sender so that you can recognize this easily while using it to create a campaign inside MoEngage.</li><li>The **Sender Name** is a mandatory field that should be between 5-50 characters.</li></ul> |
| WhatsApp business number | <ul><li>Enter the phone number with the country code registered with WhatsApp (directly or through ConvertWay) using which you want to send out WhatsApp Messages to your users.</li><li>In the ConvertWay account, you can find the sender details in **Settings** \> **WhatsApp Business Account**.</li><li>This is a mandatory field and is also referred to as the WABA number.</li></ul> |
| API URL | Collect the API URL from your vendor and enter it into the **API URL** field. Sample URL: <br />[**https://app.theconvertway.com/api/v1/send\_messages/moengage/s‬**](https://app.theconvertway.com/api/v1/send_messages/moengage/s‬)**‭ endWhatsapp‬** |
| Authorization | **Basic CW\_LICENSE\_KEY** You can find the CW\_LICENSE\_KEY under **Settings** \> **Profile** in‬ ‭ your ConvertWay account. |

5. Click **Save**.
6. Notify ConvertWay to enable/validate the API token.
7. Upon successful integration, you get the following message.\
   <img src="/images/partner_35364754398740.png" alt="SaveIntegration.png" />
