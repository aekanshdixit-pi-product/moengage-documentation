MoEngage has native integration for WhatsApp with Interakt. This integration allows businesses to send templated WhatsApp messages powered by Interakt. MoEngage can send messages and show Delivery and Read Analytics.

## Sender Configuration

To configure a Sender from Interakt on the MoEngage Dashboard, do the following:

  1.      1.         1. Go to Settings->WhatsApp->Sender Configuration.
        2. Choose Interakt from the left menu and click on \+ Sender![INterakt_SenderConfig.png](/images/partner_17963643284372.png)
3. Add the following details:

| Field Name | Description |
| --- | --- |
| Sender name | This field denotes the name you want to provide to the Sender profile. This would be available in the Sender so that you can recognize this easily while using it to create a campaign inside MoEngage. The _Sender Name_ should be between 5-50 characters. This is a mandatory field. |
| WhatsApp business number | This field denotes the phone number registered with WhatsApp (directly or via Interakt) using which you want to send out WhatsApp Messages to your users. This is the same business number that you have boarded on the Interakt platform. This is a mandatory field and is also referred to as the WABA number. ![Haptik_WABA.png](/images/partner_17963304357396.png) |
| API key | Contact the Interakt support team with the following details or send an email to [hello@interakt.ai:](mailto:hello@interakt.ai:) * Delivery Tracking URL from MoEngage account * WhatsApp Org name * WhatsApp Phone number The support team will activate the integration for you with the above details. After activation, Interakt will share a bearer token which should be used as the API key. |



## Delivery Tracking

To track the delivery of your WhatsApp Messages inside MoEngage, you would need to copy MoEngage Delivery Tracking URL and share it with Interakt and get it updated on their side. You should be able to see the URL as shown below:

![Interakt_DTURL.png](/images/partner_17963611576596.png)

<Info>
The URL is a sample URL. Copy the exact URL from your MoEngage Dashboard before you share it with the Interakt team.
</Info>
Upon successful integration, you will get the following message.

![SaveIntegration.png](/images/partner_17963253400212.png)

<Info>
* You should have pre-approved templates with Interakt to create a WhatsApp campaign for an Interakt Sender in MoEngage. You can create templates from the Interakt platform by selecting the Template Management option in the left menu in [Interakt Template Management.](https://www.interakt.shop/resource-center/whatsapp-message-notification-template-creation)
  * You can create the same template verbatim in MoEngage. For more information, refer to [Add Templates](/user-guide/campaigns-and-channels/whatsapp/getting-started-with-whatsapp/whatsapp-templates).
</Info>
