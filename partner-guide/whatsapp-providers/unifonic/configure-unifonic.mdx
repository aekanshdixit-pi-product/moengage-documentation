MoEngage offers native integration for WhatsApp through Unifonic, enabling businesses to send templated messages via WhatsApp numbers powered by Unifonic. MoEngage provides delivery and read analytics for sent messages using this integration.

## Sender Configuration

To configure a Sender from Unifonic on the MoEngage Dashboard, do the following:

  1.      1.         1. Go to **Settings** ->**WhatsApp** ->**Sender Configuration**.
        2. Click **+Add connector** on the left menu. 
        3. Choose **Unifonic** from the list of WhatsApp providers or connectors available in the dropdown and click **Done**. 

![Unifonic.png](/images/partner_21322771184404.png)

        4. Click on **\+ Sender** at the top right corner of the screen to configure the sender details.

![Unifonic_SenderConfiguration.png](/images/partner_21322771185044.png)

5. Add the following details:

| Field Name | Description |
| --- | --- |
| **Sender name** | This field denotes the name you want to provide to the Sender profile. This would be available in the Sender so that you can recognize this easily while using it to create a campaign inside MoEngage. The _Sender Name_ should be between 5-50 characters. This is a mandatory field. |
| **WhatsApp business number** | This field denotes the phone number registered with WhatsApp (directly or via Unifonic) using which you want to send out WhatsApp Messages to your users. This is the same business number that you have boarded on the Unifonic platform. This is a mandatory field and is also referred to as the WABA number. |
| **API URL** | This field denotes the Unifonic API URL to which the WhatsApp requests would be sent from MoEngage. |
| **public-id** | This field denotes the public key that is required for the Unifonic Whatsapp API requests. Follow the steps mentioned in [Get your API Access](https://docs.unifonic.com/docs/getting-your-api-access) to get your public key. |
| **secret** | This field denotes the secret key that is required for the Unifonic Whatsapp API requests. Follow the steps mentioned in [Get your API Access](https://docs.unifonic.com/docs/getting-your-api-access) to get your secret key. |
| **template-namespace** | Reach out to Unifonic support or your Account Manager to provide you with the Template namespace value. 6. Upon successful integration, you will get the following message. |

## Delivery Tracking

To track the delivery of your WhatsApp Messages inside MoEngage, you would need to copy MoEngage Delivery Tracking URL and add it as a WebHook in WhatsApp Settings in Unifonic. Do the following:

  1.      1.         1. You should be able to see the URL as shown below in MoEngage by navigating to Settings->WhatsApp->Sender Configuration-> Unifonic. 

![Unifonic_DT.png](/images/partner_21322903654676.png)

        2. Copy the exact URL from your MoEngage Dashboard.
        3. Follow the steps mentioned in [Configuring WebHook URLs](https://docs.unifonic.com/docs/message-delivery-status).
        4. You should have pre-approved templates with Unifonic to create a WhatsApp campaign for a Unifonic Sender in MoEngage.
        5. You can create templates verbatim to the pre-approved Unifonic templates in MoEngage. For more information, refer to [Add Templates](/user-guide/campaigns-and-channels/whatsapp/getting-started-with-whatsapp/whatsapp-templates).
