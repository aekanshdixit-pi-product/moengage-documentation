# Introduction

[OneXtel](https://www.onextel.com/) is a leading Communication Platform as a Service (CPaaS) provider, offering seamless, end-to-end communication solutions. Their product offering includes channels such as SMS, Voice, WhatsApp, Chatbot, RCS, and Email services tailored for businesses of all sizes.

MoEngage natively integrates with OneXtel to send WhatsApp templates directly from MoEngage to end users via Flows and campaigns. This enhances customer engagement by leveraging WhatsApp as a communication channel for automated messages and marketing campaigns.

<Info>
  Before integration, ensure you have the following:

  - An active MoEngage account
  - An active OneXtel WhatsApp account on 365cx.io
  - A WABA Number (WhatsApp Business API number)
</Info>

## Step 1: Sender Configuration

To configure a sender from OneXtel WhatsApp, perform the following steps:

### **Step 1.2 Obtain the API Key from OneXtel**

Before configuring MoEngage, generate the API key from the OneXtel portal.

1. Log in to your OneXtel portal account.
2. Navigate to the **API** section.
3. Click **Show API Key** to view your existing key.
4. If no key is available, click **Generate API Key** to create a new one. <img src="/images/partner_42821009807636.png" />

### Step 1.2 Create a Submission URL

After creating the channel in OneXtel, you will receive the Channel ID. Use this ID to construct the Submission URL for the MoEngage Custom Connector by replacing the placeholder in the following URL structure: [https://365cx.io/relaybridge/webhook/whatsapp/moengage/\<CHANNEL\_ID\>](https://365cx.io/relaybridge/webhook/whatsapp/moengage/<CHANNEL_ID>).

**Example**

If your Channel ID is `665815084ebe71d4bfe6a950`, your Submission URL is:

[https://365cx.io/relaybridge/webhook/whatsapp/moengage/665815084ebe71d4bfe6a950](https://365cx.io/relaybridge/webhook/whatsapp/moengage/665815084ebe71d4bfe6a950).

### Step 1.3 Create a WhatsApp Channel in MoEngage

1. On the left navigation menu in the MoEngage UI, click **Settings** \> **Channels** \> **WhatsApp**.
2. On the **Sender configuration** tab, click the **\+ Add connector** from the left pane.
3. Select **Dev** from the list.

![](/images/partner_42820998719892.png)

## Step 2: Delivery Tracking

To enable delivery tracking for your WhatsApp messages within MoEngage, perform the following steps:

1. On the left navigation menu in the MoEngage UI, click **Settings** \> **Channels** \> **WhatsApp**.
2. On the **Sender configuration** tab, under **Enable delivery tracking** , copy the delivery report callback URL.
3. Share this URL with the OneXtel Support Team for final setup and delivery monitoring.

### Step 2.1: Create a WhatsApp Template

To create a WhatsApp template, perform the following steps:

1. On the left navigation menu in the MoEngage UI, click **Settings** \> **Channels** \> **WhatsApp**.
2. On the **WhatsApp integration settings** page, click the **Approved templates** tab.\\ <img src="/images/partner_42820996760468.png" />
3. In the upper-right corner, click **\+ Template**.
4. On the template creation page, provide the required details:

| Field | Description |
| --- | --- |
| **Enter Template Name** | Enter a name for the template |
| **Service Provider** | Select the appropriate provider |
| **Sender profile** | Select a registered sender profile |
| **Language** | Select the language for the template body |
| **Category** | Select the template category |
| **Template type** | Select the required format |
| **Body** | Enter the text content for the message |

5. Click **Save** to submit the template for approval.

![](/images/partner_42820998720148.png)
