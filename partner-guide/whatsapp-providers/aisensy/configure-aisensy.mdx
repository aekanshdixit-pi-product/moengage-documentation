MoEngage integrates with AiSensy for WhatsApp. This integration allows businesses to send templated WhatsApp messages using Senders powered by AiSensy. MoEngage can send messages and show Delivery and Read Analytics.

## Sender Configuration

To configure a Sender from AiSensy on the MoEngage Dashboard, do the following:

  1.      1.         1. Go to Settings->WhatsApp->Sender Configuration.
        2. Click +Add connector on the left menu.
        3. Choose AiSensy from the list of WA providers or connectors available in the dropdown available and click _Done_. 

![AiSensy.png](/images/partner_16579679366932.png)

4. Click on \+ Sender at the top right corner of the screen to configure the sender details, as explained below.![AiSensy_Configuration.png](/images/partner_16579655876372.png)

| Field Name | Description |
| --- | --- |
| Sender name | This field denotes the name you want to provide to the Sender profile. This would be available in the Sender so that you can recognize this easily while using it to create a campaign inside MoEngage. The _Sender Name_ should be between 5-50 characters. This is a mandatory field. |
| WhatsApp business number | This field denotes the phone number registered with WhatsApp (directly or via AiSensy) using which you want to send out WhatsApp Messages to your users. This is a mandatory field and is also referred to as the WABA number. On AiSensy, you can find Sender Details on Channels & Numbers ->WhatsApp ->Senders. |
| authentication | This field denotes the API Key and will be used to identify the account that is making the API call. Provide the API Key of your AiSensy account here. To get your API Key: * Login to your AiSensy account. * Go to the integration section * Select MoEngage. * In the Get Started tab, navigate to the Connect MoEngage section. * Click on '_Click to copy'_ to copy the API Key. * Paste the API Key in the authentication field in the Sender Configuration details on the MoEngage Dashboard. ![moengage2.png](/images/partner_16551014107540.png) |
| delivery-report-callback-url | To track the delivery of your WhatsApp Messages in MoEngage, copy the MoEngage Delivery Tracking URL and pass it to the AiSensy team. You should be able to see the URL as shown below: * **Note** : The URL is a sample URL. Copy the exact URL from your MoEngage Dashboard while configuring the sender. To add the delivery tracking URL to AiSensy: * Login to your AiSensy account. * Go to the integration section * Select MoEngage. * In the Get Started tab, navigate to the Connect MoEngage section. * Copy and paste the delivery tracking URL provided by MoEngage (in the _Enable delivery tracking_ section) to the Enter MoEngage Webhook URL field in the AiSensy Dashboard. ![moengage2.png](/images/partner_16551014107540.png) |
  
        5. Click Save to save the sender's details. Upon successful integration, you will get a success message.

![SuccessMsg.png](/images/partner_16585441670548.png)




<Info>
* You should have pre-approved templates with AiSensy to create a WhatsApp campaign for an AiSensy Sender in MoEngage.
  * You can create the same template verbatim in MoEngage. For more information, refer to [Add Templates](/user-guide/campaigns-and-channels/whatsapp/getting-started-with-whatsapp/whatsapp-templates).
</Info>
