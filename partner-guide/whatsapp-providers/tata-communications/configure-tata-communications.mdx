MoEngage has a native integration for WhatsApp with Tata Communications, which means that MoEngage can send messages and show Delivery and Read Analytics. This integration allows businesses to send Templated WhatsApp messages using Tata Communications.

# Sender Configuration

To configure a Sender from Tata Communications on the MoEngage Dashboard:

  1.      1.         1. Navigate to _Settings- >WhatsApp->Sender Configuration_.
        2. Click _+Add connector_ on the left menu.
        3. Choose _TataCommunications_ from the list of WA providers or connectors in the dropdown and click _Done_. 

![WAConnector.png](/images/partner_19097905869972.png)

        4. Click on \+ Sender at the top right corner of the screen to configure the sender details.

![SenderConfig_TataComms.png](/images/partner_19097905879572.png)

|         5. Add the following details:  Field Name | Description

| --- | --- |  
| Sender name |  The  _Sender name_ is a name you want to provide to the Sender profile so that you can recognize this easily while using it to create a campaign inside MoEngage. The  _Sender Name_ should be between 5-50 characters.  
| WhatsApp business number |  It is the phone number registered with WhatsApp using which you want to send out WhatsApp Messages to your users. This is a mandatory field and is also referred to as the WABA number.  
Authorization

|  Enter the API Key in _Authorization._ To receive your API key, please connect with the Tata Communications team offline, and drop an email at [designanddelivery@tatacommunications.com ](mailto:designanddelivery@tatacommunications.com)mentioning “ _&lt; Your Company Name&gt; | WhatsApp API Key for MoEngage”_ in the subject line to receive your API key. Once you receive the API key, paste it in the _Authorization_ field. <br/>- Enable Delivery Tracking |  To track the delivery of your WhatsApp Messages inside MoEngage, you would need to copy the MoEngage Delivery Tracking URL (available in this section) and share it with the Tata Communications team, and get it updated on their side.  ![DT_TataComms.png](/images/partner_19097895179924.png) **Note** : This is a sample URL. Copy the exact URL from your MoEngage Dashboard and share it with the Tata Communications team. |

|  **Note** | |


  * You should have pre-approved templates with Tata Communications to create a WhatsApp campaign for a Tata Communications Sender in MoEngage.
  * You can create the same template verbatim in MoEngage. For more information, refer to [Add Templates](https://help.moengage.com/hc/en-us/articles/4951072814100-WhatsApp-Templates).
