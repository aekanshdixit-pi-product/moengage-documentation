MoEngage integrates with Route Mobile for WhatsApp. This integration allows businesses to send templated WhatsApp messages using Senders powered by Route Mobile. MoEngage can send messages and show Delivery and Read Analytics.  
  
## Sender Configuration

To configure a Sender from Route Mobile on the MoEngage Dashboard, do the following:

  1.      1.         1. Go to Settings->WhatsApp->Sender Configuration.
        2. Click +Add connector on the left menu.
        3. Choose Route Mobile from the list of WA providers or connectors available in the dropdown available and click _Done_. 

![RML_ChooseConnector.png](/images/partner_18093483788052.png)

4. Click on \+ Sender at the top right corner of the screen to configure the sender details, as explained below.![RML_SenderConfig.png](/images/partner_18093483789972.png)

| Field Name | Description |
| --- | --- |
| Sender name | This field denotes the name you want to provide to the Sender profile. This would be available in the Sender so that you can recognize this easily while using it to create a campaign inside MoEngage. The _Sender Name_ should be between 5-50 characters. This is a mandatory field. |
| WhatsApp business number | This field denotes the phone number registered with WhatsApp (directly or via Route Mobile) using which you want to send out WhatsApp Messages to your users. This is a mandatory field and is also referred to as the WABA number. |
| API KEY | This field denotes the API KEY required to authorize WhatsApp requests to Route Mobile from MoEngage. To get this information, contact support@routemobile.com and whatsapp@routemobile.com. |
| Enable delivery tracking | To track the delivery of your WhatsApp Messages in MoEngage, copy the MoEngage Delivery Tracking URL and pass it to the Route Mobile team. You should be able to see the URL as shown below: ![RML_DTConfig.png](/images/partner_18093483791380.png) **Note** : The URL is a sample URL. Copy the exact URL from your MoEngage Dashboard while configuring the sender. |
  
        5. Click Save to save the sender's details. Upon successful integration, you will get a success message.

![SuccessMsg.png](/images/partner_18069890177428.png)




## Steps to add Delivery Tracking URL

  1.      1.         1. Login to your Route Mobile’s WhatsApp account.
        2. Navigate to the WhatsApp Profile Management tab, as shown in the image below.
        3. Add the delivery callback URL available in the MoEngage Dashboard (refer to In the Enable Delivery Callback section) to the Callback URL field within the Profile Management section, as shown below.![RouteMobile_waba_config_page.png](/images/partner_18328181454228.png)
