# Introduction

MoEngage provides a native integration with NexG Platforms for WhatsApp. This integration allows you to send templated WhatsApp messages and view comprehensive delivery and read analytics within the MoEngage UI.

<Info>
Before you begin the configuration, ensure that you have the following:

  * Access to your NexG Platforms dashboard.
  * The API key associated with your NexG Platforms account.
  * A phone number registered and verified with WhatsApp for Business (WABA).
</Info>
# Configure NexG Platforms as a WhatsApp Connector

This section describes how to set up NexG Platforms as a service provider on the MoEngage UI.

## Step 1: Access Sender Configuration

To access the sender configuration, perform the following steps:

  1. Sign in to the MoEngage UI.
  2. Navigate to **Settings** > **Channels** > **WhatsApp** > **Sender configuration**.
  3. Click **\+ Add connector**. The WhatsApp connector dialog box appears.
  4. In the **Choose a connector** list, select **NexG** , and then click **Done**.  
![](/images/partner_46267770930836.png)



## Step 2: Configure Sender Details

To configure the sender details, perform the following steps:

  1. On the **Sender configuration** page, click **\+ Sender** in the upper-right corner.
  2. On the **Sender Details** page, enter the following information:

| Field | Description |
| --- | --- |
| **Sender name** | Enter a unique, recognizable name for this sender profile. This name is used to identify the sender when you create a campaign in MoEngage. Use between 5 and 50 characters. |
| **WhatsApp business number** | Enter the phone number registered with WhatsApp (WABA number) that you will use to send messages. |
| **API URL** | This field contains information about the URL that should be used to send an API request to the sender. You can get this information from the sender's API documentation. Enter the API endpoint provided by your vendor. The API URL for NexG Platforms is: [https://automate.nexgplatforms.com/api/v1/wa/moengage](https://automate.nexgplatforms.com/api/v1/wa/moengage) |
| **authorization** | Enter the API key obtained from your vendor. ![](/images/partner_46267773716628.png) |

  3. Click **Save**. Upon successful integration, a success message appears confirming that the sender's details are saved.  
![nexG.png](/images/partner_46409361111188.png)



# Delivery Tracking

To track the delivery status of your WhatsApp messages within MoEngage, you must provide MoEngage's Delivery Tracking URL to NexG Platforms and perform the following steps:

  1. On the **Sender Details** page, locate **Enable delivery tracking**.
  2. Click the copy icon ![](/images/partner_46267692555540.png)next to the delivery tracking URL.  
![](/images/partner_46267750877332.png)
  3. Provide this URL to your NexG Platforms representative or update it in your NexG Platforms account settings to enable delivery callbacks:
