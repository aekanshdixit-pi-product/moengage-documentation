MoEngage has native integration for WhatsApp with Yellow.ai. This integration allows businesses to send templated WhatsApp messages using Senders powered by Yellow.ai. MoEngage can send messages, show Delivery and Read Analytics.

## Sender Configuration

To configure a Sender from Yellow.ai on the MoEngage Dashboard, do the following:

  1. Go to **Settings** -> **WhatsApp** -> **Sender Configuration**.
  2. Click **+Add connector **on the left menu.
  3. Choose **Yellow.ai** from the list of WA providers or connectors available in the dropdown and click **Done**. 

![WAConnector_Yellow.ai.png](/images/partner_20394787315220.png)

  4. Click on **\+ Sender** at the top right corner of the screen to configure the sender details, as explained below. 

![WAConnector_Yellow.ai_Sender.png](/images/partner_20394804045844.png)

5. Add the following details:

| Field Name | Description |
| --- | --- |
| **Sender name** | This field denotes the name you want to provide to the Sender profile. This would be available in the Sender so that you can recognize this easily while using it to create a campaign inside MoEngage. The _Sender Name_ should be between 5-50 characters. This is a mandatory field. |
| **WhatsApp business number** | This field denotes the phone number registered with WhatsApp (directly or via Yellow.ai) using which you want to send out WhatsApp Messages to your users.__ This is a mandatory field and is also referred to as the WABA number. |
| **API URL** | Share the delivery tracking URL available in the Sender Configuration screen with your Yellow.ai POC to receive the API URL (Webhook URL). Paste the URL shared in the **API URL** field. |
| **Authentication** | Share the delivery tracking URL available in the Sender Configuration screen with your Yellow.ai POC to receive the Authentication (API Key). Paste the API Key shared in the **Authentication** field. |
| **Enable delivery tracking** | To track the delivery of your WhatsApp Messages inside MoEngage, you would need to copy the MoEngage Delivery Tracking URL and share it with Yellow.ai to get it updated in the MoEnagge-Yellow platform integration. The delivery tracking URL is available in the WhatsApp MoEngage Dashboard as shown below: ![Yellow.ai_DTURL.png](/images/partner_20395231941268.png) Note: The delivery tracking URL is a sample URL. Copy the exact URL from your MoEngage Dashboard before you share it with the Yellow.ai team. |
  6. Click **Save**. Upon successful integration, you will get the following message. 

![SaveIntegration.png](/images/partner_20394560973844.png)




## Steps to perform in the Yellow.ai platform's integration

  1. Go to the Yellow.ai platform's integrations page.

  2. **Copy Postback URL** — Choose MoEngage and copy the postback URL mentioned in the instruction section (on the right side of the page).

  3. **Enable Postback URL and Notification Engine API**

     1. Navigate to **Engagement** > **Preferences**.

     2. Enable both the Postback URL and Notification Engine API.

     3. Paste the Postback URL copied from the previous step

     4. Click **Save** to confirm.  
![20.png](/images/partner_29224402214676.png)




<Info>
* You should have pre-approved templates with Yellow.ai to create a WhatsApp campaign for a Yellow.ai Sender in MoEngage. You can create templates in Yellow.ai's engagement section.
  * You can create the same template verbatim in MoEngage. For more information, refer to [Add Templates](/user-guide/campaigns-and-channels/whatsapp/getting-started-with-whatsapp/whatsapp-templates).
</Info>
