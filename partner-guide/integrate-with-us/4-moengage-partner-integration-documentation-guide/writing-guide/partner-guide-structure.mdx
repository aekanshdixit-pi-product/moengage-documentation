## Troubleshooting & FAQs❓

_(List 2-3 common issues users might face and their solutions)_.

_For example:_

**Issue: Data from [Partner name] is not appearing in MoEngage**.

  * **Verify API Credentials** : Go to your **[Partner name]** dashboard and double-check that the MoEngage `APP ID` and `DATA API KEY` are correctly entered. Ensure there are no leading or trailing spaces.
  * **Check for Initial Sync Delay** : After the initial setup, it can take up to 15-30 minutes for the first set of user data and events to appear in MoEngage. Check the MoEngage dashboard again after this period.
  * **Confirm Data is Being Sent** : Check the event delivery logs or outbound data section in your **[Partner name]** account to confirm that data is actively being sent to the MoEngage endpoint. If no data is being sent, the issue lies within the **[Partner name]** configuration.
  * **Verify User ID Mapping** : Ensure that the primary user identifier (like `customer_id` or `email`) used in **[Partner name]** matches the one configured as the `User ID` in MoEngage to avoid creating duplicate profiles.
