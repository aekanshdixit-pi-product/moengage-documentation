# Introduction 

Marketers want expand on marketing personalisation to boost customer engagement and conversations. They want to insert content (such as personalised coupon codes, product catalogues, product attributes) that is accessible via APIs into MoEngage campaigns. 

# Scope 

This functionality is currently supported only for Email and Push campaigns. 

# Integration

You can use [MoEngage Content APIs](/user-guide/campaigns-and-channels/getting-started/campaign-content/content-apis) to fetch and insert personalised data directly into your campaigns. Content APIs allows to pull content either directly from a web server or from a publicly accessible APIs. Follow the steps on our [help documentation](/user-guide/campaigns-and-channels/getting-started/campaign-content/content-apis) to start integrating.

Additionally, you can also check how our other Technology Partners like [Talon.one](/partner-guide/message-orchestration/loyalty/talon-one), [Rocketium](/partner-guide/personalization/dynamic-content/rocketium) are using Content APIs to fetch coupon code and creatives respectively.
